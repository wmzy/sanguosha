# 游戏引擎增强 设计文档

## 概述

为三国杀游戏引擎添加以下功能：
1. 延时锦囊系统（闪电/乐不思蜀/兵粮寸断）
2. 无懈可击响应链
3. 桃给队友救援
4. 装备效果（八卦阵/仁王盾等）
5. 卡牌转化系统（武圣/龙胆等）

采用栈式中断架构，Promise 式异步流程控制。

## 核心架构：中断栈

### 设计原理

三国杀有大量"中断窗口"——当某个动作发生时，其他玩家可以插入响应。这些中断是 LIFO 的（后插入的先响应），适合用栈管理。

```
正常游戏流程
  └─ 杀 → 闪响应窗口
  └─ 锦囊 → 无懈可击响应窗口
       └─ 无懈可击 → 无懈可击反制窗口
  └─ 濒死 → 桃响应窗口
  └─ 延时锦囊 → 判定
```

### 中断栈实现

```typescript
// engine/interrupt.ts

interface InterruptFrame {
  type: 'kill_response' | 'trick_response' | 'dying' | 'judge';
  data: unknown;
  resolve: (result: unknown) => void;
  reject: (reason: string) => void;
}

class InterruptStack {
  private stack: InterruptFrame[] = [];

  // 发起中断，等待响应
  async wait<T>(type: InterruptFrame['type'], data: unknown): Promise<T> {
    return new Promise((resolve, reject) => {
      this.stack.push({
        type,
        data,
        resolve: resolve as (r: unknown) => void,
        reject,
      });
    });
  }

  // 响应当前中断
  resolve(result: unknown): void {
    const frame = this.stack.pop();
    frame?.resolve(result);
  }

  // 当前中断
  current(): InterruptFrame | undefined {
    return this.stack[this.stack.length - 1];
  }

  isEmpty(): boolean {
    return this.stack.length === 0;
  }
}

export const interruptStack = new InterruptStack();
```

### 使用示例

```typescript
// 杀的完整流程
async function handleKill(attacker: string, target: string) {
  // 1. 发起杀→闪中断
  const dodged = await interruptStack.wait<boolean>('kill_response', {
    attacker, target, card: killCard
  });

  if (dodged) return; // 被闪避

  // 2. 扣血
  const newHealth = targetPlayer.health - 1;

  // 3. 濒死检查
  if (newHealth <= 0) {
    const saved = await interruptStack.wait<boolean>('dying', {
      player: target
    });
    if (!saved) {
      playerDeath(target);
      return;
    }
  }

  // 4. 正常扣血
  applyDamage(target, 1);
}
```

### 判定流程

```typescript
// engine/judge.ts

interface JudgeResult {
  card: Card;             // 判定牌
  success: boolean;       // 是否生效
  effect: string;         // 效果描述
}

async function resolveJudge(
  game: GameState,
  player: string,
  trick: PendingTrick
): Promise<JudgeResult> {
  // 翻开牌堆顶的牌
  const judgeCard = game.deck[0];
  const newDeck = game.deck.slice(1);

  let success = false;
  switch (trick.name) {
    case '乐不思蜀':
      success = judgeCard.suit !== '♥';  // 红心失效
      break;
    case '兵粮寸断':
      success = judgeCard.suit !== '♣';  // 梅花失效
      break;
    case '闪电':
      success = judgeCard.suit === '♠'
        && ['2','3','4','5','6','7','8','9'].includes(judgeCard.rank);
      break;
  }

  return { card: judgeCard, success, effect: trick.name };
}
```

### 回合开始处理

```typescript
async function handlePendingTricks(game: GameState, player: string) {
  const tricks = getPlayer(player).pendingTricks ?? [];

  for (const trick of [...tricks]) {
    const result = await resolveJudge(game, player, trick);

    if (result.success) {
      switch (trick.name) {
        case '乐不思蜀':
          // 跳过出牌阶段
          skipPhase(game, player, '出牌');
          break;
        case '兵粮寸断':
          // 跳过摸牌阶段
          skipPhase(game, player, '摸牌');
          break;
        case '闪电':
          // 受到3点雷电伤害
          await applyDamage(game, player, 3, '雷电');
          // 闪电传给下家
          passLightning(game, player);
          break;
      }
    }

    // 移除延时锦囊（无论是否生效）
    removePendingTrick(game, player, trick);
  }
}
```

## 无懈可击响应链

### 响应流程

```typescript
async function handleTrickPlay(
  game: GameState,
  trick: Card,
  source: string,
  target?: string
): Promise<GameState> {
  // 发起无懈可击中断
  const nullified = await interruptStack.wait<boolean>('trick_response', {
    trick, source, target
  });

  if (nullified) {
    // 锦囊被抵消
    log('trick_nullified', { trick: trick.name, source, target });
    return game;
  }

  // 执行锦囊效果
  return executeTrickEffect(game, trick, source, target);
}
```

### 无懈可击嵌套

无懈可击可以被无懈可击反制：

```typescript
// 响应无懈可击时，可以再打出无懈可击
async function handleNullifyResponse(
  game: GameState,
  player: string
): Promise<boolean> {
  const hasNullify = getPlayer(player).hand.some(c => c.name === '无懈可击');

  if (!hasNullify) return false;

  // 发起嵌套中断
  const counterNullified = await interruptStack.wait<boolean>('trick_response', {
    trick: { name: '无懈可击' },
    source: player,
  });

  return counterNullified;
}
```

## 桃给队友救援

### 濒死处理流程

```typescript
async function handleDying(
  game: GameState,
  dyingPlayer: string
): Promise<{ saved: boolean; by?: string }> {
  // 1. 濒死者自己出桃
  const selfPeach = getPlayer(dyingPlayer).hand.find(c => c.name === '桃');
  if (selfPeach) {
    const usePeach = await interruptStack.wait<boolean>('dying', {
      player: dyingPlayer,
      canSelfSave: true,
    });
    if (usePeach) {
      return { saved: true, by: dyingPlayer };
    }
  }

  // 2. 其他存活玩家出桃救援
  const alivePlayers = getAlivePlayers(game).filter(p => p.name !== dyingPlayer);
  for (const saver of alivePlayers) {
    const peach = saver.hand.find(c => c.name === '桃');
    if (peach) {
      const willSave = await interruptStack.wait<boolean>('dying', {
        player: dyingPlayer,
        saver: saver.name,
        canSelfSave: false,
      });
      if (willSave) {
        return { saved: true, by: saver.name };
      }
    }
  }

  // 3. 无人救援，死亡
  return { saved: false };
}
```

### 规则引擎集成

```typescript
// engine/rules.ts — 扩展

function getPlayableCards(game: GameState, player: Player): Card[] {
  const realCards = player.hand;
  const convertedCards: Card[] = [];

  for (const conversion of getConversions(player)) {
    const matching = realCards.filter(c => matchesFilter(c, conversion.from));
    for (const card of matching) {
      convertedCards.push({
        ...card,
        name: conversion.to,
        _original: card,
        _conversion: conversion.name,
      });
    }
  }

  return [...realCards, ...convertedCards];
}

function matchesFilter(card: Card, filter: CardFilter): boolean {
  if (filter.name && !filter.name.includes(card.name)) return false;
  if (filter.suit && !filter.suit.includes(card.suit)) return false;
  if (filter.color) {
    const isRed = card.suit === '♥' || card.suit === '♦';
    if (filter.color === 'red' && !isRed) return false;
    if (filter.color === 'black' && isRed) return false;
  }
  return true;
}
```
