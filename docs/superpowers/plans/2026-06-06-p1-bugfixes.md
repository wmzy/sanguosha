# P1-E 实施计划：修判定牌取错 + 触发器 phase + redirect + shuffleDeck（修 §4.4/§4.6 + 抽 2 机制）

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修 3 个 §4 一致性问题（§4.4 触发器 phase 字段被忽略、§4.6 判定牌取错位置、§4.1 收尾的兜底），并抽 2 个新机制：`redirect` 钩子（天香/流离/借刀目标转移）+ `shuffleDeck` atom（鬼道/闪电改判）。

**Architecture:**
- **§4.6 判定牌**：在 `judge` atom 的 `apply` 阶段把判定牌 cardId 存到 `ctx.localVars.judgeCardId`；`getResult` 读此字段不读弃牌堆。
- **§4.4 触发器 phase**：把 `src/src/engine/skill.ts:147-153` 的 phase 检查从"仅 phaseBegin"扩展到所有事件类型。
- **redirect 钩子**：抽 `registerAtomHook(atomType, onBefore, return { redirect: newTarget })` —— damage / becomeTarget 都支持。
- **shuffleDeck atom**：把 `src/src/engine/atoms/reshuffle.ts` 复用，包装为 `shuffleDeck(target: 'deck' | 'judgmentZone')`。

**Tech Stack:** TypeScript 5.9 + vitest 4.1 + pnpm。`registerAtomHook` 扩 `redirect` 返回值；新 `shuffleDeck` atom 复用 P0 reshuffle。

**Spec:** `docs/ENGINE-DESIGN.md` §4.4（触发器 phase）+ §4.6（判定牌）+ §6 P1 `redirect/transferDamage` + `shuffleDeck` 两项。

**Non-Goals:**
- 不实现天香/流离/借刀技能本身（只做底层 redirect 钩子）
- 不实现鬼道/闪电改判（只做 shuffleDeck atom）

### 4.4 写 `src/src/engine/atoms/shuffleDeck.ts`

```ts
import type { GameState, Atom, AtomEventResult } from '../types';
import { registerAtom } from '../atom';
import { makeServerEvent, makePlayerEvent } from '../event';
import { createRng } from '../../src/shared/rng';

export function register() {
  registerAtom({
    type: 'shuffleDeck',
    apply(state: GameState, _atom: Atom & { type: 'shuffleDeck' }): GameState {
      const deck = state.zones.deck;
      if (deck.length <= 1) return state;
      const rng = createRng(state.rngState);
      const shuffled = [...deck];
      for (let i = shuffled.length - 1; i > 0; i--) {
        const j = rng.nextInt(i + 1);
        [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]];
      }
      return { ...state, zones: { ...state.zones, deck: shuffled }, rngState: rng.getState() };
    },
    toEvents(_state, _atom) {
      const server = makeServerEvent('shuffleDeck', {});
      return [server, new Map(), makePlayerEvent('shuffleDeck', {})];
    },
  });
}
```
