# 三国杀标准版 + 军争篇 卡牌详细信息

> 本文档整理三国杀标准版及军争篇扩展全部卡牌的详细规则信息。
> 参考来源：项目代码定义 (`src/shared/cards.ts`, `src/src/shared/types.ts`) 及官方规则。

---

## 一、基本牌

### 1.1 杀

| 属性 | 内容 |
|------|------|
| 名称 | 杀 |
| 类型 | 基本牌 |
| 子类型 | 杀 |
| 花色分布 | 共30张（具体花色见 `src/engine/cards/基础.ts`） |

**使用条件：**
- 出牌阶段使用
- 必须对攻击范围内的角色使用（初始攻击范围为1，装备武器后按武器范围计算）
- 每回合默认只能使用1张【杀】（装备诸葛连弩除外）

**目标限制：**
- 你攻击范围内的1名其他角色（**不能对自己使用**）

**效果：**
- 目标角色须使用1张【闪】来抵消，否则受到1点伤害
- 如果目标角色不使用【闪】，则受到1点伤害

**特殊规则：**
- 使用【杀】时可能触发武器的特殊效果（如青釭剑无视防具、雌雄双股剑对异性目标的效果等）
- 部分武将技能可以额外使用【杀】或改变【杀】的效果

### 1.2 闪

| 属性 | 内容 |
|------|------|
| 名称 | 闪 |
| 类型 | 基本牌 |
| 子类型 | 闪 |
| 花色分布 | 共15张（具体花色见 `src/engine/cards/基础.ts`） |

**使用条件：**
- 当你成为【杀】的目标时，可以使用或打出【闪】
- 不能在其他情况下主动使用

**响应时机：**
- 在【杀】对目标生效前使用
- 八卦阵等防具可以在需要使用【闪】时进行判定代替

**效果：**
- 抵消当前【杀】对你造成伤害的效果

**特殊规则：**
- 装备八卦阵时，判定为红色可视为打出了一张【闪】
- 部分武将技能可以将其他牌当【闪】使用

---

### 1.3 桃

| 属性 | 内容 |
|------|------|
| 名称 | 桃 |
| 类型 | 基本牌 |
| 子类型 | 桃 |
| 花色分布 | 共8张（具体花色见 `src/engine/cards/基础.ts`） |

**使用方法 Ⅰ（出牌阶段）：**
- 目标：包括你在内的 1 名**已受伤**的角色（可对自己，也可对其他已受伤角色）
- 效果：目标角色回复 1 点体力
- 备注：体力已满 / 未受伤的角色不能成为目标

**使用方法 Ⅱ（濒死时）：**
- 目标：1 名处于濒死状态的角色
- 效果：目标角色回复 1 点体力
- 备注：每次濒死时可以有多名角色依次使用【桃】直到脱离濒死

**效果：**
- 目标角色回复 1 点体力（不能超过体力上限）

**特殊规则：**
- 出牌阶段（方法 Ⅰ）与濒死时（方法 Ⅱ）属于不同的使用时机
- 部分武将技能可以额外使用【桃】或将其他牌当【桃】使用

### 1.4 酒（军争篇新增）

| 属性 | 内容 |
|------|------|
| 名称 | 酒 |
| 类型 | 基本牌 |
| 子类型 | 酒 |
| 花色分布 | ♠3, ♠9, ♣3, ♣9, ♦9（共5张） |

**使用方法 Ⅰ（出牌阶段，每回合限一次）：**
- 目标：**自己**（**注**：GB 文档此处写"包括你在内的一名角色"是错的，标准规则方法 Ⅰ 只能对自己使用，buff 作用于"于其上酒的该角色"的下一张杀）
- 效果：自己于此回合内使用的下一张【杀】的伤害值基数 +1
- 备注：判定【酒】（方法 Ⅰ）加成的是「下一张【杀】」，按使用时机而定

**使用方法 Ⅱ（濒死时）：**
- 目标：你
- 效果：你回复 1 点体力
- 备注：濒死时使用不附加伤害加成效果

**特殊规则：**
- 出牌阶段给自己使用是常见的玩法
- 【酒】的伤害加成与【裸衣】等技能的伤害加成叠加
- 【酒】杀命中后，造成的伤害为 2 点（基础 1 + 酒 1）
- 火杀/雷杀也可享受【酒】的伤害加成
- 被【酒】（方法 Ⅰ）加成的【杀】若未在此回合内使用，buff 不会带入下回合

**特殊规则：**
- 火属性不会被藤甲防止
- 被藤甲目标受到火杀时伤害+1
- 与普通杀共享使用次数限制

### 2.3 决斗

| 属性 | 内容 |
|------|------|
| 名称 | 决斗 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠A |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 1名其他角色

**效果/流程：**
1. 使用者对目标角色使用【决斗】
2. 由目标角色开始，其与使用者轮流打出1张【杀】
3. 首先不打出【杀】的角色受到「另一名角色」造成的1点伤害

**特殊规则：**
- 无距离限制
- 目标角色先出【杀】，处于劣势（先消耗）
- 伤害来源是「另一名角色」：例如使用者先不打出【杀】则使用者受伤，来源是目标角色
- 可以被【无懈可击】抵消

### 2.4 借刀杀人

| 属性 | 内容 |
|------|------|
| 名称 | 借刀杀人 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♣Q, ♦Q（共2张） |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 装备区里有武器牌、且其攻击范围内有「其使用【杀】的合法目标」的其他角色 A

**效果/流程：**
1. 你指定 A 为目标角色 A，并同时指定 A 攻击范围内 A 使用【杀】的合法目标 B
2. A 需对 B 使用【杀】；否则 A 须将装备区里的武器牌交给你
3. 若 A 对 B 使用了【杀】，则此【杀】正常结算（B 可用【闪】响应）

**特殊规则：**
- **需进行两次「使用【杀】的合法性检测」**（这是核心规则）：
  1. 第一次：你选择 A 和 B 时——A 的攻击范围内必须有 A 使用【杀】的合法目标 B；否则不能对 A 使用【借刀杀人】
  2. 第二次：A 实际使用【杀】时——B 必须仍然是 A 使用【杀】的合法目标
- A 的【杀】**不能"无视距离"**——两次检测都受 A 攻击范围限制
- A 可使用属性【杀】、技能【杀】等任何能对 B 使用的【杀】
- 可以被【无懈可击】抵消

| 子类型 | 普通锦囊 |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 1名「有手牌」的其他角色

**效果/流程：**

### 2.7 铁索连环（军争篇新增）

| 属性 | 内容 |
|------|------|
| 名称 | 铁索连环 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠J, ♠Q, ♣J, ♣Q, ♣K（共6张） |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 1至2名角色

**效果：**
- 目标角色选择一项：
  1. **横置**（若已横置则状态不变）
  2. **重置**（解除横置状态）
- **此牌能重铸**（弃置此牌并摸 1 张牌，不对任何角色生效）

**连环规则：**
- 当一名横置角色受到**属性伤害**（火属性或雷属性）后，触发连环传导
- 其他所有处于横置状态的角色依次受到同属性的伤害
- 每名角色每轮次只会传导一次（防止无限循环）

**特殊规则：**
- 铁索连环本身不造成伤害，是属性伤害的放大器
- 配合火杀/雷杀/火攻/闪电等可造成连锁伤害
- 庞统的连环技能可将梅花牌当铁索连环使用
- 每名角色最多可被横置一次（多次使用只重置横置状态）

- 出牌阶段使用

## 五、锦囊牌 - 响应锦囊

### 5.1 无懈可击

| 属性 | 内容 |
|------|------|
| 名称 | 无懈可击 |
| 类型 | 锦囊牌 |
| 子类型 | 响应锦囊 |
| 花色/点数 | ♠K |

**使用时机：**
- 在一张锦囊牌对目标角色生效前使用
- 可以抵消该锦囊对该目标角色的效果

**嵌套规则：**
- 无懈可击可以抵消无懈可击（嵌套使用）
- 即：A使用无懈可击抵消某锦囊 -> B使用无懈可击抵消A的无懈可击 -> C可以继续使用无懈可击...
- 直到某方不再使用无懈可击，最后使用的无懈可击生效

**效果：**
- 抵消目标锦囊对指定角色的效果
- 对全体锦囊（万箭齐发、南蛮入侵、桃园结义、五谷丰登）使用时，只抵消对特定1名角色的效果

**特殊规则：**
- 是唯一的响应型锦囊
- 无懈可击本身也是锦囊牌，也可以被无懈可击抵消
- 可以对自己使用的锦囊使用无懈可击（但通常无意义）
- 此外还有「国」版【无懈可击】（见 5.2），适用于国-标模式

- 一张锦囊牌对一个目标生效前

- 抵消此锦囊牌

#### 5.2.2 使用方法Ⅱ（势力扩散版）

| 属性 | 内容 |
|------|------|

| 适用模式 | 国-标 |

**特殊规则：**
- "势力相同"包含使用者自己
- 一旦选择扩散抵消，本回合内其他人无法再用【无懈可击】救这些势力相同的角色

## 八、关键规则补充

### 距离计算
- 两名角色之间的距离 = 逆时针方向的座位数与顺时针方向座位数中的较小值
- 装备进攻马：你计算与其他角色距离-1
- 装备防御马：其他角色计算与你的距离+1

### 锦囊响应顺序
- 全体锦囊的响应顺序：从使用者的下家开始，按逆时针方向依次响应
- 每名角色依次决定是否响应

### 无懈可击嵌套
- 无懈可击可以抵消无懈可击
- 形成链式嵌套，直到某一方放弃使用无懈可击

### 濒死处理
- 角色体力降至0或更低时进入濒死状态
- 从该角色开始，逆时针方向每名角色依次有机会使用【桃】
- 直到该角色脱离濒死（体力恢复至1以上）或无人使用【桃】则死亡

## 九、项目代码对应关系

本项目 (`src/shared/cards.ts`) 中的卡牌定义与上述规则一致：

- 基本牌在 `createDeck()` 函数中通过花色数组批量创建
- 锦囊牌定义为独立常量，包括 `trickSubtype` 字段区分普通/延时/响应
- 装备牌定义为独立常量，武器包含 `range` 字段
- 类型定义在 `src/src/shared/types.ts` 中，包括 `Card` 接口和各种类型别名
