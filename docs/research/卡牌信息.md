# 三国杀标准版卡牌详细信息

> 本文档整理三国杀标准版（风林火山之前）全部卡牌的详细规则信息。
> 参考来源：项目代码定义 (`shared/cards.ts`, `shared/types.ts`) 及官方标准版规则。

---

## 一、基本牌

### 1.1 杀

| 属性 | 内容 |
|------|------|
| 名称 | 杀 |
| 类型 | 基本牌 |
| 子类型 | 杀 |
| 花色分布 | ♠3~7, ♣3~7, ♥10~J, ♦6~10（共17张） |

**使用条件：**
- 出牌阶段使用
- 必须对攻击范围内的角色使用（初始攻击范围为1，装备武器后按武器范围计算）
- 每回合默认只能使用1张【杀】（装备诸葛连弩除外）

**目标限制：**
- 攻击范围内的1名角色（可以是自己，但通常无意义）

**效果：**
- 目标角色须使用1张【闪】来抵消，否则受到1点伤害
- 如果目标角色不使用【闪】，则受到1点伤害

**特殊规则：**
- 使用【杀】时可能触发武器的特殊效果（如青釭剑无视防具、雌雄双股剑对异性目标的效果等）
- 部分武将技能可以额外使用【杀】或改变【杀】的效果

---

### 1.2 闪

| 属性 | 内容 |
|------|------|
| 名称 | 闪 |
| 类型 | 基本牌 |
| 子类型 | 闪 |
| 花色分布 | ♥3~6, ♦10~K（共8张） |

**使用条件：**
- 当你成为【杀】的目标时，可以使用或打出【闪】
- 不能在其他情况下主动使用

**响应时机：**
- 在【杀】对目标生效前使用
- 八卦阵等防具可以在需要使用【闪】时进行判定代替

**效果：**
- 抵消当前【杀】对你造成伤害的效果

**特殊规则：**
- 装备八卦阵时，判定为红色可视为打出了一张【闪】
- 部分武将技能可以将其他牌当【闪】使用

---

### 1.3 桃

| 属性 | 内容 |
|------|------|
| 名称 | 桃 |
| 类型 | 基本牌 |
| 子类型 | 桃 |
| 花色分布 | ♥7~9, ♣Q（共4张） |

**使用条件（对自己）：**
- 出牌阶段，对自己使用
- 回复1点体力
- 体力已满时不能对自己使用

**使用条件（对他人）：**
- 当其他角色处于濒死状态时，你可以对其使用【桃】
- 濒死状态：体力值降至0或更低时
- 每次濒死时，可以有多名角色依次使用【桃】直到脱离濒死

**效果：**
- 目标角色回复1点体力（不能超过体力上限）

**特殊规则：**
- 出牌阶段对自己使用与濒死时对他人使用是不同的时机
- 部分武将技能可以额外使用【桃】或将其他牌当【桃】使用

## 二、锦囊牌 - 普通锦囊（单目标）

### 2.1 过河拆桥

| 属性 | 内容 |
|------|------|
| 名称 | 过河拆桥 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠3 |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 1名其他角色（不能对自己使用）

**效果：**
- 你选择该角色区域内（手牌、装备区、判定区）的1张牌并弃置之

**特殊规则：**
- 无需距离限制，可对任意距离的其他角色使用
- 可以选择目标角色判定区的延时锦囊弃置（相当于帮其解除延时锦囊）

---

### 2.2 顺手牵羊

| 属性 | 内容 |
|------|------|
| 名称 | 顺手牵羊 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠3 |

**使用条件：**
- 出牌阶段使用
- 距离限制：目标必须在你距离1以内

**目标限制：**
- 距离1以内的1名其他角色

**效果：**
- 你选择该角色区域内（手牌、装备区、判定区）的1张牌并获得之（收入手牌）

- 距离1以内意味着相邻角色或装备进攻马后的相邻角色

### 2.3 决斗

| 属性 | 内容 |
|------|------|
| 名称 | 决斗 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠A |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 1名其他角色

**效果/流程：**
1. 使用者对目标角色使用【决斗】
2. 由目标角色开始，双方轮流打出1张【杀】
3. 首先不打出【杀】的角色受到1点伤害

**特殊规则：**
- 无距离限制
- 目标角色先出【杀】，即目标处于劣势（先消耗）
- 可以被【无懈可击】抵消

## 四、锦囊牌 - 延时锦囊

### 4.1 乐不思蜀

| 属性 | 内容 |
|------|------|
| 名称 | 乐不思蜀 |
| 类型 | 锦囊牌 |
| 子类型 | 延时锦囊 |
| 花色/点数 | ♠6 |

**使用条件：**
- 出牌阶段使用
- 距离限制：目标必须在你距离1以内

**目标限制：**
- 距离1以内的1名其他角色

**判定条件：**
- 目标角色判定阶段进行判定
- 若判定结果不为♥（红桃），则跳过该角色的出牌阶段
- 若判定结果为♥（红桃），则无效果，乐不思蜀弃置

**特殊规则：**
- 延时锦囊放置在目标角色的判定区
- 在目标角色的判定阶段开始时进行判定
- 可以被【无懈可击】抵消（判定前抵消或判定后抵消效果）
- 可以被【过河拆桥】拆掉

---

### 4.2 兵粮寸断

| 属性 | 内容 |
|------|------|
| 名称 | 兵粮寸断 |
| 类型 | 锦囊牌 |
| 子类型 | 延时锦囊 |
| 花色/点数 | ♣10 |

**使用条件：**
- 出牌阶段使用
- 距离限制：目标必须在你距离1以内

**目标限制：**
- 距离1以内的1名其他角色

**判定条件：**
- 目标角色判定阶段进行判定
- 若判定结果不为♣（梅花），则跳过该角色的摸牌阶段
- 若判定结果为♣（梅花），则无效果，兵粮寸断弃置

**特殊规则：**
- 延时锦囊放置在目标角色的判定区
- 跳过摸牌阶段意味着该角色不能从牌堆摸牌
- 可以被【无懈可击】抵消
- 可以被【过河拆桥】拆掉

---

### 4.3 闪电

| 属性 | 内容 |
|------|------|
| 名称 | 闪电 |
| 类型 | 锦囊牌 |
| 子类型 | 延时锦囊 |
| 花色/点数 | ♠A |

**使用条件：**
- 出牌阶段使用
- 放置在自己的判定区

**判定条件：**
- 判定阶段进行判定
- 若判定结果为♠2~9（黑桃2至黑桃9），则你受到3点雷电伤害，然后闪电弃置
- 其他结果，闪电传给下家（逆时针方向下一个角色）

**传递规则：**
- 判定不是♠2~9时，闪电传递给逆时针方向的下一个角色
- 传递时放置在下一个角色的判定区
- 如果下一个角色的判定区已有延时锦囊，则跳过该角色继续传递

**特殊规则：**
- 闪电是对所有角色都有威胁的延时锦囊
- 3点雷电伤害是游戏中单次最高的固定伤害之一
- 可以被【无懈可击】抵消
- 可以被【过河拆桥】拆掉
- 闪电是唯一会传递的延时锦囊

## 五、锦囊牌 - 响应锦囊

### 5.1 无懈可击

| 属性 | 内容 |
|------|------|
| 名称 | 无懈可击 |
| 类型 | 锦囊牌 |
| 子类型 | 响应锦囊 |
| 花色/点数 | ♠K |

**使用时机：**
- 在一张锦囊牌对目标角色生效前使用
- 可以抵消该锦囊对该目标角色的效果

**嵌套规则：**
- 无懈可击可以抵消无懈可击（嵌套使用）
- 即：A使用无懈可击抵消某锦囊 -> B使用无懈可击抵消A的无懈可击 -> C可以继续使用无懈可击...
- 直到某方不再使用无懈可击，最后使用的无懈可击生效

**效果：**
- 抵消目标锦囊对指定角色的效果
- 对全体锦囊（万箭齐发、南蛮入侵、桃园结义、五谷丰登）使用时，只抵消对特定1名角色的效果

**特殊规则：**
- 是唯一的响应型锦囊
- 无懈可击本身也是锦囊牌，也可以被无懈可击抵消
- 可以对自己使用的锦囊使用无懈可击（但通常无意义）

- 被麒麟弓击中时可以被弃置

## 八、关键规则补充

### 距离计算
- 两名角色之间的距离 = 逆时针方向的座位数与顺时针方向座位数中的较小值
- 装备进攻马：你计算与其他角色距离-1
- 装备防御马：其他角色计算与你的距离+1

### 锦囊响应顺序
- 全体锦囊的响应顺序：从使用者的下家开始，按逆时针方向依次响应
- 每名角色依次决定是否响应

### 无懈可击嵌套
- 无懈可击可以抵消无懈可击
- 形成链式嵌套，直到某一方放弃使用无懈可击

### 濒死处理
- 角色体力降至0或更低时进入濒死状态
- 从该角色开始，逆时针方向每名角色依次有机会使用【桃】
- 直到该角色脱离濒死（体力恢复至1以上）或无人使用【桃】则死亡

## 九、项目代码对应关系

本项目 (`shared/cards.ts`) 中的卡牌定义与上述规则一致：

- 基本牌在 `createDeck()` 函数中通过花色数组批量创建
- 锦囊牌定义为独立常量，包括 `trickSubtype` 字段区分普通/延时/响应
- 装备牌定义为独立常量，武器包含 `range` 字段
- 类型定义在 `shared/types.ts` 中，包括 `Card` 接口和各种类型别名
