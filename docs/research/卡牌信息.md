# 三国杀标准版 + 军争篇 卡牌详细信息

> 本文档整理三国杀标准版及军争篇扩展全部卡牌的详细规则信息。
> 参考来源：项目代码定义 (`src/shared/cards.ts`, `src/src/shared/types.ts`) 及官方规则。

---

## 一、基本牌

### 1.1 杀

| 属性 | 内容 |
|------|------|
| 名称 | 杀 |
| 类型 | 基本牌 |
| 子类型 | 杀 |
| 花色分布 | ♠3~7, ♣3~7, ♥10~J, ♦6~10（共17张） |

**使用条件：**
- 出牌阶段使用
- 必须对攻击范围内的角色使用（初始攻击范围为1，装备武器后按武器范围计算）
- 每回合默认只能使用1张【杀】（装备诸葛连弩除外）

**目标限制：**
- 攻击范围内的1名角色（可以是自己，但通常无意义）

**效果：**
- 目标角色须使用1张【闪】来抵消，否则受到1点伤害
- 如果目标角色不使用【闪】，则受到1点伤害

**特殊规则：**
- 使用【杀】时可能触发武器的特殊效果（如青釭剑无视防具、雌雄双股剑对异性目标的效果等）
- 部分武将技能可以额外使用【杀】或改变【杀】的效果

---

### 1.2 闪

| 属性 | 内容 |
|------|------|
| 名称 | 闪 |
| 类型 | 基本牌 |
| 子类型 | 闪 |
| 花色分布 | ♥3~6, ♦10~K（共8张） |

**使用条件：**
- 当你成为【杀】的目标时，可以使用或打出【闪】
- 不能在其他情况下主动使用

**响应时机：**
- 在【杀】对目标生效前使用
- 八卦阵等防具可以在需要使用【闪】时进行判定代替

**效果：**
- 抵消当前【杀】对你造成伤害的效果

**特殊规则：**
- 装备八卦阵时，判定为红色可视为打出了一张【闪】
- 部分武将技能可以将其他牌当【闪】使用

---

### 1.3 桃

| 属性 | 内容 |
|------|------|
| 名称 | 桃 |
| 类型 | 基本牌 |
| 子类型 | 桃 |
| 花色分布 | ♥7~9, ♣Q（共4张） |

**使用条件（对自己）：**
- 出牌阶段，对自己使用
- 回复1点体力
- 体力已满时不能对自己使用

**使用条件（对他人）：**
- 当其他角色处于濒死状态时，你可以对其使用【桃】
- 濒死状态：体力值降至0或更低时
- 每次濒死时，可以有多名角色依次使用【桃】直到脱离濒死

**效果：**
- 目标角色回复1点体力（不能超过体力上限）

**特殊规则：**
- 出牌阶段对自己使用与濒死时对他人使用是不同的时机
- 部分武将技能可以额外使用【桃】或将其他牌当【桃】使用

---

### 1.4 酒（军争篇新增）

| 属性 | 内容 |
|------|------|
| 名称 | 酒 |
| 类型 | 基本牌 |
| 子类型 | 酒 |
| 花色分布 | ♠3, ♠9, ♣3, ♣9, ♦9（共5张） |

**使用条件（出牌阶段）：**
- 对自己使用后，本回合你使用的下一张【杀】伤害+1
- 每回合只能使用1次【酒】（提升杀伤害）

**使用条件（濒死时）：**
- 当一名角色处于濒死状态时，你可以对其使用【酒】回复1点体力
- 濒死时使用不附加伤害加成效果

**效果：**
- 出牌阶段使用：本回合下一张【杀】伤害+1
- 濒死时使用：目标回复1点体力

**特殊规则：**
- 【酒】的伤害加成与【裸衣】等技能的伤害加成叠加
- 【酒】杀命中后，造成的伤害为2点（基础1+酒1）
- 火杀/雷杀也可享受【酒】的伤害加成

### 1.5 火杀（军争篇新增）

| 属性 | 内容 |
|------|------|
| 名称 | 火杀 |
| 类型 | 基本牌 |
| 子类型 | 杀 |
| 属性 | 火 |
| 花色分布 | ♥4, ♥7, ♥10, ♦4, ♦5（共5张） |

**使用条件：**
- 与普通【杀】相同，出牌阶段使用
- 受攻击范围限制
- 每回合使用次数限制与普通杀共享

**效果：**
- 对目标造成1点火焰伤害
- 可触发铁索连环传导（火焰伤害传导）
- 可触发藤甲火焰伤害+1效果

**特殊规则：**
- 火属性不会被藤甲防止
- 被藤甲目标受到火杀时伤害+1
- 与普通杀共享使用次数限制

---

| 属性 | 内容 |
|------|------|
| 名称 | 雷杀 |
| 类型 | 基本牌 |
| 子类型 | 杀 |
| 属性 | 雷 |
| 花色分布 | ♠4~♠8（5张）, ♣5~♣8（4张），共9张 |

- 与普通【杀】相同，出牌阶段使用
- 受攻击范围限制
- 每回合使用次数限制与普通杀共享

- 神诸葛亮的大雾技能可防止雷电以外的伤害

| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠A |

- 出牌阶段使用

- 1名其他角色

**效果/流程：**
1. 使用者对目标角色使用【决斗】
2. 由目标角色开始，双方轮流打出1张【杀】

- 无距离限制

### 2.4 借刀杀人

| 属性 | 内容 |
|------|------|
| 名称 | 借刀杀人 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♣Q, ♦Q（共2张） |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 装备区有武器牌的1名其他角色

**效果/流程：**
1. 你指定目标角色A（需有武器）
2. 你指定A使用【杀】攻击其攻击范围内的另一名角色B（你指定B）
3. A必须选择一项：
   - 对B使用1张【杀】（无视距离限制）
   - 将武器牌交给使用者（你）

**特殊规则：**
- 目标A可以正常使用杀应对（包括属性杀、技能杀等）
- 若A对B使用杀，则杀的效果正常结算（B可用闪响应）
- A不执行杀则武器归你所有
- 可以被【无懈可击】抵消

### 2.6 火攻（军争篇新增）

| 属性 | 内容 |
|------|------|
| 名称 | 火攻 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♥2, ♥3, ♦Q（共3张） |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 任意1名其他角色

**效果/流程：**
1. 目标角色展示1张手牌
2. 你需弃置1张与展示牌花色相同的手牌
3. 若弃置了同花色牌，目标受到1点火焰伤害
4. 若没有同花色牌可弃，无事发生

**特殊规则：**
- 可以看到目标展示的手牌再决定是否继续
- 火攻消耗：1张火攻+1张同花色牌=1点火焰伤害
- 可配合铁索连环传导火焰伤害
- 对藤甲目标伤害+1

### 2.7 铁索连环（军争篇新增）

| 属性 | 内容 |
|------|------|
| 名称 | 铁索连环 |
| 类型 | 锦囊牌 |
| 子类型 | 普通锦囊 |
| 花色/点数 | ♠J, ♠Q, ♣J, ♣Q, ♣K（共6张） |

**使用条件：**
- 出牌阶段使用

**目标限制：**
- 任意1至2名角色（可包括自己）

**效果：**
- 选择一项：
  1. **连环**：横置目标角色（若已横置则不变）
  2. **重铸**：弃置此牌并摸1张牌（不对任何角色生效）

**连环规则：**
- 当一名横置角色受到**属性伤害**（火属性或雷属性）后，触发连环传导
- 其他所有处于横置状态的角色依次受到同属性的伤害
- 每名角色每轮次只会传导一次（防止无限循环）

**特殊规则：**
- 铁索连环本身不造成伤害，是属性伤害的放大器
- 配合火杀/雷杀/火攻/闪电等可造成连锁伤害
- 庞统的连环技能可将梅花牌当铁索连环使用
- 每名角色最多可被横置一次（多次使用只重置横置状态）

### 4.3 闪电

| 属性 | 内容 |
|------|------|
| 名称 | 闪电 |
| 类型 | 锦囊牌 |
| 子类型 | 延时锦囊 |
| 花色/点数 | ♠A |

**使用条件：**
- 出牌阶段使用
- 放置在自己的判定区

**判定条件：**
- 判定阶段进行判定
- 若判定结果为♠2~9（黑桃2至黑桃9），则你受到3点雷电伤害，然后闪电弃置
- 其他结果，闪电传给下家（逆时针方向下一个角色）

**传递规则：**
- 判定不是♠2~9时，闪电传递给逆时针方向的下一个角色
- 传递时放置在下一个角色的判定区
- 如果下一个角色的判定区已有延时锦囊，则跳过该角色继续传递

**特殊规则：**
- 闪电是对所有角色都有威胁的延时锦囊
- 3点雷电伤害是游戏中单次最高的固定伤害之一
- 可以被【无懈可击】抵消
- 可以被【过河拆桥】拆掉
- 闪电是唯一会传递的延时锦囊

## 五、锦囊牌 - 响应锦囊

### 5.1 无懈可击

| 属性 | 内容 |
|------|------|
| 名称 | 无懈可击 |
| 类型 | 锦囊牌 |
| 子类型 | 响应锦囊 |
| 花色/点数 | ♠K |

**使用时机：**
- 在一张锦囊牌对目标角色生效前使用
- 可以抵消该锦囊对该目标角色的效果

**嵌套规则：**
- 无懈可击可以抵消无懈可击（嵌套使用）
- 即：A使用无懈可击抵消某锦囊 -> B使用无懈可击抵消A的无懈可击 -> C可以继续使用无懈可击...
- 直到某方不再使用无懈可击，最后使用的无懈可击生效

**效果：**
- 抵消目标锦囊对指定角色的效果
- 对全体锦囊（万箭齐发、南蛮入侵、桃园结义、五谷丰登）使用时，只抵消对特定1名角色的效果

## 八、关键规则补充

### 距离计算
- 两名角色之间的距离 = 逆时针方向的座位数与顺时针方向座位数中的较小值
- 装备进攻马：你计算与其他角色距离-1
- 装备防御马：其他角色计算与你的距离+1

### 锦囊响应顺序
- 全体锦囊的响应顺序：从使用者的下家开始，按逆时针方向依次响应
- 每名角色依次决定是否响应

### 无懈可击嵌套
- 无懈可击可以抵消无懈可击
- 形成链式嵌套，直到某一方放弃使用无懈可击

### 濒死处理
- 角色体力降至0或更低时进入濒死状态
- 从该角色开始，逆时针方向每名角色依次有机会使用【桃】
- 直到该角色脱离濒死（体力恢复至1以上）或无人使用【桃】则死亡

## 九、项目代码对应关系

本项目 (`src/shared/cards.ts`) 中的卡牌定义与上述规则一致：

- 基本牌在 `createDeck()` 函数中通过花色数组批量创建
- 锦囊牌定义为独立常量，包括 `trickSubtype` 字段区分普通/延时/响应
- 装备牌定义为独立常量，武器包含 `range` 字段
- 类型定义在 `src/src/shared/types.ts` 中，包括 `Card` 接口和各种类型别名
