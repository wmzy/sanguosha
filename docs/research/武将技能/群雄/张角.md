# 张角

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 张角 |
| 势力 | 群 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 雷击 | 主动技 | 当你使用或打出【闪】时，可令任意一名角色判定，若结果为黑桃，你对该角色造成2点雷电伤害。 |
| 鬼道 | 主动技 | 当一名角色的判定牌生效前，你可以用一张黑色牌替换之。 |
| 黄天 | 主公技 | 其他群势力角色可以在其出牌阶段将一张【闪】或【闪电】交给你。 |

## 技能详细规则

### 雷击

- **触发时机**：你使用或打出【闪】时（结算前）
- **发动条件**：无特殊条件
- **效果**：
  1. 令任意一名角色判定
  2. 若判定结果为黑桃，该角色受到2点雷电伤害
- **限制**：每张【闪】限一次
- **备注**：
  - 雷击的目标可以是任意角色，不一定是攻击者
  - 判定结果非黑桃则无事发生
  - 配合八卦阵判定出的【闪】也可触发雷击

### 鬼道

- **触发时机**：判定牌生效前
- **发动条件**：有黑色手牌
- **效果**：用一张黑色牌替换当前的判定牌
- **限制**：无次数限制
- **备注**：
  - 可以改变任何角色的判定结果
  - 最适合改自己的雷击判定为黑桃
  - 也可以改闪电判定等

### 黄天

- **触发时机**：其他群势力角色的出牌阶段
- **发动条件**：张角为主公
- **效果**：其他群势力角色可将一张【闪】或【闪电】交给你
- **限制**：主公技，每名角色每回合限一次

## FAQ/争议点

1. **Q：张角在出【闪】时可以同时发动鬼道和雷击吗？**
   A：可以。先打【闪】触发雷击，判定后用鬼道改判定牌为黑桃，确保雷击命中。

2. **Q：雷击造成的伤害来源是谁？**
   A：雷击的伤害来源是张角本人，与【闪电】不同。

3. **Q：雷击能否被【无懈可击】抵消？**
   A：不能。雷击是技能伤害，不是锦囊牌。
