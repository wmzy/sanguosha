# 华佗

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 华佗 |
| 势力 | 群 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 急救 | 主动技 | 你的回合外，你可以将一张红色牌当【桃】使用。 |
| 青囊 | 主动技 | 出牌阶段，你可以弃置一张手牌，然后令一名角色回复1点体力。 |

## 技能详细规则

### 急救

- **触发时机**：回合外（任何角色的回合）
- **发动条件**：有红色牌（手牌或装备区的牌）
- **效果**：将一张红色牌当【桃】使用
- **限制**：无次数限制
- **备注**：
  - 红色牌包括红桃和方块花色的牌
  - 可以使用装备区的红色牌
  - 使用后原牌进入弃牌堆
  - 可以对自己或濒死的角色使用

### 青囊

- **触发时机**：出牌阶段
- **发动条件**：有手牌
- **效果**：
  1. 弃置一张手牌
  2. 令一名角色回复1点体力
- **限制**：每回合限一次
- **备注**：
  - 可以对自己使用
  - 回复体力不能超过体力上限
  - 弃置的牌必须是手牌

## FAQ/争议点

1. **Q：急救能否在自己的回合内使用？**
   A：不能。急救只能在回合外使用。

2. **Q：青囊能否对自己使用？**
   A：可以。青囊可以选择任何角色（包括自己）。

3. **Q：急救能否使用装备区的红色牌？**
   A：可以。但装备会因此被弃置。
