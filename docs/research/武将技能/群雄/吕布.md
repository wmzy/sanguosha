# 吕布

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 吕布 |
| 势力 | 群 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 无双 | 锁定技 | 当你使用【杀】指定一名角色为目标时，该角色需连续使用两张【闪】才能抵消；与你进行【决斗】的角色每次需连续打出两张【杀】。 |

## 技能详细规则

### 无双

- **触发时机**：
  1. 使用【杀】指定目标时
  2. 进行【决斗】时
- **发动条件**：无特殊条件（锁定技，自动生效）
- **效果**：
  1. 目标角色需要连续使用两张【闪】才能抵消【杀】
  2. 与吕布进行【决斗】的角色每次需要连续打出两张【杀】
- **限制**：锁定技，自动生效
- **备注**：
  - 如果目标角色只有一张【闪】，则无法完全抵消
  - 【决斗】中每次响应都需要两张【杀】
  - 可以与【青釭剑】等武器配合

## FAQ/争议点

1. **Q：无双能否被【八卦阵】抵消？**
   A：【八卦阵】判定出的【闪】只能算一张，因此需要判定两次才能完全抵消。

2. **Q：无双能否被【倾国】等技能抵消？**
   A：可以。但需要连续使用两张【闪】（或等效的牌）。

3. **Q：无双能否在【决斗】中使用【杀】抵消？**
   A：可以。但需要连续打出两张【杀】。
