# 蔡文姬

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 蔡文姬 |
| 势力 | 群 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 悲歌 | 被动技 | 当一名角色受到【杀】造成的伤害后，你可以弃置一张牌，然后令该角色判定，根据判定结果执行效果。 |
| 断肠 | 锁定技 | 杀死你的角色立即失去所有技能直到游戏结束。 |

## 技能详细规则

### 悲歌

- **触发时机**：一名角色受到【杀】的伤害后
- **发动条件**：有手牌可弃
- **效果**：
  1. 弃置一张牌
  2. 令受伤害的角色判定
  3. 根据判定结果执行：
     - ♥（红桃）：受伤角色回复1点体力
     - ♦（方块）：受伤角色摸两张牌
     - ♣（梅花）：伤害来源弃置两张牌
     - ♠（黑桃）：伤害来源将其武将牌翻面
- **限制**：无次数限制
- **备注**：
  - 技能非常全面——回血、摸牌、拆牌、翻面
  - 可以根据局势选择是否发动
  - 弃牌是发动代价

### 断肠

- **触发时机**：杀死蔡文姬的角色
- **发动条件**：锁定技，自动触发
- **效果**：杀死蔡文姬的角色立即失去所有技能直到游戏结束
- **限制**：锁定技
- **备注**：
  - 这是蔡文姬的终极威慑技能
  - 杀死蔡文姬的人变白板
  - 因此很少有人敢主动杀蔡文姬
  - 是个"碰不得"的角色

## FAQ/争议点

1. **Q：断肠失去所有技能后装备是否还生效？**
   A：装备效果仍然生效（如诸葛连弩的多杀效果等）。

2. **Q：悲歌判定为黑桃时，翻面效果对已翻面的角色如何？**
   A：翻面后再翻回正面，恢复正常状态。

3. **Q：如果蔡文姬被技能杀死（非杀伤害），悲歌能否触发？**
   A：悲歌只在受到【杀】的伤害时触发，其他伤害不触发。
