# 董卓

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 董卓 |
| 势力 | 群 |
| 体力上限 | 8 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 酒池 | 主动技 | 你可以将一张黑桃手牌当【酒】使用。 |
| 肉林 | 锁定技 | 你对女性角色/女性角色对你使用【杀】时，需连续使用两张【闪】才能抵消。 |
| 崩坏 | 锁定技 | 回合结束阶段，若你的体力不是全场最少的（或同时为最少），你须减1点体力或1点体力上限。 |
| 暴虐 | 主公技 | 其他群雄角色每造成一次伤害，可进行一次判定，若结果为黑桃，你回复1点体力。 |

## 技能详细规则

### 酒池

- **触发时机**：需要使用【酒】时
- **发动条件**：有黑桃手牌
- **效果**：将一张黑桃手牌当【酒】使用
- **限制**：无次数限制
- **备注**：可以随时将黑桃牌转化为酒使用

### 肉林

- **触发时机**：使用【杀】指定女性目标/成为女性使用【杀】的目标时
- **发动条件**：锁定技，自动生效
- **效果**：目标需连续使用两张【闪】才能抵消
- **限制**：锁定技
- **备注**：与吕布的无双类似，只针对女性角色

### 崩坏

- **触发时机**：回合结束阶段
- **发动条件**：你的体力不是全场最少的（或同时为最少）
- **效果**：必须选择一项：减1点体力，或减1点体力上限
- **限制**：锁定技
- **备注**：
  - 董卓体力上限会逐渐降低
  - 可以通过选择减体力而非体力上限来延缓死亡

### 暴虐

- **触发时机**：其他群雄角色造成伤害后
- **发动条件**：董卓为主公
- **效果**：进行判定，若为黑桃，董卓回复1点体力并获得该判定牌
- **限制**：主公技

## FAQ/争议点

1. **Q：董卓的体力上限为8，崩坏会使之降低到多少？**
   A：最少可降至1。每次崩坏可选择减体力或体力上限。

2. **Q：酒池使用【酒】是否有次数限制？**
   A：无次数限制。每回合可以使用多个酒。

3. **Q：崩坏能否选择不执行？**
   A：不能。崩坏是锁定技，必须选择减1点体力或1点体力上限。
