## 技能详解

### 酒池（转化技）

> **机制简述**：黑桃手牌当酒使用

**官方效果**：你可以将一张黑桃手牌当【酒】使用。

**实现要点**：
1. 组合 action（镜像武圣）：前端选黑桃手牌 → 点酒池 → 提交 `preceding=[酒池.transform]` + 主 action=酒.use
2. 酒池.transform 创建影子「酒」入 cardMap，酒技能零感知酒池（看到的永远是 cardMap 里的「酒」）
3. 黑桃 = ♠（仅黑桃，非黑色；与红颜/断粮的「黑色」语义不同）
4. 无次数限制

**实现差异/边界**：实现与官方描述一致。

### 肉林（锁定技）

> **机制简述**：与女性互杀需两张闪抵消

**官方效果**：锁定技，你对女性角色使用的【杀】和女性角色对你使用的【杀】均需使用两张【闪】才能抵消。

**实现要点**：
1. 镜像无双实现：辅助函数 enforceRoulinDodge 嵌入「杀」的 execute
2. 因等待型 atom（询问闪）被 before-hook cancel（如八卦阵判红放虚拟闪）后 after-hook 不触发，无法用 hook 拦截追加第二轮询问，故在 execute 内 applyAtom(询问闪) 返回后主动调用 enforceRoulinDodge 检查肉林条件并追加第二次询问
3. 触发条件（任一）：董卓为 source 且 target 女性 / 董卓为 target 且 source 为女性

**实现差异/边界**：实现与官方描述一致。机制上需嵌入杀的 execute（而非纯 hook），与无双同构。

### 崩坏（锁定技）

> **机制简述**：非最低体力时失血或减上限

**官方效果**：锁定技，结束阶段，若你不是体力值最小的角色，你失去1点体力或减少1点体力上限。

**实现要点**：
1. 锁定技，after hook 挂「阶段开始」，phase='回合结束'（结束阶段）
2. 触发条件解析（对齐官方规则）：体力严格大于全场最小值则触发；体力 == 全场最小（含并列最少）则不触发
3. 选择：减1点体力上限（设上限）或 减1点体力（失去体力）
4. 体力上限最低降至 1（设上限 atom.validate 要求 amount>0）

**实现差异/边界**：实现与官方描述一致。「不是全场最少（或之一）」= 体力严格大于全场最小值，触发判定正确。

### 暴虐（主公技）

> **机制简述**：群将伤人后令你判定，黑桃则回血

**官方效果**：主公技，当其他群势力角色造成伤害后，其可以令你判定，若结果为黑桃，你回复1点体力。

**实现要点**：
1. 主公技（仅董卓为主公 `identity==='主公'` 时触发），两个 after hook
2. 造成伤害 after：其他群雄角色（source≠董卓 + faction=群）造成伤害 → 询问董卓是否判定
3. 判定 after：暴虐判定（judgeType='暴虐'）→ 黑桃（♠）→ 回复1点体力
4. 系统伤害（source<0，如闪电）不触发
5. 判定牌在 frame.cards 末尾（判定 atom 自身 afterHooks 移入弃牌堆之前读取）

**实现差异/边界**：
- **边界裁定差异**：详细规则提到「并获得该判定牌」，但主描述（表格，唯一事实来源）只说「回复1点体力」。实现以主描述为准，**仅实现回复体力，未实现获得判定牌**。若按详细规则裁定则缺一个获得判定牌的效果。
