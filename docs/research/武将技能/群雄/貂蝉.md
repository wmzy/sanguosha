# 貂蝉

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 貂蝉 |
| 势力 | 群 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 离间 | 主动技 | 出牌阶段，你可以弃置一张牌，然后令一名男性角色视为对另一名男性角色使用一张【决斗】。 |
| 闭月 | 被动技 | 回合结束阶段，你可以摸一张牌。 |

## 技能详细规则

### 离间

- **触发时机**：出牌阶段
- **发动条件**：
  1. 有牌可以弃置
  2. 场上有至少两名存活的男性角色
- **效果**：
  1. 弃置一张牌
  2. 选择一名男性角色A
  3. 选择另一名男性角色B
  4. A视为对B使用一张【决斗】
- **限制**：每回合限一次
- **备注**：
  - 弃置的牌可以是手牌或装备牌
  - A和B必须是两名不同的男性角色
  - 【决斗】的发起者是A，目标是B
  - 可以被【无懈可击】抵消

### 闭月

- **触发时机**：回合结束阶段
- **发动条件**：无特殊条件
- **效果**：摸一张牌
- **限制**：每回合限一次
- **备注**：
  - 闭月是主动发动的，可以选择不发动
  - 摸牌后加入手牌

## FAQ/争议点

1. **Q：离间能否选择女性角色？**
   A：不能。只能选择男性角色。

2. **Q：离间能否选择同一名角色？**
   A：不能。A和B必须是两名不同的角色。

3. **Q：闭月能否在回合外发动？**
   A：不能。闭月只能在回合结束阶段发动。
