## 技能详解

### 双雄（转化技）

> **机制简述**：摸牌判色，异色手牌当决斗使用

**官方效果**：摸牌阶段，你可以改为进行一次判定，你获得判定牌且本回合可以将一张与之颜色不同的手牌当【决斗】使用。

**实现要点**：

- 两部分组合：A) 摸牌阶段 before-hook（镜像再起/突袭）触发，B) 转化 transform action（preceding 决斗.use，镜像武圣单卡转化）
- A 流程：询问发动 → peek 牌堆顶 2 张 → 选其一 → 记 `turn.vars['双雄/suit']=所选牌花色` → 两张牌均弃置（牌堆→弃牌堆） → applyAtom(阶段结束, 摸牌) + return cancel 跳过默认摸牌
- B：把一张花色匹配的手牌当【决斗】（影子卡 outputName='决斗'）
- 花色状态同步：`turn.vars['双雄/suit']` 由 before-hook 写入 state，view 侧经「回合用量」atom 同步到 `players[me].turnUsage['双雄/suit']`

**实现差异/边界**：**重大差异（机制完全不同）**：

1. **触发动作**：官方「进行一次**判定**」+「获得**判定牌**」（1 张）；实现是「展示牌堆顶**两张**」+「选**其一**」（2 张选 1，且两张均弃置不入手）。

2. **转化条件**：官方「一张与之**颜色不同**的手牌当决斗」；实现「一张与此牌**同花色**的手牌当决斗」——条件相反（颜色 vs 花色，且不同 vs 相同）。

实现与官方描述在两个核心机制上均不一致。
