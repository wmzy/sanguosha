# 颜良文丑

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 颜良文丑 |
| 势力 | 群 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 双雄 | 主动技 | 摸牌阶段，你可以放弃摸牌，改为展示牌堆顶两张牌并选择其中一张，然后本回合你可以将一张与此牌同花色的手牌当【决斗】使用。 |

## 技能详细规则

### 双雄

- **触发时机**：摸牌阶段
- **发动条件**：无特殊条件
- **效果**：
  1. 放弃摸牌（不摸2张牌）
  2. 展示牌堆顶两张牌
  3. 选择其中一张
  4. 本回合你可以将一张与所选牌同花色的手牌当【决斗】使用
- **限制**：每回合限一次
- **备注**：
  - 每回合只能发动一次
  - 展示的两张牌都不收入手牌，均弃置
  - 本回合可使用无限次【决斗】（只要有对应花色手牌）
  - 决斗目标的【杀】使用次数受限制

## FAQ/争议点

1. **Q：双雄能否在一回合内多次使用不同花色的决斗？**
   A：只能使用与所选那张牌同花色的手牌当决斗。每张符合条件的手牌都可当决斗。

2. **Q：双雄展示的两张牌能否获得？**
   A：不能。展示后选择一张作为花色参考，两张都弃置。

3. **Q：如果展示的两张牌没有合适的花色怎么办？**
   A：必须选择其中一张的花色作为本回合决斗的花色，如果手里没有这个花色的牌就无法决斗。
