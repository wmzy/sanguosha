# 贾诩

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 贾诩 |
| 势力 | 群 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 完杀 | 锁定技 | 在你的回合，除你以外，只有处于濒死状态的角色才能使用【桃】。 |
| 乱武 | 限定技 | 出牌阶段，你可以令所有其他角色依次对与其距离最近的另一名角色使用一张【杀】，无法如此做者失去1点体力。 |
| 帷幕 | 锁定技 | 你不能成为黑色锦囊的目标。 |

## 技能详细规则

### 完杀

- **触发时机**：濒死结算时
- **发动条件**：在你的回合内
- **效果**：只有濒死角色本人可以对自己使用【桃】，其他角色不能对其使用【桃】
- **限制**：锁定技
- **备注**：
  - 这是贾诩的招牌技能
  - 可配合AOE或直接输出确保击杀
  - 桃园结义和类似的群体回复不受影响（因为不指定目标）

### 乱武

- **触发时机**：出牌阶段
- **发动条件**：无特殊条件
- **效果**：
  1. 所有其他角色依次执行：
  2. 对与其距离最近的角色使用一张【杀】
  3. 无法使用【杀】者失去1点体力
- **限制**：限定技（整局游戏一次）
- **备注**：
  - 强制角色自相残杀
  - 没有杀或不想用杀则会失去体力
  - 配合完杀可以造成致命效果

### 帷幕

- **触发时机**：成为锦囊目标时
- **发动条件**：锁定技，自动生效
- **效果**：你不能成为黑色锦囊的目标
- **限制**：锁定技
- **备注**：
  - 黑色锦囊包括：过河拆桥、顺手牵羊、南蛮入侵、决斗等
  - 红色锦囊（如桃园结义、万箭齐发、五谷丰登等）不受影响
  - 黑色延时锦囊（乐不思蜀、兵粮寸断）也不受影响
  - 是对黑色锦囊的完全免疫

## FAQ/争议点

1. **Q：完杀状态下，濒死角色能否对自己使用【桃】？**
   A：可以。完杀只禁止其他角色使用【桃】，濒死者自己可以吃桃。

2. **Q：乱武能否被【无懈可击】抵消？**
   A：不能。乱武是技能效果，不是锦囊牌。

3. **Q：帷幕能否防止黑色装备牌的效果？**
   A：帷幕只防止黑色锦囊牌成为目标，不防止装备效果。
