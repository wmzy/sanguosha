# 界貂蝉

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界貂蝉 |
| 势力 | 群 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 离间 | 主动技 | 出牌阶段，你可以弃置一张牌，令一名男性角色视为对另一名男性角色使用一张【决斗】。每回合限一次。 |
| 闭月 | 被动技 | 结束阶段，你可以摸一张牌。或你弃置一张牌，然后令一名其他角色摸一张牌。 |

## 技能详细规则

### 离间

- **变化**：从只能弃手牌改为弃置一张牌（手牌或装备）

### 闭月

- **变化**：新增选项——可弃牌令其他角色摸牌
- **效果**：
  - 选项1：自己摸一张牌（原版）
  - 选项2：弃一张牌，令一名其他角色摸一张牌

## FAQ/争议点

1. **Q：闭月可以给队友补牌吗？**
   A：可以。选项2就是辅助摸牌。
