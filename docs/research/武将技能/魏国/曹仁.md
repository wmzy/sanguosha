## 技能详解

### 据守（被动技）

> **机制简述**：结束阶段翻面摸四牌并弃或用装备

**官方效果**：结束阶段，你可以翻面并摸四张牌，然后弃置一张手牌，若为装备牌，你改为使用之。

**实现要点**：

- 主动 use action：在 弃牌/回合结束 阶段发动
- 效果：加 翻面 标签 + 摸 3 张 + 标记本回合已用
- 翻面标签触发下一回合跳过：阶段开始(准备) before-hook 检测标签 → 移除 + 设 skipAll + cancel
- skipAll 标志存在时 cancel 所有阶段；阶段结束(准备) before-hook 清标志 + 主动推进回合（清过期标记 + 下一玩家 + 回合结束）
- 副作用：据守触发回合的 phase-end 链不消费任何摸牌/出牌/弃牌副作用

**实现差异/边界**：**重大差异（旧版/新版机制不符）**：官方 hero/29 现行描述为「翻面并**摸四张牌**，然后**弃置一张手牌，若为装备牌你改为使用之**」（OL 加强版）；引擎实现的是**旧版/标版**据守——「翻面并**摸三张牌**，然后**跳过下一回合**」。两者机制完全不同：新版摸 4 + 弃 1（不跳过整回合），旧版摸 3 + 跳过整回合。

**未实现**：新版「弃一张手牌/装备使用」分支、新版摸牌数量（4 而非 3）。

引擎实现的是 风林火山 原始版本，与官方现行 hero 页描述不一致。

## FAQ / 备注

- **版本不一致**：hero/29 现行官方描述为加强版「翻面摸 4 + 弃一张手牌/装备使用」；引擎实现为 风林火山 原版「翻面摸 3 + 跳过下一回合」。两版机制完全不同。
- 引擎实现的是标版/旧版，与官方现行 hero 页描述不一致。加强版据守在 界曹仁 中已实现（见 界曹仁.md）。
