# 曹丕

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 曹丕 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 行殇 | 被动技 | 你可以立即获得死亡角色的所有牌。 |
| 放逐 | 被动技 | 每当你受到一次伤害后，可以令除你以外的任一角色补X张牌（X为你已损失体力值），然后该角色将其武将牌翻面。 |
| 颂威 | 主公技 | 其他魏势力角色的判定牌结果为黑色且生效后，可以让你摸一张牌。 |

## 技能详细规则

### 行殇

- **触发时机**：任意角色死亡时
- **发动条件**：无特殊条件
- **效果**：获得死亡角色的所有牌（手牌、装备牌、判定区牌）
- **限制**：无次数限制
- **备注**：
  - 死亡角色剩余的所有牌进入曹丕手牌
  - 可以叠加积累大量牌

### 放逐

- **触发时机**：受到伤害后
- **发动条件**：有其他存活角色
- **效果**：
  1. 令一名其他角色摸X张牌（X=曹丕已损失体力值）
  2. 该角色将其武将牌翻面
- **限制**：无次数限制
- **备注**：
  - 翻面是该角色跳过下一回合
  - 翻面效果可以用来控制敌人，也可以帮助队友（队友可以翻面后下回合多摸牌）
  - X为已损失体力值，曹丕血越少摸牌越多

### 颂威

- **触发时机**：魏势力角色的黑色判定牌生效后
- **发动条件**：曹丕为主公
- **效果**：摸一张牌
- **限制**：主公技

## FAQ/争议点

1. **Q：放逐能否选择自己？**
   A：不能。只能选择除自己以外的角色。

2. **Q：放逐的摸牌数和翻面效果可以分开选择吗？**
   A：不可拆分。目标角色同时获得摸牌和翻面。

3. **Q：行殇能否选择不获得牌？**
   A：可以。行殇是主动发动的，可以选择不发动。
