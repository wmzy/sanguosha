## 技能详解

### 屯田（被动技）

> **机制简述**：回合外失牌判非红桃得田减距离

**官方效果**：当你于回合外失去牌后，你可以判定，当非红桃判定牌生效后，你将此牌置于你的武将牌上，称为“田”。你计算与其他角色的距离-X（X为你“田”的数量）。

**实现要点**：

- 被动技；after hook 挂「获得」「移动牌」「弃置」（覆盖回合外失去牌的主要路径）
- 「回合外」判定：currentPlayerIndex !== ownerId
- 判定结果在 判定 after hook 中捕获花色（判定牌在 frameCards 末尾，在判定 atom 自身 afterHooks 把它移入弃牌堆之前）
- 非红桃时：把判定牌从 frame.cards 拿出（直接 mutate——无 atom 支持「置于武将牌上」），加田标记
- 田存储：每张田 = 一个 mark，id 形如 `屯田/田:N`，payload.cardId 携带判定牌 id
- 距离修正：每张田使 距离/进攻修正 +1（更新 vars，后端 effectiveDistance 据此生效）

**实现差异/边界**：**已知引擎限制**：距离修正的 vars 变更不经 atom，`view.distanceVars` 不实时同步——后端正确，前端展示需额外通道（仅装备/卸下 atom 提供 view 同步通道）。

**流程细节**：官方「判定，**当非红桃判定牌生效后**，你将此牌置于武将牌上」——实现是「判定后捕获花色，非红桃则直接把判定牌从 frame 拿出加田」（判定 atom 后续 splice 末尾时 frame 已空，no-op）；语义一致但路径略不同。

一次失去事件触发一次判定（不按卡牌数量重复）。

### 凿险（觉醒技）

> **机制简述**：田≥3减上限获急袭

**官方效果**：觉醒技，准备阶段，若“田”的数量不小于3，你减少1点体力上限，然后获得“<font href='139'>急袭</font>”。

**实现要点**：

- 觉醒技；after hook 挂「阶段开始」(phase=准备)
- 触发：田数量（marks 中 `屯田/田:` 前缀的 mark 数量）≥ 3
- 效果：设上限（maxHealth-1）→ 添加技能 急袭（将一张「田」当【顺手牵羊】使用）
- 整局一次：`player.vars['凿险/awakened']` 防重入
- 减上限后若体力>新上限，设上限 atom 自动 clamp

**实现差异/边界**：实现与官方描述一致。
