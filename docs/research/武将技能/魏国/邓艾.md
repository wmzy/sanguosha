# 邓艾

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 邓艾 |
| 势力 | 魏 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 屯田 | 被动技 | 每次当你于回合外失去牌时，可进行一次判定，将非红桃的判定牌置于你的武将牌上，称为"田"；每有一张田，你计算与其他角色的距离便减少1。 |
| 凿险 | 觉醒技 | 准备阶段，若"田"的数量≥3，你须减1点体力上限，然后获得技能"急袭"（你可以将一张"田"当【顺手牵羊】使用）。 |

## 技能详细规则

### 屯田

- **触发时机**：回合外失去牌时
- **发动条件**：无特殊条件
- **效果**：
  1. 进行一次判定
  2. 若结果不为红桃，将判定牌置于武将牌上作为"田"
  3. 每张"田"使邓艾与其他角色距离-1
- **限制**：无次数限制
- **备注**：
  - 回合外失去牌包括被顺、被拆、被借刀、被拼点、被弃等
  - 判红桃则无事发生
  - 田的数量决定偷牌的距离

### 凿险（觉醒技）

- **触发时机**：准备阶段
- **发动条件**：田的数量≥3
- **效果**：
  1. 减1点体力上限（永久）
  2. 获得技能"急袭"：可将一张"田"当【顺手牵羊】使用
- **限制**：觉醒技，触发一次后永久改变

### 急袭

- **触发时机**：出牌阶段
- **发动条件**：有"田"
- **效果**：将一张"田"当【顺手牵羊】使用
- **限制**：无次数限制

## FAQ/争议点

1. **Q：觉醒后体力上限降低，会影响当前体力吗？**
   A：不影响当前体力。若体力值为4，体力上限从4降到3，则体力值变为3。

2. **Q：田被当顺手牵羊使用后，还作为田存在吗？**
   A：不。田被消耗了，不再提供距离减少效果。

3. **Q：邓艾能否在回合内通过弃牌触发屯田？**
   A：不能。屯田指定"回合外"失去牌。
