# 张辽

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 张辽 |
| 势力 | 魏 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 突袭 | 主动技 | 摸牌阶段，你可以放弃摸牌，改为获得最多两名其他角色的各一张手牌。 |

## 技能详细规则

### 突袭

- **触发时机**：摸牌阶段
- **发动条件**：场上存在其他有手牌的角色
- **效果**：放弃摸牌，选择至多两名其他角色，获得他们各一张手牌
- **限制**：每回合限一次
- **备注**：
  - 可以选择1名或2名角色
  - 获得的牌是随机的（不能指定获得哪张）
  - 如果选择的角色没有手牌，则不能选择该角色
  - 获得的牌直接加入手牌，不展示

## FAQ/争议点

1. **Q：突袭能否选择同一名角色的两张牌？**
   A：不能。必须选择至多两名不同角色，每人获得一张。

2. **Q：突袭能否选择没有手牌的角色？**
   A：不能。必须选择有手牌的角色。

3. **Q：突袭获得的牌能否被反馈等技能获得？**
   A：可以。突袭获得的牌在加入手牌后，可以被其他技能影响。
