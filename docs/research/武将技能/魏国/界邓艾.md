## 技能详解

### 界屯田（被动技）

> **机制简述**：回合外失牌或回合内弃杀后判定蓄田，每田减1距离

**官方效果**：当你于回合外失去牌后，或你于回合内弃置【杀】后，你可以判定，若结果不为红桃，将判定牌置于你的武将牌上，称为“田”。你计算与其他角色的距离-X（X为“田”的数量）。

**实现要点**：
1. after hook 挂「获得」「移动牌」「弃置」（覆盖失去牌的主要路径）
2. 触发条件（两路）：「回合外」失去牌（`currentPlayerIndex !== ownerId`）；「回合内弃置杀」（`currentPlayerIndex === ownerId` 且 弃置 atom.cardIds 含 杀）
3. 判定 → 非红桃（非♥）→ 加「田」标记 + 更新距离修正
4. 田的存储：每张田 = 一个 mark，id 形如 `屯田/田:N`，payload.cardId 携带判定牌 id；count = marks 中前缀为 `屯田/田:` 的数量
5. 距离修正：每张田使 `距离/进攻修正` +1（更新 vars，后端 effectiveDistance 据此生效）
6. 一次失去事件触发一次判定（不按卡牌数量重复）
7. 判定结果在「判定」after hook 中捕获花色（判定牌在 frameCards 末尾，在判定 atom 自身 afterHooks 移入弃牌堆之前）；非红桃时直接 mutate frame 拿走判定牌再加田标记

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：标版屯田仅「回合外失去牌」触发；界版额外在「回合内弃置杀」时触发（仅弃置杀，其他牌型不触发）——实现正确落地了此差异。

### 界凿险（觉醒技）

> **机制简述**：田≥3觉醒获急袭并执行额外回合

**官方效果**：觉醒技，准备阶段，若“田”的数量不小于3，你减少1点体力上限，然后获得“<font href='139'>急袭</font>”，你于本回合结束后执行一个额外回合。

**实现要点**：
1. 觉醒技（强制，整局一次），after hook 挂「阶段开始」phase='准备'
2. 触发条件：田数量 ≥3 且未觉醒（`player.vars['凿险/awakened']` 防重入）
3. 减1点体力上限（设上限 amount=maxHealth-1；减上限后若体力>新上限，设上限 atom 自动 clamp）
4. 永久获得「急袭」技能（需在 skills/index.ts 注册）
5. 觉醒时额外设 `界凿险/extraTurn` 标志，本回合结束时触发额外回合
6. 额外回合机制（参考放权/界放权）：before hook 挂「回合结束」→ cancel 正常回合结束 → 手动清理 per-turn 状态 → 亲自启动 ownerId 的额外回合；额外回合结束后正常座次顺序推进
7. 额外回合仅触发一次（标志在消费后立即清除）

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：标版凿险觉醒仅获急袭；界版官方追加「本回合结束后执行一个额外回合」——实现正确落地了额外回合机制。

### 急袭（转化技）

> **机制简述**：将一张田当顺手牵羊使用（觉醒获得）

**官方效果**：（hero页无对应技能描述；由界凿险觉醒获得）

**实现要点**：
1. 组合 action：`preceding=[急袭.transform]` + 主 action=顺手牵羊.use
2. 急袭.transform：选一张田 → 去标记（移除田）→ 创建影子「顺手牵羊」入对手牌 → 更新距离修正
3. 顺手牵羊.use：读手牌中的影子卡（name=顺手牵羊），正常流程执行
4. 与武圣/倾国区别：武圣/倾国是手牌→影子卡（用「当作」atom）；急袭是田(mark)→影子卡（田不在手牌，不能用「当作」，直接 mutate cardMap 创建影子卡 + push 手牌，通过「去标记」atom 移除田）
5. 田消耗后距离修正 vars 更新（每张田 -1 距离）
6. rollback 恢复田标记 + 距离修正 + 删影子卡；影子卡入弃牌堆时引擎用 shadowOf 还原为原卡（判定牌）
7. 顺手牵羊的距离检查在 use.validate 中：此时田已消耗、距离修正已更新，若距离变远导致 use validate 失败，rollback 恢复一切（安全）

**实现差异/边界**：hero页无独立描述（觉醒获得技），实现按「将一张田当顺手牵羊使用」的通用转化模型落地。
