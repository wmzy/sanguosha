## 技能详解

### 断粮（转化技）

> **机制简述**：黑色基本或装备牌当兵粮寸断，对牌多者无距离

**官方效果**：你可以将一张黑色基本牌或黑色装备牌当【兵粮寸断】使用。你对手牌数不小于你的角色使用【兵粮寸断】无距离限制。

**实现要点**：
1. 采用独立 use action（不复用 兵粮寸断.use，因后者距离校验硬编码 `effectiveDistance<=1`，无法承载「无距离限制」放松规则）
2. validate 校验：黑色（♠/♣）基本牌或装备牌
3. 距离规则：默认目标须在距离1以内（与兵粮寸断一致）；放松条件——目标手牌数 ≥ 徐晃手牌数 → 无距离限制
4. execute 直接放置 `name='兵粮寸断'` 的延时锦囊
5. 后续判定/跳过摸牌阶段效果由现有 兵粮寸断.ts hooks 处理（它们只认 pendingTricks.name，故复用）
6. 无次数限制（描述明确）

**实现差异/边界**：实现与官方描述一致。独立 use 路径解决了原兵粮寸断距离硬编码无法承载放松规则的问题，且转化牌与判定效果解耦清晰。
