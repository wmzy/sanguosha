# 界甄姬

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界甄姬 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 倾国 | 主动技 | 你可以将一张黑色牌当【闪】使用或打出。 |
| 洛神 | 被动技 | 准备阶段，你可以进行判定，若结果为黑色，你获得此判定牌，然后可以继续判定。若结果为红色，你也可获得此判定牌，但结束此流程。 |

## 技能详细规则

### 倾国

- **变化**：原版只能黑色手牌，界版可用黑色牌（包括装备牌）

### 洛神

- **变化**：原版红色直接中断不给牌，界版可获红色牌但流程结束
- **效果**：
  - 黑色：获得判定牌，继续判定
  - 红色：获得判定牌，但停止判定

## FAQ/争议点

1. **Q：界甄姬洛神判定红色时一定能获得牌吗？**
   A：可以。红色判定牌也会被获得，但洛神流程结束。
