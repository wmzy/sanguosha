# 甄姬

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 甄姬 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 倾国 | 主动技 | 你可以将一张黑色手牌当【闪】使用或打出。 |
| 洛神 | 被动技 | 准备阶段，你可以进行一次判定，若结果为黑色，你可以获得此判定牌，并重复此流程。 |

## 技能详细规则

### 倾国

- **触发时机**：需要使用或打出【闪】时
- **发动条件**：有黑色手牌
- **效果**：将一张黑色手牌当【闪】使用或打出
- **限制**：无次数限制
- **备注**：
  - 黑色手牌包括黑桃和梅花花色的牌
  - 使用后原牌进入弃牌堆

### 洛神

- **触发时机**：准备阶段
- **发动条件**：无特殊条件
- **效果**：
  1. 进行一次判定
  2. 若结果为黑色（黑桃或梅花），获得该判定牌
  3. 可以重复此流程，直到判定结果为红色或选择停止
- **限制**：无次数限制（理论上可以无限次，直到出现红色）
- **备注**：
  - 洛神是主动发动的，可以选择不发动
  - 获得的判定牌加入手牌
  - 每次判定独立，之前的判定结果不影响后续

## FAQ/争议点

1. **Q：洛神获得的判定牌能否被天妒获得？**
   A：不能。洛神已经获得了判定牌，天妒无法再获得。

2. **Q：倾国能否被【无懈可击】抵消？**
   A：倾国是当【闪】使用，不是锦囊牌，因此不能被【无懈可击】抵消。

3. **Q：洛神能否在装备【八卦阵】时发动？**
   A：可以。洛神和【八卦阵】是独立的，可以同时存在。
