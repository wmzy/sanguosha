## 技能详解

### 倾国（转化技）

> **机制简述**：将一张黑色手牌当闪使用或打出

**官方效果**：你可以将一张黑色手牌当【闪】使用或打出。

**实现要点**：

- 组合 action 模型（与武圣对称）：前端两步 UI（点倾国给黑牌加"闪"显示→点出闪）。
- 提交时 ClientMessage：preceding=[倾国.transform] + 主 action=闪.respond；后端 dispatch 先执行 倾国.transform（创建影子闪），再 闪.respond validate 看到"闪"通过。
- 【闪】技能零感知倾国——它看到的永远是 cardMap 里的一张"闪"。
- 黑色判定：花色为 ♠ 或 ♣；仅限手牌（描述明确"黑色手牌"）。

**实现差异/边界**：实现与官方描述一致。

### 洛神（主动技）

> **机制简述**：准备阶段判定，黑色则获得并重复

**官方效果**：准备阶段，你可以判定，若结果为黑色，你获得此牌，然后你可以重复此流程。

**实现要点**：

- 阶段开始（准备） after-hook → 询问是否发动 → 循环：判定 → 判定 after-hook 读判定牌花色 → 黑色：把判定牌从弃牌堆移到手牌（获得）→ 询问是否继续 → 重复 / 停止；红色：退出。
- 判定牌时序：判定 atom 的 skill after-hooks 在 def.afterHooks（把判定牌移入弃牌堆）之前跑；洛神主循环在 applyAtom(判定) 返回后，从弃牌堆顶读判定牌。
- 每获得一张黑色判定牌可继续，红色或玩家主动停止则结束。

**实现差异/边界**：实现与官方描述一致。准备阶段触发，可重复判定直至红色或主动停止。
