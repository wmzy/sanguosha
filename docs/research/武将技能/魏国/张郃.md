## 技能详解

### 巧变（主动技）

> **机制简述**：弃牌跳阶段换摸牌或移场上牌

**官方效果**：你可以弃置一张手牌并跳过一个阶段（准备阶段和结束阶段除外），若为：摸牌阶段，你可以获得至多两名角色各一张手牌；出牌阶段，你可以移动场上一张牌。

**实现要点**：
- before hook 挂在「阶段开始」：判定/摸牌/出牌/弃牌阶段开始时询问是否发动；发动则选弃牌 + 跳过当前阶段。
- 跳过摸牌阶段：询问选 1–2 名有手牌的其他角色，各获得其一张手牌。
- 跳过出牌阶段：询问源玩家+源牌+目标玩家，通过「移动牌」atom 完成场上牌移动。
- 跳过判定/弃牌阶段：仅弃牌 + 跳过，无附加效果。
- 跳过阶段手法（同神速/兵粮寸断）：`applyAtom(阶段结束, 当前阶段)` 推进到下一阶段，再 cancel 当前「阶段开始」atom。
- 出牌阶段移动的牌支持：手牌/装备/判定区 → 手牌/装备（装备→装备需 slot 不冲突；判定区→手牌视为"移回手牌"）。

**实现差异/边界**：
- **获得手牌的选择方**：实现"取目标手牌第 0 张"（固定首位），官方"获得...一张手牌"未明确选择方——按面杀常见 FAQ 应由张郃选择获得哪张，实现采用固定取第 0 张，目标手牌顺序不公开时近似"随机/盲取"。
- 每个阶段独立发动，无总次数限制——符合官方"你可以弃置一张手牌并跳过一个阶段"的逐阶段语义。
