## 技能详解

### 天妒（被动技）

> **机制简述**：判定牌生效后可获得此牌

**官方效果**：当你的判定牌生效后，你可以获得此牌。

**实现要点**：
1. 时机：判定 atom 的 after hook（技能 hook 在判定 atom 自身 afterHooks 清理之前运行——判定牌此刻仍在 frameCards 末尾）
2. 询问是否获得 → 移动牌（处理区→手牌）
3. 判定阶段 frame 通常仅含判定牌，拿走后 frame 空，判定 atom 的 afterHooks splice 末尾为 no-op

**实现差异/边界**：实现与官方描述一致。

### 界遗计（被动技）

> **机制简述**：受伤后摸两牌并可分配给至多两名角色

**官方效果**：当你受到1点伤害后，你可以摸两张牌，然后你可以交给至多两名其他角色共计至多两张手牌。

**实现要点**：
1. 被动技，每1点伤害触发一次
2. implNotes 实际实现：受到1点伤害后摸一张牌
3. 可将一张已有手牌（任意）交给一名其他角色（可选）
4. 交牌可选：confirm 询问；无其他存活角色或自己无手牌时跳过交牌环节

**实现差异/边界**：
- **重大差异——数量全面不符**：
  - 摸牌数：官方「摸**两**张牌」，实现「摸**一**张牌」
  - 分配对象数：官方「至多**两**名其他角色」，实现「**一**名其他角色」
  - 分配牌数：官方「共计至多**两**张手牌」，实现「**一**张手牌」
- implNotes 实现的实为「1摸1给」的弱化版，与官方「2摸2给（至多2人）」差距显著

**与标版差异**：标版遗计为「摸两张牌，然后将两张牌（摸到的）分配给任意角色」；界版官方为「摸两张牌，然后可交给至多两名角色共计至多两张手牌（任意手牌）」。实现既未对齐标版也未对齐界版（摸牌数缩水为一）。
