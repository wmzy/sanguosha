## 技能详解

### 界神速（主动技）

> **机制简述**：跳阶段视为出杀，至多三项，发动后可移动场上装备

**官方效果**：你可以选择至多三项：1.跳过判定阶段和摸牌阶段；2.跳过出牌阶段并弃置一张装备牌；3.跳过弃牌阶段并翻面。你每选择一项，你视为使用一张无距离限制的【杀】。

**实现要点**：
1. 在判定阶段 before-hook 统一询问各选项，集中结算（区别于标版在各阶段分别触发）
2. 选项1：加「跳过摸牌」标签 + 虚拟杀 + 跳过判定阶段(skipPhase)
3. 选项2：弃装备 + 加「跳过出牌」标签 + 虚拟杀
4. 选项3（implNotes 版）：**失去1点体力** + 加「跳过弃牌」标签 + 虚拟杀
5. 界版新增：发动后询问是否移动场上装备（卸下→移动牌→装备 序列）
6. 虚拟杀同标版：无实体卡，走 指定目标→成为目标→检测有效性→询问闪→伤害/抵消，不消耗手牌、不计入 杀/quota、无距离限制
7. usedThisTurn 后缀由 回合结束 atom 自动清理

**实现差异/边界**：
- **重大差异——选项3代价不符**：官方选项3为「跳过弃牌阶段并**翻面**」，实现选项3为「跳过弃牌阶段并**失去1点体力**」——**代价从翻面变为失去体力**
- **多出效果**：implNotes「界版新增移动场上装备」，但官方界神速 officialDesc（与标版神速完全相同）**未提及移动场上装备**——此为实现凭空添加

**与标版差异**：标版神速仅 2 选项且各阶段单独触发；界版官方为 3 选项统一询问。实现做到了 3 选项统一询问（贴近界版结构），但选项3代价与官方不符（翻面→失体力），并多出无依据的移动装备效果。
