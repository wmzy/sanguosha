# 司马懿

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 司马懿 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 反馈 | 被动技 | 你可以立即获得对你造成伤害的来源的一张牌。 |
| 鬼才 | 主动技 | 在任意角色的判定牌生效前，你可以打出一张手牌代替之。 |

## 技能详细规则

### 反馈

- **触发时机**：当你受到伤害后
- **发动条件**：伤害来源有牌（手牌或装备牌）
- **效果**：你可以获得伤害来源的一张牌
- **限制**：无次数限制
- **备注**：
  - 可以选择获得手牌或装备牌
  - 由司马懿选择获得哪张牌（不能观看手牌后选择）

### 鬼才

- **触发时机**：判定牌生效前（翻开判定牌后，结算效果前）
- **发动条件**：司马懿有手牌
- **效果**：打出一张手牌代替原判定牌
- **限制**：无次数限制
- **备注**：
  - 可以改变任何角色的判定结果（包括自己）
  - 可以用来改变【闪电】、【乐不思蜀】、【兵粮寸断】的判定
  - 替代牌的花色和点数以打出的手牌为准

## FAQ/争议点

1. **Q：反馈能否获得装备牌？**
   A：可以。反馈可以获得伤害来源的任意一张牌（手牌或装备区的牌）。

2. **Q：鬼才能否在判定牌翻开前发动？**
   A：不能。必须等判定牌翻开后，在生效前才能发动。

3. **Q：鬼才能否改变【闪电】的判定？**
   A：可以。当判定牌翻开后，可以用手牌替代。
