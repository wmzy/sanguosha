# 郭嘉

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 郭嘉 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 天妒 | 被动技 | 当你的判定牌生效后，你可以获得此判定牌。 |
| 遗计 | 被动技 | 当你受到1点伤害后，你可以摸两张牌，然后将两张牌交给任意角色（可以是自己）。 |

## 技能详细规则

### 天妒

- **触发时机**：判定牌生效后
- **发动条件**：判定牌存在（未被其他技能移除）
- **效果**：获得该判定牌
- **限制**：无次数限制
- **备注**：
  - 可以获得自己的判定牌
  - 判定牌通常是牌堆顶的一张牌

### 遗计

- **触发时机**：受到1点伤害后
- **发动条件**：无特殊条件
- **效果**：
  1. 摸两张牌
  2. 将两张牌交给任意角色（可以是自己，也可以分配给不同角色）
- **限制**：每受到1点伤害可发动一次
- **备注**：
  - 受到2点伤害可发动两次遗计
  - 交给的牌可以是摸到的牌，也可以是原有的手牌
  - 可以将两张牌分别交给两名不同角色

## FAQ/争议点

1. **Q：天妒能否获得被鬼才改变后的判定牌？**
   A：天妒获得的是最终生效的判定牌。如果被鬼才改变，则获得替代后的牌。

2. **Q：遗计能否将牌交给已死亡的角色？**
   A：不能。只能交给存活的角色。

3. **Q：遗计的两张牌能否都给自己？**
   A：可以。两张牌可以都给自己，也可以分配给不同角色。
