## 技能详解

### 界据守（被动技）

> **机制简述**：结束阶段翻面摸四牌并弃或用装备

**官方效果**：结束阶段，你可以翻面并摸四张牌，然后弃置一张手牌，若为装备牌，你改为使用之。

**实现要点**：

- 主动 use action：结束阶段发动 → 加翻面标签 + 摸 4 张 → 选一张手牌弃置/装备 → 标记已用
- 翻面标签触发下一回合跳过（机制与标版据守完全一致，before-hook 消费标签 + cancel 阶段）
- 弃置/装备交互：use execute 内嵌 请求回应（requestType='据守/弃牌'），玩家选一张手牌 → 装备牌则装备（含旧装备替换），否则弃置
- 内部标签/localVars/requestType 键名保持原前缀 '据守/xxx'

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：标版据守：翻面 + 摸 3 张 + 跳过下一回合（无弃牌/装备分支）。

界版据守：翻面 + 摸 **4** 张 + **弃一张手牌（装备则使用之）** + 仍跳过下一回合。新增摸牌数+1 与弃牌/装备分支。

### 解围（转化技）

> **机制简述**：装备当无懈，翻正面时可移场上牌

**官方效果**：你可以将装备区里的牌当【无懈可击】使用。当你从背面翻至正面时，你可以弃置一张牌，然后移动场上一张牌。

**实现要点**：

- 效果 ①（转化技，镜像看破/奇袭）：transform action——装备区装备牌 → 卸下（装备区→手牌）+ 移除装备技能 → 当作 → 影子无懈可击。作为 preceding 在 无懈可击.respond 之前执行
- 效果 ②（被动触发，镜像巧变移动）：after hook 挂「去标签」——检测 owner 的 '/翻面' 后缀标签被移除（=翻回正面）
- 效果 ② 流程：询问 → 选弃牌（手牌）→ 选源玩家 → 选源牌（pickTargetCard）→ 选目标玩家 → 移动场上牌（获得 atom，与巧变 moveFieldCard 一致）
- 效果 ① activeWhen 检测无懈可击广播窗口且有装备；不受自己回合限制
- 效果 ② 触发时机：据守消费翻面标签时（下一回合准备阶段）或被其他技能（界放逐）翻回正面时

**实现差异/边界**：实现与官方描述一致。
