# 界司马懿

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界司马懿 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 反馈 | 被动技 | 当你受到伤害后，你可以获得伤害来源的一张牌。 |
| 鬼才 | 主动技 | 当一名角色的判定牌生效前，你可以打出一张手牌或装备牌代替之。 |

## 技能详细规则

### 反馈

与原版相同，获得伤害来源一张牌（手牌或装备）。

### 鬼才

- **变化**：原版只能用手牌，界版可以用装备牌代替判定牌
- **效果**：用一张手牌或装备牌替换当前判定牌

## FAQ/争议点

1. **Q：界司马懿可以用装备区的牌改判定吗？**
   A：可以。装备牌也可以用来替换判定牌。
