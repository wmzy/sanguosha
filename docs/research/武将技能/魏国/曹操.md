# 曹操

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 曹操 |
| 势力 | 魏 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 奸雄 | 被动技 | 当你受到伤害后，你可以选择一项：①摸一张牌；②获得造成此伤害的牌。 |
| 护驾 | 主动技 | 主公技，当你需要使用或打出一张【闪】时，你可以令其他魏势力角色选择是否打出一张【闪】（视为由你使用或打出）。 |

## 技能详细规则

### 奸雄

- **触发时机**：每当你受到伤害后
- **发动条件**：造成此伤害的牌存在（无来源伤害则无牌可获得）
- **效果**：你可以选择一项：
  1. 摸一张牌
  2. 获得造成此伤害的牌
- **限制**：无次数限制
- **备注**：
  - 选 ②「获得牌」时，若伤害来源是【杀】/【决斗】/【南蛮入侵】/【万箭齐发】等实体牌，可以获得该牌
  - 选 ②「获得牌」时，若伤害是属性伤害（火【杀】/雷【杀】/【火攻】等），获得的是原始实体牌
  - 选 ②「获得牌」时，若伤害无来源（如【闪电】），则无牌可获得，只能选 ①
  - 选 ①「摸一张牌」则不获得牌（适用于无来源伤害场景）

### 护驾

- **触发时机**：当你需要使用或打出【闪】时
- **发动条件**：存活的其他魏势力角色
- **效果**：其他魏势力角色可以打出一张【闪】（视为由你使用或打出）
- **限制**：每回合无次数限制
- **备注**：主公技，仅当曹操为主公时可用

## FAQ/争议点

1. **Q：奸雄能否获得被【无懈可击】抵消的锦囊牌？**
   A：不能。被抵消的锦囊牌不会造成伤害，因此不触发奸雄。

2. **Q：奸雄能否获得【万箭齐发】？**
   A：可以。当【万箭齐发】对你造成伤害时，你可以获得该【万箭齐发】。

3. **Q：护驾能否被【无懈可击】抵消？**
   A：不能。护驾是打出【闪】，不是使用锦囊牌。
