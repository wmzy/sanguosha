# 曹操

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 曹操 |
| 势力 | 魏 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 奸雄 | 被动技 | 可以立即获得对你造成伤害的牌。 |
| 护驾 | 主动技 | 主公技，当你需要使用或打出一张【闪】时，你可以令其他魏势力角色选择是否打出一张【闪】（视为由你使用或打出）。 |

## 技能详细规则

### 奸雄

- **触发时机**：当你受到伤害后
- **发动条件**：造成伤害的牌（如果有的话）在弃牌堆中
- **效果**：你可以获得造成伤害的那张牌
- **限制**：无次数限制
- **备注**：
  - 如果伤害来源是【杀】，可以获得该【杀】
  - 如果伤害来源是【决斗】，可以获得该【决斗】
  - 如果伤害来源是【南蛮入侵】或【万箭齐发】，可以获得该锦囊牌
  - 如果伤害来源是技能（如【雷击】），则没有牌可以获取

### 护驾

- **触发时机**：当你需要使用或打出【闪】时
- **发动条件**：存活的其他魏势力角色
- **效果**：其他魏势力角色可以打出一张【闪】（视为由你使用或打出）
- **限制**：每回合无次数限制
- **备注**：主公技，仅当曹操为主公时可用

## FAQ/争议点

1. **Q：奸雄能否获得被【无懈可击】抵消的锦囊牌？**
   A：不能。被抵消的锦囊牌不会造成伤害，因此不触发奸雄。

2. **Q：奸雄能否获得【万箭齐发】？**
   A：可以。当【万箭齐发】对你造成伤害时，你可以获得该【万箭齐发】。

3. **Q：护驾能否被【无懈可击】抵消？**
   A：不能。护驾是打出【闪】，不是使用锦囊牌。
