## 技能详解

### 神速（主动技）

> **机制简述**：跳过阶段换取视为使用无距离限制的杀

**官方效果**：你可以选择至多三项：1.跳过判定阶段和摸牌阶段；2.跳过出牌阶段并弃置一张装备牌；3.跳过弃牌阶段并翻面。你每选择一项，你视为使用一张无距离限制的【杀】。

**实现要点**：
1. 选项1：阶段开始(判定) before-hook 询问发动 → 加「跳过摸牌」标签 + 虚拟杀 + cancel 判定阶段
2. 选项2：阶段开始(出牌) before-hook 询问发动 → 弃一张装备牌 + 虚拟杀 + cancel 出牌阶段
3. 虚拟杀无实体卡，直接走「指定目标→成为目标→检测有效性→询问闪→伤害/抵消」完整结算段，与 杀.use 结算一致
4. 虚拟杀不消耗手牌、不计入 `杀/quota`（规则视为出杀，但本实现简化为不占 quota——神速的杀发生在出牌阶段之前/之中，与正常出杀统计独立）
5. 跳过阶段手法同兵粮寸断/乐不思蜀：applyAtom(阶段结束, 当前阶段) 推进到下一阶段，再 cancel

**实现差异/边界**：
- **重大差异——选项3未实现**：官方明确为「至多三项」，第三项是「跳过弃牌阶段并翻面」；implNotes 自述仅实现「一至两项」，**第三项「跳过弃牌阶段并翻面」完全未实现**，故本武将实际只能选择前两项
- 其余两项（跳判定+摸牌、跳出牌+弃装备）与官方一致，虚拟杀机制亦与「视为使用无距离杀」语义吻合
