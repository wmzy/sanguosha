## 技能详解

### 界断粮（转化技）

> **机制简述**：黑色基本或装备牌当兵粮寸断

**官方效果**：你可以将一张黑色基本牌或黑色装备牌当【兵粮寸断】使用。若你本回合未造成过伤害，你使用【兵粮寸断】无距离限制。

**实现要点**：

- 主动技/转化：独立 use action（非沿用 兵粮寸断.use）。
- validate 校验黑色基本牌或装备牌 + 放松距离（目标手牌数 ≥ 徐晃手牌数 → 无距离限制；否则距离 ≤ 1）。
- execute 直接放置 name="兵粮寸断" 的延时锦囊到目标判定区。
- 判定/跳过摸牌阶段的效果由现有 兵粮寸断.ts 的 hooks 处理（只认 pendingTricks.name）。
- 无次数限制。

**实现差异/边界**：官方"黑色基本牌或装备牌"——实现一致（限基本+装备）；官方"本回合未造成过伤害则无距离限制"——实现采用的是旧版规则"目标手牌数 ≥ 自己则无距离限制"，与 OL 现行界版不同。请以官方描述为准评估。

**与标版差异**：标版断粮：黑色基本牌或装备牌当兵粮寸断使用（距离 1）。界版（OL 现行）：可转任意黑色牌（不限基本/装备）+"本回合未造成过伤害则无距离限制"。引擎实现的"黑色基本或装备+手牌数比较"是介于标版与界版之间的中间版本。

### 截辎（被动技）

> **机制简述**：他人跳摸牌后给标记或摸牌

**官方效果**：当一名角色跳过摸牌阶段后，你可以选择一名角色，若其手牌数全场最少且没有“辎”，其获得“辎”标记，否则其摸一张牌。有“辎”的角色于其摸牌阶段结束时移除“辎”，然后执行一个额外的摸牌阶段。

**实现要点**：

- 锁定技，检测策略：① 阶段开始(摸牌) after-hook：标记 normalDrawPhase=player（被跳过时不执行）。② 阶段结束(摸牌) after-hook：若 normalDrawPhase !== player → 该玩家跳过了摸牌阶段 → 若是其他角色且自己存活 → 摸一张牌。
- 捕获所有跳过摸牌阶段的情况（兵粮寸断/神速/界神速/巧变），且与 skipPhase 的具体实现无关。
- 锁定技，无条件触发，无需询问玩家。

**实现差异/边界**：官方"当一名角色跳过摸牌阶段后，你可以选择一名角色，若其手牌数全场最少且没有"辎"，其获得"辎"标记，否则其摸一张牌。有"辎"的角色于其摸牌阶段结束时移除"辎"，然后执行一个额外的摸牌阶段"——实现完全不同！实现仅"他人跳摸牌后徐晃摸一张牌"的极简版本，未实现"辎"标记系统、额外摸牌阶段、"全场最少手牌"判断等机制。请以官方描述为准，重新评估此技能。

**与标版差异**：OL 现行截辎是全新的"辎"标记机制，标版无此技能。
