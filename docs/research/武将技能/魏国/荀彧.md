# 荀彧

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 荀彧 |
| 势力 | 魏 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 驱虎 | 主动技 | 出牌阶段，你可以与一名角色拼点，若你赢，该角色对其攻击范围内另一名角色造成1点伤害；若你没赢，该角色对你造成1点伤害。 |
| 节命 | 被动技 | 当你受到1点伤害后，你可以令一名角色将手牌摸至X张（X为其体力上限且最多为5）。 |

## 技能详细规则

### 驱虎

- **触发时机**：出牌阶段
- **发动条件**：有手牌且目标有手牌（拼点）
- **效果**：
  1. 与一名角色拼点
  2. 若你赢，拼点角色对其攻击范围内另一名角色造成1点伤害
  3. 若你没赢，拼点角色对你造成1点伤害
- **限制**：每回合限一次
- **备注**：
  - 拼点赢后，由荀彧选择攻击范围内的一名目标接受伤害
  - 拼点输时无法选择，只能承受伤害
  - 这是强制伤害，不能防止

### 节命

- **触发时机**：受到1点伤害后
- **发动条件**：有可令其摸牌的角色
- **效果**：令一名角色将手牌补至X张（X=其体力上限，最多5张）
- **限制**：无次数限制
- **备注**：
  - 最适合给手牌少的队友补牌
  - 体力上限高的角色补牌更多
  - 可以给自己补牌

## FAQ/争议点

1. **Q：驱虎能否对没有攻击范围的角色发动？**
   A：若荀彧拼点赢，目标必须在其攻击范围内有角色，否则不能造成伤害。

2. **Q：节命能否在受到多点伤害时发动多次？**
   A：节命按受伤次数计算。受到1点伤害可发动一次，受到2点伤害可发动两次（如酒杀）。

3. **Q：驱虎是否消耗双方的手牌？**
   A：是。拼点双方各出一张手牌，拼点后弃置两张牌。
