## 技能详解

### 驱虎（主动技）

> **机制简述**：与高体力者拼点，赢则其伤人，没赢则其伤你

**官方效果**：出牌阶段限一次，你可以与体力值大于你的一名角色拼点：若你赢，其对其攻击范围内一名由你选择的角色造成1点伤害；若你没赢，其对你造成1点伤害。

**实现要点**：
1. 出牌阶段限一次（`player.vars['驱虎/usedThisTurn']` + 回合用量 atom 同步 view）
2. 目标须为体力值大于自己的角色
3. 拼点点数：A=1(最小), 2-10=面值, J=11, Q=12, K=13；严格大者赢，相等算「没赢」
4. 拼点双方各出一张手牌，拼点后两张牌进入弃牌堆
5. 赢：该角色对其攻击范围内一名由荀彧选择的角色造成1点伤害
6. 没赢：该角色对荀彧造成1点伤害
7. respond action 为所有玩家注册（目标需 respond 选拼点牌，validate 严格检查 pending requestType，非驱虎 pending 时拒绝）

**实现差异/边界**：实现与官方描述一致。

### 节命（被动技）

> **机制简述**：受伤后令角色将手牌摸至体力上限（至多5）

**官方效果**：当你受到1点伤害后，你可以令一名角色将手牌摸至X张（X为其体力上限且至多为5）。

**实现要点**：
1. 被动技，after hook 挂「造成伤害」
2. 触发条件：target=自己 + amount>0
3. X = `min(目标.maxHealth, 5)`（体力上限封顶 5）
4. 补牌数 = X - 目标.hand.length，≤0 则不摸（但技能仍算发动）
5. 目标可选任意存活角色（包括自己）
6. 按一次伤害一次触发（与遗计/反馈/放逐一致）

**实现差异/边界**：实现与官方描述一致。
