# 附录

## 技能类型说明

| 类型 | 说明 |
|------|------|
| 锁定技 | 满足条件自动发动，玩家无法选择是否发动 |
| 主动技 | 玩家可以选择是否发动，需要满足发动条件 |
| 被动技 | 满足条件自动触发，但玩家可以选择是否执行效果 |

## 常见关键词解释

| 关键词 | 说明 |
|--------|------|
| 判定 | 从牌堆顶翻开一张牌，根据花色和点数决定效果 |
| 濒死状态 | 体力值降至0或更低时进入的状态 |
| 回合 | 一个完整的回合包括：准备阶段、判定阶段、摸牌阶段、出牌阶段、弃牌阶段、结束阶段 |
| 距离 | 两名角色之间的座位间隔数，受武器和技能影响 |
| 攻击范围 | 武器牌决定的基础攻击范围，受技能影响 |
