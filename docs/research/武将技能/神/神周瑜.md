# 神周瑜

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神周瑜 |
| 势力 | 神 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 琴音 | 主动技 | 弃牌阶段结束时，若你于此阶段弃置了至少两张手牌，你可以令所有角色各失去1点体力或回复1点体力。每回合限一次。 |
| 业炎 | 限定技 | 出牌阶段，你可以选择一项：1.令至多三名角色各受到1点火焰伤害；2.令一名角色受到3点火焰伤害。然后你弃置四张花色各不相同的手牌并失去3点体力。 |

## 技能详细规则

### 琴音

- **效果**：弃牌阶段弃2+牌后，可令全场扣血或回血
- **备注**：强力AOE或群体回复

### 业炎

- **限定技**：整局游戏一次
- **效果**：大范围高额火焰伤害，代价是弃4花色手牌+失去3体力
