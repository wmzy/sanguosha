# 神赵云

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神赵云 |
| 势力 | 神 |
| 体力上限 | 2 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 绝境 | 锁定技 | 你的手牌上限+2；当你进入濒死状态时，你可以弃置两张牌，然后回复1点体力。 |
| 龙魂 | 主动技 | 你可以将一张同花色牌当桃/火杀/酒/闪使用或打出。每种花色的牌每回合限转化一次。 |

## 技能详细规则

### 绝境

- **效果**：手牌上限+2（2血武将手牌上限为4）
- **濒死自救**：弃2牌回复1血

### 龙魂

- **效果**：
  - 红桃→桃
  - 方块→火杀
  - 黑桃→酒
  - 梅花→闪
- **备注**：万能转化，极度灵活

## FAQ/争议点

1. **Q：神赵云只有2血，会不会太脆？**
   A：有绝境+龙魂的强力自保能力，实际生存能力很强。
