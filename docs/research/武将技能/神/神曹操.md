# 神曹操

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神曹操 |
| 势力 | 神 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 飞影 | 锁定技 | 其他角色计算与你的距离时，始终+1。 |
| 归心 | 被动技 | 每当你受到1点伤害后，每名其他角色各选择一项：1.将一张牌交给你；2.受到你造成的1点伤害。 |

## 技能详细规则

### 飞影

- **效果**：其他人到神曹操的距离+1（相当于装备了+1马）

### 归心

- **效果**：受伤后全场每人要交牌或受伤
- **备注**：受伤收益极高的卖血技，常被称为"吸全场"

## FAQ/争议点

1. **Q：归心能触发几次？**
   A：每受到1点伤害触发一次，如受到酒杀的2点伤害触发两次。
