# 神诸葛亮

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神诸葛亮 |
| 势力 | 神 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 七星 | 主动技 | 游戏开始时，你将牌堆顶七张牌扣置于武将牌上，称为"星"。你可以将一张"星"置入弃牌堆，然后摸一张牌。 |
| 狂风 | 主动技 | 结束阶段，你可以移去一张"星"，令一名角色获得"狂风"标记，其受到的火焰伤害+1。 |
| 大雾 | 主动技 | 结束阶段，你可以移去一张"星"，令一名角色获得"大雾"标记，防止其受到的所有非雷电伤害。 |

## 技能详细规则

### 七星

- **效果**：开局7星，每弃一星可摸一张牌（相当于存了7张备用牌）

### 狂风

- **效果**：标记角色受到火焰伤害+1
- **备注**：配合火攻、火杀等

### 大雾

- **效果**：标记角色免疫非雷电伤害
- **备注**：超强防御技能，可持续保护关键角色

## FAQ/争议点

1. **Q：大雾能防所有非雷电伤害吗？**
   A：是的。包括杀、锦囊、技能伤害等，只要不是雷电伤害都能防止。
