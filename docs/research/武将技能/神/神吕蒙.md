# 神吕蒙

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神吕蒙 |
| 势力 | 神 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 涉猎 | 主动技 | 摸牌阶段，你可以放弃摸牌，改为展示牌堆顶五张牌，然后获得其中任意数量的牌，其余置于牌堆底。 |
| 攻心 | 主动技 | 出牌阶段，你可以观看一名其他角色的手牌，并选择其中一张扣置于其武将牌上，称为"心"；若如此做，本回合其使用与"心"花色相同的牌时，你摸一张牌。 |

## 技能详细规则

### 涉猎

- **效果**：看5张拿想要的部分，剩余放底
- **备注**：顶级过牌技能

### 攻心

- **效果**：看对方手牌+标记一张，用同花色牌就摸牌
- **备注**：强力信息获取+控制
