# 神吕布

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神吕布 |
| 势力 | 神 |
| 体力上限 | 5 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 神威 | 锁定技 | 摸牌阶段，你多摸两张牌；你的手牌上限+2。 |
| 无双 | 锁定技 | 同原版无双——需两张闪才能抵消你的杀，决斗需两张杀。 |
| 神愤 | 限定技 | 出牌阶段，你可以弃置六枚"暴怒"标记，然后令所有其他角色各弃置装备区的所有牌，再弃置四张手牌，最后受到1点伤害。你翻面。 |

## 技能详细规则

### 神威

- **效果**：每回合摸4张，手牌上限+2
- **备注**：顶级过牌量

### 神愤

- **限定技**
- **效果**：全场炸装备+炸手牌+受伤害，代价是自己翻面
- **备注**：最强大的清场技能之一

## FAQ/争议点

1. **Q：神吕布的暴怒标记如何获得？**
   A：原版神吕布需要累积"暴怒"标记才能发动神愤。
