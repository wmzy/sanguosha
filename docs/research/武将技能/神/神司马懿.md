# 神司马懿

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神司马懿 |
| 势力 | 神 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 忍戒 | 被动技 | 当你受到伤害后或于弃牌阶段外弃牌后，你获得一枚"忍"标记。每回合限获得两枚。 |
| 拜印 | 觉醒技 | 准备阶段，若"忍"标记数≥4，你减1点体力上限，获得技能"极略"。 |
| 极略 | 主动技 | 你可以弃置一枚"忍"标记，发动下列一个技能：完杀、鬼才、放逐、集智、制衡、连破。每回合每项限一次。 |
| 连破 | 被动技 | 当你杀死一名角色后，你可以进行一个额外回合。 |

## 技能详细规则

### 忍戒

- **效果**：受伤或弃牌（非弃牌阶段）获得"忍"标记

### 拜印

- **觉醒**：4忍后觉醒，获得极略

### 极略

- **效果**：消耗忍标记使用六大技能
- **备注**：非常全面强大

### 连破

- **效果**：击杀后额外回合
