# 神关羽

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 神关羽 |
| 势力 | 神 |
| 体力上限 | 5 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 武神 | 锁定技 | 你的红桃手牌视为【杀】；你使用红桃【杀】无距离限制且无视防具。 |
| 武魂 | 锁定技 | 当你死亡时，你可以令一名其他角色进行判定，若非♠️，其获得"梦魇"标记；有"梦魇"标记的角色在其回合开始时需令你摸一张牌，且当其死亡时，你令其获得技能"武魂"。 |

## 技能详细规则

### 武神

- **效果**：红桃牌都是杀，且红桃杀无距离+破防
- **备注**：5血+所有红桃都是杀，输出稳定

### 武魂

- **效果**：死后诅咒一名角色
- **备注**：死亡威慑技能，极强
