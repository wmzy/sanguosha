# 太史慈

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 太史慈 |
| 势力 | 吴 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 天义 | 主动技 | 出牌阶段，你可以与一名角色拼点，若你赢，本回合你攻击范围无限、可额外使用一张【杀】、使用【杀】时可额外指定一个目标；若你没赢，你不能使用【杀】直到回合结束。 |

## 技能详细规则

### 天义

- **触发时机**：出牌阶段
- **发动条件**：有手牌且目标有手牌（拼点）
- **效果**：
  1. 与一名角色拼点
  2. 若你赢，本回合获得以下效果直到回合结束：
     - 攻击范围无限
     - 可额外使用一张【杀】
     - 使用【杀】时可额外指定一个目标
  3. 若你没赢，你不能使用【杀】直到回合结束
- **限制**：每回合限一次
- **备注**：
  - 若拼点赢，一杀可以打两个人，且不受距离限制
  - 配合多目标可以打出双杀四目标的效果
  - 拼点输了会损失一回合的输出能力

## FAQ/争议点

1. **Q：天义拼点赢后，能否使用三张【杀】？**
   A：可以额外使用一张【杀】，即本回合最多使用两张。

2. **Q：天义拼点赢后，使用【杀】能否额外指定两个目标？**
   A：只能额外指定一个目标，即一张杀可以打两个目标。

3. **Q：如果拼点双方点数相同，视为谁赢？**
   A：点数相同视为太史慈没有赢（没赢）。
