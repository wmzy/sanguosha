## 技能详解

### 天义（主动技）

> **机制简述**：拼点赢则杀次数+1且无视距离

**官方效果**：出牌阶段限一次，你可以拼点：若你赢，你本回合使用【杀】的次数+1、无距离限制且可以多指定一个目标；若你没赢，你本回合不能使用【杀】。

**实现要点**：

- 主动技 use action：出牌阶段限一次，对一名角色发起拼点。
- 拼点规则：A=1（最小），2-10=面值，J=11/Q=12/K=13；点数大者赢，相等算"没赢"；拼点双方各出一张手牌进入弃牌堆。
- 赢（三项效果，挂 turn.vars，回合结束自动清空）：① 攻击范围无限（distance.ts inAttackRange 据此放行）；② 注册 slashMaxProvider 贡献 +1；③ 杀.validate 不限目标数（配合①任意两名存活角色生效）。
- 输（turn.vars["天义/lost"]）：注册 slashBlocker 返回 true → canSlash=false → 杀.use validate 拒绝；连弩也禁杀。
- 每回合限一次：player.vars["天义/usedThisTurn"]。
- 目标需 respond 选拼点牌——respond action 为所有玩家注册（validate 严格校验 pending requestType）。

**实现差异/边界**：官方"赢则杀次数+1、无距离限制、可多指定一个目标"——实现为攻击范围无限+杀次数+1+杀不限目标数，效果等价；输则本回合不能用杀（含连弩），符合官方"不能使用杀"语义。
