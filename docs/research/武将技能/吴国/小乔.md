# 小乔

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 小乔 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 天香 | 主动技 | 当你受到伤害时，你可以弃置一张红桃手牌转移此伤害给任意一名其他角色，然后该角色摸X张牌（X为其已损失体力值）。 |
| 红颜 | 锁定技 | 你的黑桃牌均视为红桃牌。 |

## 技能详细规则

### 天香

- **触发时机**：受到伤害时（伤害结算前）
- **发动条件**：有红桃手牌，场上有其他存活角色
- **效果**：
  1. 弃置一张红桃手牌
  2. 将伤害转移给任意一名其他角色
  3. 该角色摸X张牌（X为其已损失体力值）
- **限制**：无次数限制
- **备注**：
  - 转移的是伤害本身，不是攻击，无法被防御
  - 被转移的角色先扣减体力再摸牌（可能先濒死再摸牌）
  - 处于连环状态时，属性伤害转移后若目标也横置，可再次传导触发天香
  - 不能转移给自己

### 红颜

- **触发时机**：任何涉及花色判断的时候
- **发动条件**：锁定技，自动生效
- **效果**：小乔的所有黑桃牌视为红桃牌
- **限制**：锁定技
- **备注**：
  - 包括手牌、装备牌、判定牌
  - 【闪电】判定对小乔无效（黑桃2-9视为红桃）
  - 红颜使小乔的天香发动条件更容易满足

## FAQ/争议点

1. **Q：天香能否转移属性伤害？**
   A：可以。并且如果目标处于横置状态，会触发连环传导。

2. **Q：天香转移伤害杀死反贼，谁获得奖励？**
   A：由伤害的源头（造成伤害的角色）获得奖励。

3. **Q：红颜是否影响【八卦阵】判定？**
   A：是的，小乔使用【八卦阵】判定时，黑桃牌视为红桃，可以生效。
