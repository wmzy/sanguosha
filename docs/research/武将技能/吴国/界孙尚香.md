# 界孙尚香

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界孙尚香 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 结姻 | 主动技 | 出牌阶段，你可以弃置一张手牌，令一名已受伤的其他男性角色回复1点体力。每回合限一次。 |
| 枭姬 | 被动技 | 当你失去一张装备区的牌时，你可以摸两张牌。 |

## 技能详细规则

### 结姻

- **变化**：从弃两张手牌改为弃一张手牌
- **备注**：消耗降低，辅助效率更高

### 枭姬

与原版相同，摸两张牌。
