# 孙尚香

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 孙尚香 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 结姻 | 主动技 | 出牌阶段，你可以弃置两张手牌，然后令一名已受伤的男性角色回复1点体力。 |
| 枭姬 | 被动技 | 当你失去一张装备区的牌时，你可以摸两张牌。 |

## 技能详细规则

### 结姻

- **触发时机**：出牌阶段
- **发动条件**：
  1. 有至少两张手牌
  2. 场上有已受伤的男性角色
- **效果**：
  1. 弃置两张手牌
  2. 令一名已受伤的男性角色回复1点体力
- **限制**：每回合限一次
- **备注**：
  - 只能选择已受伤的男性角色
  - 回复体力不能超过体力上限
  - 弃置的两张牌必须是手牌

### 枭姬

- **触发时机**：失去一张装备区的牌时
- **发动条件**：失去装备区的牌
- **效果**：摸两张牌
- **限制**：无次数限制
- **备注**：
  - 失去装备的方式包括被拆、被顺、自己替换等
  - 每失去一张装备牌触发一次
  - 可以与其他技能配合

## FAQ/争议点

1. **Q：结姻能否选择女性角色？**
   A：不能。只能选择男性角色。

2. **Q：结姻能否选择未受伤的角色？**
   A：不能。只能选择已受伤的角色。

3. **Q：枭姬能否在装备武器时触发？**
   A：不能。只有失去装备时才触发。
