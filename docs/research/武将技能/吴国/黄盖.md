# 黄盖

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 黄盖 |
| 势力 | 吴 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 苦肉 | 主动技 | 出牌阶段，你可以失去1点体力，然后摸两张牌。 |

## 技能详细规则

### 苦肉

- **触发时机**：出牌阶段
- **发动条件**：体力值大于0
- **效果**：
  1. 失去1点体力
  2. 摸两张牌
- **限制**：无次数限制
- **备注**：
  - 失去体力不是受到伤害，因此不触发与伤害相关的技能
  - 可以多次发动，但要注意体力值
  - 如果体力值降为0，进入濒死状态

## FAQ/争议点

1. **Q：苦肉失去的体力能否被【桃】恢复？**
   A：可以。失去体力后，可以使用【桃】恢复体力。

2. **Q：苦肉能否被【奸雄】等技能获得牌？**
   A：不能。苦肉不是造成伤害，不触发相关技能。

3. **Q：苦肉能否在装备【诸葛连弩】后多次发动？**
   A：可以。苦肉无次数限制。
