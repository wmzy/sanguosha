# 界陆逊

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界陆逊 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 谦逊 | 锁定技 | 你不能成为【顺手牵羊】和【乐不思蜀】的目标。 |
| 连营 | 被动技 | 当你失去最后的手牌时，你可以令最多X名角色各摸一张牌（X为你的体力值）。 |

## 技能详细规则

### 谦逊

与原版相同。

### 连营

- **变化**：从自己摸一张改为令最多X名角色摸一张牌（X=体力值）
- **备注**：变成了群体辅助技能，可以给队友补牌

## FAQ/争议点

1. **Q：连营最多能让几个人摸牌？**
   A：最多为陆逊当前体力值的人数（如3血可令3名角色摸牌）。
