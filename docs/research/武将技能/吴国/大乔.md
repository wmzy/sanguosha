# 大乔

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 大乔 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 国色 | 主动技 | 你可以将一张方块牌当【乐不思蜀】使用。 |
| 流离 | 主动技 | 当你成为【杀】的目标时，你可以弃置一张牌，将此【杀】转移给你攻击范围内的一名其他角色。 |

## 技能详细规则

### 国色

- **触发时机**：需要使用【乐不思蜀】时
- **发动条件**：有方块牌（手牌或装备区的牌）
- **效果**：将一张方块牌当【乐不思蜀】使用
- **限制**：无次数限制
- **备注**：
  - 可以使用装备区的方块牌
  - 使用后原牌进入弃牌堆
  - 当做【乐不思蜀】使用，可以被【无懈可击】抵消

### 流离

- **触发时机**：成为【杀】的目标时
- **发动条件**：有牌可以弃置，且攻击范围内有其他角色
- **效果**：
  1. 弃置一张牌
  2. 将【杀】转移给攻击范围内的一名其他角色
- **限制**：无次数限制
- **备注**：
  - 弃置的牌可以是手牌或装备牌
  - 转移后的【杀】目标改变，原目标不再是目标
  - 攻击范围由大乔的武器和距离计算决定

## FAQ/争议点

1. **Q：国色能否被【无懈可击】抵消？**
   A：可以。国色当做【乐不思蜀】使用，因此可以被【无懈可击】抵消。

2. **Q：流离能否转移给不在攻击范围内的角色？**
   A：不能。必须在大乔的攻击范围内。

3. **Q：流离能否在【杀】已经造成伤害后发动？**
   A：不能。必须在【杀】指定目标后、生效前发动。
