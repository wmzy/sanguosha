# 周瑜

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 周瑜 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 英姿 | 被动技 | 摸牌阶段，你可以额外摸一张牌。 |
| 反间 | 主动技 | 出牌阶段，你可以令一名其他角色选择一种花色，然后获得你的一张手牌并展示之，若此牌与所选花色不同，该角色受到1点伤害。 |

## 技能详细规则

### 英姿

- **触发时机**：摸牌阶段
- **发动条件**：无特殊条件
- **效果**：额外摸一张牌（正常摸2张，改为摸3张）
- **限制**：每回合限一次
- **备注**：
  - 英姿是主动发动的，可以选择不发动
  - 发动后摸3张牌

### 反间

- **触发时机**：出牌阶段
- **发动条件**：有手牌，且场上有其他存活角色
- **效果**：
  1. 选择一名其他角色
  2. 该角色选择一种花色
  3. 该角色获得你的一张手牌并展示
  4. 若牌的花色与所选花色不同，该角色受到1点伤害
- **限制**：每回合限一次
- **备注**：
  - 由目标角色选择花色
  - 获得的牌是随机的（不能指定）
  - 如果花色相同则无事发生

## FAQ/争议点

1. **Q：反间获得的牌能否被【反馈】等技能获得？**
   A：反间后牌在目标角色手中，可以被其他技能影响。

2. **Q：反间能否选择没有手牌的角色？**
   A：可以。反间是由周瑜给出手牌，不依赖目标角色的手牌。

3. **Q：英姿能否与【裸衣】叠加？**
   A：可以。英姿增加摸牌数，裸衣减少摸牌数，两者独立计算。
