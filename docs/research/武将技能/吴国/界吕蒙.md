## 技能详解

### 克己（主动技）

> **机制简述**：本回合未用打出杀则可跳过弃牌阶段

**官方效果**：若你未于本回合出牌阶段使用或打出过【杀】，你可以跳过弃牌阶段。

**实现要点**：

- 移动牌 after-hook：监听"杀"从自己手牌进入处理区（同时覆盖"使用"杀.use 与"打出"杀.respond）。命中后置 turn.vars["克己/playedSlash"]=true（随回合结束 atom 自动清空）。
- 阶段开始 before-hook：弃牌阶段开始时，若本回合未出杀，询问是否发动克己；确认 → 触发阶段结束(弃牌) 推进到回合结束，并 cancel 当前阶段开始(弃牌)。
- 回合管理的 阶段结束 after-hook 检测到 state.phase 已偏离"弃牌" → 跳过弃牌 pending 创建。
- respond action：玩家确认/取消 → 写 localVars["克己/confirmed"]。

**实现差异/边界**：官方"若你未于本回合出牌阶段使用或打出过杀"——实现检查的是"本回合"（不限出牌阶段，整个回合任一阶段出过杀都计数）。严格按官方应是"出牌阶段内"，实现更严格（在出牌阶段外出杀也会阻止克己），属保守实现，不会破坏游戏平衡。

**与标版差异**：标版克己：跳过出牌阶段则本回合手牌上限为 0（即可不弃牌）。界版（OL 现行）：未于出牌阶段使用/打出杀则可跳过弃牌阶段——更直接、更易触发。

### 勤学（觉醒技）

> **机制简述**：手牌比体力多2及以上时减体力上限并回血或摸牌获攻心

**官方效果**：觉醒技，准备阶段或结束阶段，若你的手牌数比体力值多2或更多，你减少1点体力上限，回复1点体力或摸两张牌，然后获得“<font href='304'>攻心</font>”。

**实现要点**：

- 觉醒技，阶段开始（准备 或 回合结束）after-hook（ownerId 是吕蒙）。
- 条件：手牌数 - 体力值 ≥ 2，且未觉醒。
- 强制流程：① 减 1 体力上限；② 二选一（摸两张牌 / 回复 1 点体力）；③ 永久获得"攻心"。
- 觉醒标记：player.vars["勤学/awakened"]（永久 vars，整局一次，不被「回合结束」atom 清理）。
- 模式参考志继/若愚，但挂在「阶段开始」而非「回合开始」（触发点是"阶段"）。

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：OL 现行勤学（2022 加强版）即为此形式，标版无对应技能。

### 博图（被动技）

> **机制简述**：每轮限次，回合后弃牌堆含四花色则执行额外回合

**官方效果**：每轮限X次（X为存活角色数且至多为3），回合结束后，若本回合置入弃牌堆的牌包含四种花色，你可以执行一个额外回合。

**实现要点**：

- 花色追踪：回合开始 after-hook 记录弃牌堆基线长度 turn.vars["博图/base"]；回合结束 before-hook 扫描 discardPile.slice(base)，检查是否含 ♠♥♣♦ 四花色。
- 用快照而非逐 atom hook：覆盖所有置入弃牌堆的路径（弃置/拼点/移动牌/判定），基线随 turn.vars 自动清空重置。
- 每轮限 X 次：player.vars["博图/lastRound"] + ["博图/count"]；轮次变化时重置 count；X = min(存活角色数, 3)。
- 额外回合：参考界凿险/放权——回合结束 before-hook：cancel 正常回合结束 → 手动 clearPerTurnState → 亲自启动 ownerId 额外回合；before 先于 回合管理 after-hook(findNextAlive)，cancel 后下家不启动。
- 主动询问：满足条件时询问吕蒙是否执行额外回合；确认才 cancel+启动。
- 额外回合结束：count 已+1，达 X 或不满足花色 → 不再 cancel → 正常推进座次。

**实现差异/边界**：实现与官方描述一致。clearPerTurnState 复刻回合结束 atom 的 per-turn 清理。

**与标版差异**：OL 现行博图（2020 新增），标版无此技能。
