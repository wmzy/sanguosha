## 技能详细规则

### 克己

- **官方效果**：若你未于本回合出牌阶段使用或打出过【杀】，你可以跳过弃牌阶段。
- **实现要点**（`src/engine/skills/克己.ts`，复用标版）：
  - 移动牌 after-hook：监听【杀】从自己手牌进入处理区，同时覆盖「使用」（杀.use 主动出杀）与「打出」（杀.respond 应对决斗/南蛮入侵等），命中后置 `turn.vars['克己/playedSlash']=true`（随「回合结束」atom 自动清空）。
  - 阶段开始 before-hook：弃牌阶段开始时，若本回合未出杀，询问是否发动克己；确认 → 触发「阶段结束」推进到回合结束并 cancel 当前「阶段开始」（与兵粮寸断/乐不思蜀"跳过阶段"同构）。
  - respond action：玩家确认/取消 → 写 `localVars['克己/confirmed']`。
- **与标版差异**：复用标版，无差异。OL 界吕蒙的克己与标吕蒙完全相同，故引擎直接复用标 `克己.ts`，不注册界克己。

### 勤学

- **官方效果**：觉醒技，准备阶段或结束阶段，若你的手牌数比体力值多2或更多，你减少1点体力上限，回复1点体力或摸两张牌，然后获得"攻心"。
- **实现要点**（`src/engine/skills/勤学.ts`）：
  - 时机：「阶段开始」atom 的 `phase='准备'` 或 `phase='回合结束'` after-hook（对应吕蒙的准备阶段 / 结束阶段两个触发点）。
  - 条件：手牌数 − 体力值 ≥ 2，且未觉醒。
  - 效果：减 1 点体力上限 → 二选一（摸两张牌 / 回复 1 点体力）→ 永久获得衍生技"攻心"。
  - 觉醒标记：`player.vars['勤学/awakened']`（永久 vars，整局一次，不被「回合结束」atom 清理）。
  - 模式参考志继/若愚（觉醒 after-hook + 二选一 + 设上限 + 添加技能），但挂在「阶段开始」而非「回合开始」。
- **与标版差异**：勤学为界吕蒙新增觉醒技（标吕蒙仅有克己，无勤学）。

### 博图

- **官方效果**：每轮限X次（X为存活角色数且至多为3），回合结束后，若本回合置入弃牌堆的牌包含四种花色，你可以执行一个额外回合。
- **实现要点**（`src/engine/skills/博图.ts`）：
  - 花色追踪：回合开始 after-hook 记录弃牌堆基线长度 `turn.vars['博图/base']`；回合结束 before-hook 扫描 `discardPile.slice(base)` 检查是否含 ♠♥♣♦ 四花色。基线随「回合结束」atom 自动随 `turn.vars` 清空重置。
  - 每轮限 X 次：`player.vars['博图/lastRound']` + `['博图/count']`，轮次变化时重置 count；X = min(存活角色数, 3)。
  - 额外回合：参考界凿险/放权——回合结束 before-hook cancel 正常回合结束 → 手动 `clearPerTurnState`（cancel 后 apply 不执行，需手动复刻「回合结束」atom 的 per-turn 清理）→ 亲自启动 ownerId 的额外回合。额外回合结束后 count 已 +1，达 X 或不满足花色则不再 cancel，正常推进座次。
  - 满足条件时**主动询问**吕蒙是否执行额外回合；确认才 cancel + 启动；死亡/不满足条件则放行正常回合结束。
  - 「置入弃牌堆」= 本回合新进弃牌堆的牌，含使用/打出/弃置/拼点/被拆等所有路径（故用快照而非逐 atom hook）。
- **与标版差异**：博图为界吕蒙（OL 2020 新增）一般技（标吕蒙无此项）。

### 攻心（勤学衍生技）

- **官方效果**：出牌阶段限一次，你可以观看一名其他角色的手牌，然后你可以展示其中一张♥牌并选择一项：1.弃置此牌；2.将此牌置于牌堆顶。
- **实现要点**（`src/engine/skills/攻心.ts`，由勤学觉醒后动态挂载，不在 character 文件静态注册）：
  - 主动技，出牌阶段限一次（`攻心/usedThisTurn`，once-per-turn 工具）。
  - 流程：选一名有手牌的其他角色 → `pickProcessingCard` 列出目标手牌中的♥牌给吕蒙（观看 = 仅吕蒙可见，信息分级按 target 分发，他人只见"观看"事件）→ 吕蒙可选一张♥（展示）或放弃（pass/超时=不展示）→ 若选了♥：展示（复用「展示」atom 公开牌面）→ 二选一：弃置（目标弃此牌）/ 置牌堆顶（此牌→牌堆顶，牌堆顶 = deck 末尾，摸牌从末尾抽）。
  - 目标无♥牌时：观看后直接结束（无可展示牌）。
- **与标版差异**：攻心为勤学衍生技（标吕蒙无此项）；觉醒前不拥有，觉醒后永久获得。

## FAQ / 备注

- **攻心是衍生技**：OL hero/306 技能名列表仅列克己/勤学/博图三项，攻心由勤学觉醒效果"然后获得'攻心'"动态获得，故 character 文件 `skills` 数组不含攻心，攻心技能模块由勤学在觉醒时通过「添加技能」atom 挂载。
- **克己"使用或打出过【杀】"**：通过监听【杀】从手牌进入处理区统一判定，覆盖主动使用与应对决斗/南蛮入侵等打出两种情形。
- **勤学两个触发点**：准备阶段或结束阶段，任一阶段满足手牌 − 体力 ≥ 2 即可触发觉醒（整局一次）。
- **博图"本回合置入弃牌堆"**：以回合开始时的弃牌堆长度为基线，统计本回合新增入堆的牌（任意路径），需同时包含四种花色才可发动。
- **博图额外回合**：执行额外回合会消耗本轮额度（count+1），达 X = min(存活角色数, 3) 次后本轮不再可发动。
