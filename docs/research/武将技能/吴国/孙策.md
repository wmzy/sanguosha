# 孙策

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 孙策 |
| 势力 | 吴 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 激昂 | 被动技 | 每当你使用（指定目标后）或被使用（成为目标后）一张【决斗】或红色的【杀】时，你可以摸一张牌。 |
| 魂姿 | 觉醒技 | 回合开始阶段，若你的体力为1，你须减1点体力上限，并永久获得技能"英姿"和"英魂"。 |
| 制霸 | 主公技 | 其他吴势力角色的出牌阶段，可与你进行一次拼点，若该角色没赢，你可以获得双方拼点的牌；觉醒后你可拒绝此拼点。每回合限一次。 |

## 技能详细规则

### 激昂

- **触发时机**：使用/成为目标时（【决斗】或红色【杀】）
- **发动条件**：无特殊条件
- **效果**：摸一张牌
- **限制**：无次数限制
- **备注**：
  - 无论孙策是使用方还是目标方，都可触发
  - 红色杀包括普通红杀和火杀

### 魂姿（觉醒技）

- **触发时机**：回合开始阶段
- **发动条件**：体力为1
- **效果**：
  1. 减1点体力上限（4→3）
  2. 永久获得技能"英姿"和"英魂"
- **限制**：觉醒技

**英姿**（觉醒后获得）：摸牌阶段，你可以额外摸一张牌（共摸3张）。

**英魂**（觉醒后获得）：同孙坚的英魂——回合开始阶段，若你已受伤，可令一名角色执行选项。

### 制霸

- **触发时机**：其他吴势力角色的出牌阶段
- **发动条件**：孙策为主公
- **效果**：
  1. 对方与你拼点
  2. 若对方没赢，你可以获得双方的拼点牌
  3. 觉醒后你可以拒绝拼点
- **限制**：每名角色每回合限一次

## FAQ/争议点

1. **Q：魂姿觉醒后，体力上限从4降到3，当前体力为1，会怎样？**
   A：当前体力保持1，体力上限变为3。

2. **Q：激昂在一回合内能触发多少次？**
   A：无次数限制。只要满足条件（使用/被使用决斗或红杀）即可触发。

3. **Q：觉醒后的孙策有多强？**
   A：觉醒后孙策拥有英姿（摸3牌）+英魂（控制），配合激昂的过牌，是顶级强度的武将。
