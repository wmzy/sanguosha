# 陆逊

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 陆逊 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 谦逊 | 锁定技 | 你不能成为【顺手牵羊】和【乐不思蜀】的目标。 |
| 连营 | 被动技 | 当你失去最后的手牌时，你可以摸一张牌。 |

## 技能详细规则

### 谦逊

- **触发时机**：成为【顺手牵羊】或【乐不思蜀】的目标时
- **发动条件**：无特殊条件（锁定技，自动生效）
- **效果**：不能成为【顺手牵羊】和【乐不思蜀】的目标
- **限制**：锁定技，自动生效
- **备注**：
  - 已经被指定为目标后，再失去手牌不影响该锦囊的结算
  - 谦逊只防止成为目标，不影响已经生效的效果

### 连营

- **触发时机**：失去最后的手牌时
- **发动条件**：手牌数变为0
- **效果**：摸一张牌
- **限制**：无次数限制
- **备注**：
  - 失去手牌的方式包括使用、打出、弃置等
  - 摸牌后如果手牌仍为0，可以再次触发连营
  - 理论上可以无限连营（直到牌堆耗尽）

## FAQ/争议点

1. **Q：谦逊能否防止被【借刀杀人】指定？**
   A：谦逊只防止【顺手牵羊】和【乐不思蜀】，不影响【借刀杀人】。

2. **Q：连营能否在弃牌阶段触发？**
   A：可以。弃牌阶段失去手牌也可以触发连营。

3. **Q：连营能否被【集智】等技能触发？**
   A：连营是独立的技能，与其他技能无关。
