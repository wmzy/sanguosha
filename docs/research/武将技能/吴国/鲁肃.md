# 鲁肃

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 鲁肃 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 好施 | 主动技 | 摸牌阶段，你可以额外摸两张牌，若此时你的手牌数超过五张，你必须将一半（向下取整）的手牌交给除你外手牌数最少的一名角色。 |
| 缔盟 | 主动技 | 出牌阶段，你可以选择两名其他角色，弃置等同于这两名角色手牌数差的牌，然后交换他们的手牌。每回合限一次。 |

## 技能详细规则

### 好施

- **触发时机**：摸牌阶段
- **发动条件**：无特殊条件
- **效果**：
  1. 额外摸两张牌（共摸4张）
  2. 若手牌超过5张，须将一半手牌（向下取整）交给手牌数最少的其他角色
- **限制**：每回合限一次
- **备注**：
  - 这个技能可以快速过牌
  - 给牌效果强制触发（如果手牌＞5张必须给）
  - 优先找到手牌最少的人

### 缔盟

- **触发时机**：出牌阶段
- **发动条件**：场上有至少两名其他角色
- **效果**：
  1. 选择两名其他角色
  2. 弃置等同于两人手牌数差的牌
  3. 交换两人的手牌
- **限制**：每回合限一次
- **备注**：
  - 鲁肃弃牌，不是目标弃牌
  - 可以把队友的烂牌和敌人的好牌交换
  - 是游戏中控制力最强的技能之一

## FAQ/争议点

1. **Q：缔盟交换手牌后，两人的手牌数变成什么？**
   A：手牌数不变，只是交换手牌内容。

2. **Q：好施给牌的"一半"怎么计算？**
   A：向下取整。如手牌数为7张，则给3张（7÷2=3.5→3）。

3. **Q：缔盟能否选择自己？**
   A：不能。只能选择其他两名角色。
