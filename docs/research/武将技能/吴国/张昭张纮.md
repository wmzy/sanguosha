# 张昭张纮

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 张昭张纮 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 直谏 | 主动技 | 出牌阶段，你可以将手牌中的一张装备牌置于一名其他角色的装备区（不得替换原装备），然后摸一张牌。 |
| 固政 | 被动技 | 其他角色的弃牌阶段结束时，你可以将弃牌堆中一张该角色弃置的牌返回其手牌，然后获得其余弃牌。 |

## 技能详细规则

### 直谏

- **触发时机**：出牌阶段
- **发动条件**：手牌中有装备牌，目标装备区有空位（不能替换已有装备）
- **效果**：
  1. 将一张装备牌置于一名其他角色的装备区
  2. 摸一张牌
- **限制**：无次数限制
- **备注**：
  - 不能替换目标已有的装备（但可以装在不同位置）
  - 可以给队友关键的装备（武器、防具、马）
  - 配合枭姬（孙尚香）可以产生联动

### 固政

- **触发时机**：其他角色的弃牌阶段结束时
- **发动条件**：该角色在本阶段弃置过牌
- **效果**：
  1. 将该角色弃置的一张牌返回其手牌
  2. 获得本阶段该角色弃置的其余所有牌
- **限制**：每回合限一次
- **备注**：
  - 每个角色的弃牌阶段结束时都可以发动
  - 相当于把队友弃的牌回收很大一部分
  - 可以帮队友保存关键牌

## FAQ/争议点

1. **Q：直谏能否给装备牌替换自己已有的装备？**
   A：不能。只能将牌放入空装备栏，不能替换已有装备。

2. **Q：固政是否每次都能获得所有弃牌？**
   A：先还给该角色一张，然后获得剩余的牌。

3. **Q：固政能否对自己发动？**
   A：不能。固政指定"其他角色"。
