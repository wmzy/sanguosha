# 孙权

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 孙权 |
| 势力 | 吴 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 制衡 | 主动技 | 出牌阶段，你可以弃置任意数量的牌，然后摸等量的牌。 |
| 救援 | 被动技 | 主公技，其他吴势力角色对濒死状态的你使用【桃】时，该角色额外回复1点体力。 |

## 技能详细规则

### 制衡

- **触发时机**：出牌阶段
- **发动条件**：有牌可以弃置
- **效果**：
  1. 弃置任意数量的牌（手牌或装备牌）
  2. 摸等量的牌
- **限制**：每回合限一次
- **备注**：
  - 可以弃置0张牌（但这样没有意义）
  - 弃置的牌进入弃牌堆
  - 摸的牌来自牌堆顶

### 救援

- **触发时机**：其他吴势力角色对濒死状态的孙权使用【桃】时
- **发动条件**：孙权处于濒死状态，其他吴势力角色使用【桃】
- **效果**：使用【桃】的吴势力角色额外回复1点体力
- **限制**：无次数限制
- **备注**：
  - 主公技，仅当孙权为主公时生效
  - 额外回复体力的是使用【桃】的角色，不是孙权

## FAQ/争议点

1. **Q：制衡能否弃置判定区的牌？**
   A：不能。只能弃置手牌和装备区的牌。

2. **Q：救援能否对自己使用？**
   A：不能。救援是其他吴势力角色对孙权使用【桃】时触发。

3. **Q：制衡能否在装备武器后立即制衡掉？**
   A：可以。装备和制衡都是出牌阶段的合法操作。
