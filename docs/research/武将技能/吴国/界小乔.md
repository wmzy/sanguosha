## 技能详细规则
### 界红颜（技能文件引用名）
- **官方效果**：锁定技，你的黑桃牌和黑桃判定牌视为红桃牌。若你的装备区里有红桃牌，你的手牌上限等于体力上限。
- **实现要点**：
  - 判定 `before` hook：自己执行判定时，将牌堆顶黑桃牌的花色改为红桃（在翻开前改花色，确保展示为红桃）。
  - 手牌上限覆盖型 provider（`registerHandLimitProvider`）：装备区存在红桃牌时返回体力上限；无红桃装备时不返回（走默认公式 = 当前体力），不强行压低上限。
- **与标版差异**：标版红颜为「锁定技，你的黑桃牌均视为红桃牌」。界版额外：① 将花色转换范围扩展到自己的黑桃判定牌；② 新增"装备区有红桃牌时手牌上限 = 体力上限"。

### 界天香（技能文件引用名）
- **官方效果**：当你受到伤害时，你可以弃置一张红桃牌防止之并选择一项，令一名其他角色：1.受到伤害来源的1点伤害并摸X张牌（X为其已损失体力值且至多为5）；2.失去1点体力并获得你弃置的牌。
- **实现要点**：
  - 时机：造成伤害 `before` hook，target=自己且 amount>0 时触发。
  - 合法弃牌：红桃牌（手牌或装备区）；拥有红颜/界红颜时黑桃也视为红桃。
  - 流程：弃一张红桃牌 + 选一名其他角色 → 二选一（confirm）→ 执行选项 → `cancel` 原伤害 atom（"防止之"=界小乔不受伤）。
  - 选项①：所选角色受原伤害来源造成的 1 点伤害；然后摸 X 张牌（X = maxHealth − health，至多 5）。
  - 选项②：所选角色失去 1 点体力（非伤害，不触发反馈/奸雄）；然后获得本次弃置的红桃牌（弃牌堆→其手牌）。
  - 目标不能是界小乔自己（官方："令一名其他角色"）。
- **与标版差异**：标版天香为「转移伤害」（目标代替界小乔承受原伤害）。界版改为「防止自身受伤 + 令他人二选一（受来源1点伤害摸X / 失1体力获弃牌）」，语义完全不同。

### 飘零
- **官方效果**：结束阶段，你可以判定，若结果为红桃，你将判定牌置于牌堆顶或交给一名角色，若该角色是你，你弃置一张牌。
- **实现要点**：
  - 时机：阶段开始 `after` hook（回合结束 = 结束阶段）。
  - 流程：confirm 是否判定 → 判定{player=自己, judgeType='飘零'}（红颜/界红颜会把黑桃判定牌改为红桃）→ 读弃牌堆顶判定牌。
  - 非红桃：无效果。红桃：confirm 是否交给一名角色。
    - 确认 → 选目标（choosePlayer，含自己）→ 移动牌（弃牌堆→目标手牌）；若目标==自己 → 弃一张手牌。
    - 不确认 → 移动牌（弃牌堆→牌堆顶）。
- **与标版差异**：标版小乔无此技能，飘零为界版新增。

## FAQ / 备注
- 界天香的"防止之"是真正的伤害防止（cancel 原 atom），界小乔本身不受伤、不触发受伤后置事件。
- 界天香选项①的摸牌数 X 按"已损失体力值"（maxHealth − health）计算且至多为 5，在伤害结算后计算。
- 界红颜的手牌上限 provider 是覆盖型：仅在装备区有红桃牌时返回体力上限，无红桃装备时不干预默认计算。
- 飘零交给自己时会额外弃一张手牌；交给他人或置于牌堆顶则无需弃牌。
