## 技能详解

### 界好施（主动技）

> **机制简述**：多摸两牌，超5则分给手牌最少者

**官方效果**：摸牌阶段，你可以多摸两张牌，然后若你的手牌数大于5，你将一半的手牌（向下取整）交给手牌最少的一名其他角色，然后直到你的下回合开始，当你成为【杀】或普通锦囊牌的目标后，其可以交给你一张手牌。

**实现要点**：

- 摸牌逻辑（镜像标好施）：before hook +2 张 / after hook >5 分半给最少者
- 被动追踪：给牌后记 `localVars[受益者/被动激活]`；回合开始 after hook 清
- 被动触发——杀：成为目标 after hook（atom.target===owner 且卡是杀）
- 被动触发——普通锦囊：请求回应 before hook（cancelTarget===owner 且卡非延时锦囊）——全体/单目标锦囊在结算前均经 询问无懈可击（cancelTarget=目标座次）
- respond 注册到每个座次：鲁肃处理 confirm/target/give；其他座次处理 passiveGive
- 命名：文件名/loader key/character name = '界好施'（避标版冲突）；内部 Skill.name = '好施'

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：标版好施：摸牌阶段多摸 2 张，若手牌>5 分一半给手牌最少者。

界版好施：在标版基础上**新增跨回合被动**——「然后直到你的下回合开始，当你成为【杀】或普通锦囊牌的目标后，其（受益者）可以交给你一张手牌」。

### 界缔盟（主动技）

> **机制简述**：令两人交换手牌后弃差值

**官方效果**：出牌阶段限一次，你可以令两名其他角色交换手牌（两者手牌数之差不大于你的牌数量），若如此做，出牌阶段结束时，你弃置X张牌（X为这两名角色手牌数之差）。

**实现要点**：

- use action：选两名角色 → 校验差值 ≤ 鲁肃牌数（手牌+装备）→ 立即交换手牌 → 记录 X
- 阶段结束 after hook：出牌阶段结束时，鲁肃弃 X 张牌（X=交换时记录的手牌数差）
- 出牌阶段限一次
- 命名：文件名/loader key/character name = '界缔盟'；内部 Skill.name = '缔盟'

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：标版缔盟：选两名其他角色 → 弃 X 张牌（X=手牌数差）→ 交换手牌（先弃后换，弃在前）。

界版缔盟三处实质差异：① **前置条件**：两者手牌数差 ≤ 鲁肃的牌数（手牌+装备）；② **结算顺序**：先交换手牌，后弃牌（标版是先弃后换）；③ **弃牌时机**：出牌阶段结束时延迟弃（标版是交换前立即弃）。
