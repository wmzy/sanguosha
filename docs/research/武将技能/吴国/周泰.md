## 技能详解

### 不屈（锁定技）

> **机制简述**：濒死置创点数不同则回至1血

**官方效果**：锁定技，当你处于濒死状态时，你将牌堆顶的一张牌置于你的武将牌上，称为“创”，若此牌点数与其他“创”均不同，你回复至1点体力，否则移去此牌。若你的武将牌上有“创”，你的手牌上限为“创”的数量。

**实现要点**：

- 锁定技；时机：`陷入濒死` after-hook（target === ownerId）
- 流程：`置创牌{player=自己}` 翻牌堆顶一张 → atom 判定点数重复并写入 `localVars['不屈/重复']`
- 不重复 → 设 `localVars['不屈/存活']=ownerId`，runDyingFlow 据此跳过求桃+击杀，**以 0 体力存活**
- 重复 → 不设标记，runDyingFlow 继续求桃流程；无人救则击杀
- 「创」牌累积于 `player.vars['不屈/创牌']`

**实现差异/边界**：**重大差异**：官方描述为「若点数与其他创均不同，你**回复至 1 点体力**」，实现为「**以 0 体力存活**」（不回复体力）。实现采用的是标准三国杀规则（备注与任务描述均为 0 体力存活），与官方 hero 页文本不一致。

**未实现**：官方「若你的武将牌上有『创』，你的手牌上限为『创』的数量」——此标版周泰未实现手牌上限规则（界周泰中已实现）。
