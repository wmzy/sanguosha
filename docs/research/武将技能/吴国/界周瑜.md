# 界周瑜

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界周瑜 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 英姿 | 主动技 | 摸牌阶段，你可以额外摸一张牌。摸牌阶段结束时，若你本阶段摸牌数大于1，你本回合手牌上限+1。 |
| 反间 | 主动技 | 出牌阶段，你可以令一名其他角色选择一种花色，然后该角色获得你的一张手牌并展示，若此牌花色与所选不同，你对其造成1点伤害。 |

## 技能详细规则

### 英姿

- **变化**：增加手牌上限+1的额外效果
- **效果**：摸3张牌后，本回合手牌上限+1

### 反间

- **变化**：伤害来源从"牌与花色不同"改为周瑜主动造成伤害
- **备注**：反间造成的伤害来源是周瑜，可触发周瑜的技能

## FAQ/争议点

1. **Q：英姿的手牌上限+1持续多久？**
   A：仅持续到本回合结束。
