# 吕蒙

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 吕蒙 |
| 势力 | 吴 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 克己 | 主动技 | 出牌阶段，若你未使用或打出过【杀】，你可以跳过弃牌阶段。 |

## 技能详细规则

### 克己

- **触发时机**：出牌阶段结束时
- **发动条件**：本回合未使用或打出过【杀】
- **效果**：跳过弃牌阶段
- **限制**：每回合限一次
- **备注**：
  - 必须在整个出牌阶段都没有使用或打出【杀】
  - 跳过弃牌阶段意味着不需要弃牌到手牌上限
  - 使用【闪】或锦囊牌不影响克己的发动

## FAQ/争议点

1. **Q：克己能否在回合外发动？**
   A：不能。克己是出牌阶段的技能。

2. **Q：打出【闪】是否影响克己？**
   A：不影响。克己只关心是否使用或打出过【杀】。

3. **Q：克己能否在装备【诸葛连弩】后发动？**
   A：只有在本回合没有使用【杀】的情况下才能发动。
