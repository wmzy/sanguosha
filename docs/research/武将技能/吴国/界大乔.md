# 界大乔

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界大乔 |
| 势力 | 吴 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 国色 | 主动技 | 你可以将一张方块牌当【乐不思蜀】使用。若你以此法使用的【乐不思蜀】被弃置，你可以将之收回手牌。 |
| 流离 | 主动技 | 当你成为【杀】的目标时，你可以弃置一张牌，将此【杀】转移给你攻击范围内的一名其他角色。 |

## 技能详细规则

### 国色

- **变化**：从方块牌改为方块手牌/装备均可；结算是若乐被弃可回收手牌
- **备注**：回收效果大大减少乐不思蜀的消耗

### 流离

与原版相同。

## FAQ/争议点

1. **Q：国色的回收效果何时触发？**
   A：当乐不思蜀因弃置、被拆等方式进入弃牌堆时，可收回手牌。
