## 技能详细规则
### 流离（复用标版技能文件）
- **官方效果**：当你成为【杀】的目标时，你可以弃置一张牌并将此【杀】转移给你攻击范围内的一名其他角色。
- **实现要点**：
  - 时机：成为目标 `after` hook（结算阶段），修改杀帧的 `resolvedTargets`。
  - respond action 处理 `流离/confirm` 与 `流离/chooseTarget` 两步询问。
  - 合法目标：自己攻击范围内（`inAttackRange`）的一名其他角色。
- **与标版差异**：复用标版，无差异（character 文件直接引用 `../skills/流离`）。

### 界国色（技能文件引用名）
- **官方效果**：出牌阶段限一次，你可以选择一项，然后摸一张牌：1.将一张方片牌当【乐不思蜀】使用；2.弃置一张方片牌和场上一张【乐不思蜀】。
- **实现要点**：
  - 主动 `use` action，validate：出牌阶段 + 本回合未使用过（限一次，`markOncePerTurn`）+ 手中有方片牌。
  - execute：询问选项（confirm，`界国色/option`）→ 按选项询问牌/目标 → 执行 → 摸一张牌（"然后摸一张牌"是两选项共同效果）。
  - 合法牌：方片（♦）牌，手牌或装备区均可。
  - 选项①（use）：方片牌当【乐不思蜀】使用；延时锦囊转化复用国色/断粮的独立 use 模型，目标距离≤1、不能对自己；后续判定/跳过出牌/无懈抵消由 `乐不思蜀.ts` 的 hooks 处理（只认 trick.name）。
  - 选项②（remove）：弃置一张方片牌（弃置 atom）+ 弃置场上任意一张【乐不思蜀】（移除延时锦囊 atom，位于任意角色判定区均可）。
  - 选项②需场上存在【乐不思蜀】（`anyLeOnField`）才能选择。
- **与标版差异**：标版国色为「你可以将一张方块牌当【乐不思蜀】使用」（无次数限制、单一效果）。界版：① 改为每回合限一次；② 二选一（新增"弃方片+弃场上乐"）；③ 两选项均附带"摸一张牌"。

## FAQ / 备注
- 界大乔 character 文件 skills 数组顺序为 [界国色, 流离]，hero 页逐字顺序为 [流离, 国色]，本表按 hero 页顺序排列。
- 选项①当乐使用时，距离限制为 ≤1 且不能对自己（与标版国色/断粮一致）。
- 选项②的"场上一张【乐不思蜀】"可位于任意角色判定区，不限于自己或目标。
