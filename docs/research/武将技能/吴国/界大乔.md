## 技能详解

### 界国色（主动技）

> **机制简述**：出牌阶段方片当乐不思蜀或弃方片拆之，并摸牌

**官方效果**：出牌阶段限一次，你可以选择一项，然后摸一张牌：1.将一张方片牌当【乐不思蜀】使用；2.弃置一张方片牌和场上一张【乐不思蜀】。

**实现要点**：
- 主动 use action + 多步请求回应：use action validate（出牌阶段 + 限一次 + 有方片牌）→ execute：markOncePerTurn → 询问选项（confirm）→ 按选项询问牌/目标 → 执行 → 摸一张牌。
- **选项 ①**（方片当乐使用）：方片牌（手牌或装备区；距离 ≤1，不能对自己）。延时锦囊转化复用国色/断粮的独立 use 模型：原卡 id 永驻 cardMap，判定/跳过出牌/无懈抵消由 `乐不思蜀.ts` 的 hooks 处理（只认 trick.name）。
- **选项 ②**（弃方片+弃场上乐）：弃置一张方片牌（手牌或装备区）+ 弃置场上任意一张乐不思蜀（任意角色判定区）。选项 ② 用「弃置」atom + 「移除延时锦囊」atom。
- "然后摸一张牌"是两选项共同的效果（选完一项后必摸一张）。

**实现差异/边界**：实现与官方描述一致。

### 流离（被动技）

> **机制简述**：成为杀目标时弃牌将杀转给攻击范围内他人

**官方效果**：当你成为【杀】的目标时，你可以弃置一张牌并将此【杀】转移给你攻击范围内的一名其他角色。

**实现要点**：
- 时机：「成为目标」after hook（结算阶段），修改杀帧的 `resolvedTargets`。
- 弃一张牌 → 将此杀转移给攻击范围内的一名其他角色。
- 与标版大乔流离共享实现机制。

**实现差异/边界**：实现与官方描述一致。

**与标版差异**：标版国色仅"将一张方块牌当【乐不思蜀】使用"（无限次）；界版改为限一次/回合，二选一（当乐使用 或 弃方片拆场上乐）+ 必摸一张牌。流离界版与标版机制一致。
