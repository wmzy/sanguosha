# 庞统

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 庞统 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 连环 | 主动技 | 你可以将一张梅花手牌当【铁索连环】使用或重铸。 |
| 涅槃 | 限定技 | 当你处于濒死状态时，你可以弃置所有牌和判定区的牌，重置武将牌，摸三张牌并回复至3点体力。 |

## 技能详细规则

### 连环

- **触发时机**：出牌阶段
- **发动条件**：有梅花手牌
- **效果**：
  - 将梅花手牌当【铁索连环】使用
  - 或重铸（弃掉梅花牌摸一张牌）
- **限制**：无次数限制
- **备注**：
  - 铁索连环可横置或重置角色
  - 可以连环敌人配合属性伤害打出高额输出
  - 也可以连环队友和自己触发某些技能

### 涅槃

- **触发时机**：处于濒死状态时
- **发动条件**：无特殊条件
- **效果**：
  1. 弃置所有牌（手牌、装备牌）和判定区的牌
  2. 重置武将牌（解除连环状态）
  3. 摸三张牌
  4. 体力回复至3点
- **限制**：限定技（整局游戏仅一次）
- **备注**：
  - 涅盘后体力回复至3点，与濒死前体力值无关
  - 涅盘后所有技能状态重置
  - 限定技，整局游戏只能发动一次

## FAQ/争议点

1. **Q：涅盘发动后能否保留装备区的牌？**
   A：不能。需要弃置所有牌（包括装备牌）。

2. **Q：涅盘是否相当于"第二条命"？**
   A：是的。濒死时可以原地复活至3体力，但只能用一次。

3. **Q：连环能否对自己使用？**
   A：可以。铁索连环可以对任意角色使用，包括自己。
