# 界关羽

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界关羽 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 武圣 | 主动技 | 你可以将一张红色牌当【杀】使用或打出。 |
| 义绝 | 主动技 | 出牌阶段，你可以弃置一张牌，然后令一名其他角色展示所有手牌，你选择其中一张，其弃置此牌，然后若其弃置的牌不为♥，你摸一张牌。 |

## 技能详细规则

### 武圣

- **变化**：从红色手牌改为红色牌（包括装备）

### 义绝

- **新技能**（取代原版无额外技能）
- **效果**：
  1. 弃置一张牌
  2. 令一名角色展示全部手牌
  3. 你选择其中一张，目标弃置之
  4. 若弃置牌不是♥，你摸一张牌

## FAQ/争议点

1. **Q：义绝能否看对方手牌？**
   A：可以。这是强力的信息获取+拆牌技能。
