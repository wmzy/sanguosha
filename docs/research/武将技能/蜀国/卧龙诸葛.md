## 技能详解

### 八阵（锁定技）

> **机制简述**：无防具时视为装备八卦阵

**官方效果**：锁定技，若你的装备区里没有防具牌，你视为装备着【八卦阵】。

**实现要点**：
- 复用 `八卦阵.ts` 的「询问闪」before-hook 逻辑（询问是否发动 → 判定 → 红色放虚拟闪 + cancel 主询问闪），仅在 owner 无防具时生效。
- 无防具检查：防具 slot 名为 `'防具'`，`equipment['防具']` 为空即无防具。
- 区别：八卦阵是装备防具技（随装备加载/卸载）；八阵是角色锁定技（常驻）。一旦装备真实防具（仁王盾/藤甲/八卦阵等）八阵失效，卸下后自动恢复。

**实现差异/边界**：实现与官方描述一致。

### 火计（转化技）

> **机制简述**：红色手牌当火攻使用

**官方效果**：你可以将一张红色手牌当【火攻】使用。

**实现要点**：
- 组合 action 模型（镜像武圣）：前端选红牌 → 选目标（有手牌的其他角色） → 提交 `preceding=[火计.transform]` + 主 `action=火攻.use`。
- 后端 dispatch 先执行 `火计.transform`（创建影子火攻），再 `火攻.use` validate 看到"火攻"通过。
- 火攻技能零感知火计——它看到的永远是 cardMap 里的"火攻"。

**实现差异/边界**：实现与官方描述一致（限红色**手牌**，符合官方字面）。

### 看破（转化技）

> **机制简述**：黑色手牌当无懈可击使用

**官方效果**：你可以将一张黑色手牌当【无懈可击】使用。

**实现要点**：
- 组合 action 模型（镜像武圣但主 action 是 respond）：无懈可击窗口期间选黑牌 → 提交 `preceding=[看破.transform]` + 主 `action=无懈可击.respond`（影子 cardId）。
- 后端 dispatch 先执行 `看破.transform`（创建影子无懈可击），再 `无懈可击.respond` validate 检查 `requestType='无懈可击'` 且 `card.name==='无懈可击'` 通过。
- 无懈可击技能零感知看破。
- 看破不受自己回合限制（无懈可击任意时机可打）；`activeWhen` 检测无懈可击广播窗口（`requestType='无懈可击'`, `target<0`）且有黑牌。

**实现差异/边界**：实现与官方描述一致（限黑色**手牌**，符合官方字面）。
