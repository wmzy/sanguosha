# 卧龙诸葛

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 卧龙诸葛 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 八阵 | 锁定技 | 当你没有装备防具时，始终视为你装备着【八卦阵】。 |
| 火计 | 主动技 | 你可以将一张红色手牌当【火攻】使用。 |
| 看破 | 主动技 | 你可以将一张黑色手牌当【无懈可击】使用。 |

## 技能详细规则

### 八阵

- **触发时机**：需要使用【闪】时（判定）
- **发动条件**：没有装备防具
- **效果**：视为装备着【八卦阵】
- **限制**：锁定技，自动生效
- **备注**：
  - 一旦装备了真实的防具（如仁王盾、藤甲），八阵失效
  - 八阵的判定结果与其他八卦阵相同

### 火计

- **触发时机**：出牌阶段
- **发动条件**：有红色手牌
- **效果**：将一张红色手牌当【火攻】使用
- **限制**：无次数限制
- **备注**：
  - 火攻需要目标展示一张手牌
  - 可多次发动

### 看破

- **触发时机**：需要使用【无懈可击】时
- **发动条件**：有黑色手牌
- **效果**：将一张黑色手牌当【无懈可击】使用
- **限制**：无次数限制
- **备注**：
  - 可以无懈任何需要无懈的锦囊
  - 是最强的大规模反无懈技能

## FAQ/争议点

1. **Q：卧龙诸葛装备了其他防具后，能否再使用八阵？**
   A：不能。八阵仅在无防具时生效。

2. **Q：火计能否在一回合内使用多次？**
   A：可以，无次数限制。

3. **Q：看破使用的无懈可击是否计入使用次数？**
   A：正常计入，与其他无懈可击相同。
