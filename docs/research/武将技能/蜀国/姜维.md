# 姜维

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 姜维 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 挑衅 | 主动技 | 出牌阶段，你可以指定一名使用【杀】能攻击到你的角色，该角色需对你使用一张【杀】，否则你弃其一张牌。每回合限一次。 |
| 志继 | 觉醒技 | 回合开始阶段，若你没有手牌，你须回复1点体力或摸两张牌，然后减1点体力上限，并永久获得技能"观星"。 |

## 技能详细规则

### 挑衅

- **触发时机**：出牌阶段
- **发动条件**：目标使用【杀】能攻击到你
- **效果**：
  1. 指定一名角色
  2. 该角色需对你使用一张【杀】
  3. 若其不如此做，你弃其一张牌
- **限制**：每回合限一次
- **备注**：
  - 无论对方出不出杀，姜维都不亏
  - 对方出杀则消耗其一张杀
  - 对方不出杀则被姜维拆牌
  - 可以配合空城的诸葛亮（不能成为杀的目标）

### 志继（觉醒技）

- **触发时机**：回合开始阶段
- **发动条件**：没有手牌
- **效果**：
  1. 选择：回复1点体力，或摸两张牌
  2. 减1点体力上限
  3. 永久获得技能"观星"
- **限制**：觉醒技，触发一次后永久改变

### 观星（觉醒后获得）

与诸葛亮的观星相同：准备阶段，观看牌堆顶X张牌（X为存活角色数，最多5张），然后任意调整顺序。

## FAQ/争议点

1. **Q：挑衅能否选择姜维自己？**
   A：不能。必须选择能攻击到姜维的其他角色。

2. **Q：志继觉醒后，观星和挑衅是否可以同时使用？**
   A：可以。观星在准备阶段，挑衅在出牌阶段，互不影响。

3. **Q：志继觉醒时能否同时回复体力加摸牌？**
   A：不能。只能二选一。
