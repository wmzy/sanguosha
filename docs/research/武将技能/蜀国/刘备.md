# 刘备

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 刘备 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 仁德 | 主动技 | 出牌阶段，你可以将任意数量的手牌交给其他角色，若你给出的牌不少于两张，你回复1点体力。 |
| 激将 | 主动技 | 主公技，当你需要使用或打出一张【杀】时，你可以令其他蜀势力角色选择是否打出一张【杀】（视为由你使用或打出）。 |

## 技能详细规则

### 仁德

- **触发时机**：出牌阶段
- **发动条件**：有手牌
- **效果**：
  1. 将至少一张手牌交给其他角色
  2. 若你本回合以此法交给其他角色的手牌数累计不少于两张，回复1点体力
- **限制**：出牌阶段限一次
- **备注**：
  - 必须给出至少一张手牌（不能给 0 张）
  - 可以给同一名角色多张牌
  - 可以给不同角色牌
  - 回复体力的条件是本回合累计给出不少于两张牌
  - 回复体力的体力值不能超过体力上限

### 激将

- **触发时机**：需要使用或打出【杀】时
- **发动条件**：存活的其他蜀势力角色
- **效果**：其他蜀势力角色可以打出一张【杀】（视为由你使用或打出）
- **限制**：每回合无次数限制
- **备注**：主公技，仅当刘备为主公时可用

## FAQ/争议点

1. **Q：仁德能否给装备区的牌？**
   A：不能。仁德只能给出手牌。

2. **Q：仁德给出一张牌后，再给一张牌，能否回复体力？**
   A：可以。只要本回合累计给出不少于两张牌即可。

3. **Q：激将能否被【无懈可击】抵消？**
   A：激将是打出【杀】，不是使用锦囊牌，因此不能被【无懈可击】抵消。
