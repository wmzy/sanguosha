# 界黄月英

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界黄月英 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 集智 | 被动技 | 当你使用一张锦囊牌时，你可以展示牌堆顶一张牌，若为基本牌，你弃置此牌或将其交给一名角色；若不为基本牌，你获得之。 |
| 奇才 | 锁定技 | 你使用锦囊牌无距离限制。其他角色不能弃置你装备区里的防具牌。 |

## 技能详细规则

### 集智

- **变化**：改为展示牌堆顶牌，基本牌可交人，锦囊牌获得
- **备注**：增加了分配能力

### 奇才

- **变化**：额外增加保护防具的效果

## FAQ/争议点

1. **Q：集智展示的基本牌可以给自己吗？**
   A：可以交给任意一名角色（包括自己）。
