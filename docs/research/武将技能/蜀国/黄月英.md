# 黄月英

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 黄月英 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 集智 | 被动技 | 每当你使用一张非延时锦囊牌时，你可以摸一张牌。 |
| 奇才 | 锁定技 | 你使用任何锦囊牌无距离限制。 |

## 技能详细规则

### 集智

- **触发时机**：使用非延时锦囊牌时
- **发动条件**：无特殊条件
- **效果**：摸一张牌
- **限制**：无次数限制
- **备注**：
  - 非延时锦囊牌包括：【过河拆桥】、【顺手牵羊】、【无中生有】、【无懈可击】、【借刀杀人】、【桃园结义】、【五谷丰登】、【南蛮入侵】、【万箭齐发】、【决斗】等
  - 不包括延时锦囊牌：【乐不思蜀】、【兵粮寸断】、【闪电】

### 奇才

- **触发时机**：使用锦囊牌时
- **发动条件**：无特殊条件（锁定技，自动生效）
- **效果**：使用任何锦囊牌无距离限制
- **限制**：锁定技，自动生效
- **备注**：
  - 包括所有锦囊牌（延时和非延时）
  - 【顺手牵羊】等受距离限制的锦囊牌也可以对任意角色使用

## FAQ/争议点

1. **Q：集智能否在使用【无懈可击】时触发？**
   A：可以。【无懈可击】是非延时锦囊牌。

2. **Q：奇才能否使【借刀杀人】无距离限制？**
   A：可以。但【借刀杀人】还需要考虑武器距离。

3. **Q：集智能否与其他摸牌效果叠加？**
   A：可以。集智是独立的摸牌效果。
