# 界诸葛亮

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界诸葛亮 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 观星 | 主动技 | 准备阶段，你可以观看牌堆顶的X张牌（X为存活角色数且至多为5），然后以任意顺序置于牌堆顶或牌堆底。若你观星数≥3，你可以将其中任意数量的牌置于牌堆底，其余置于牌堆顶。 |
| 空城 | 锁定技 | 当你没有手牌时，你不能成为【杀】或【决斗】的目标。当你使用的【杀】被【闪】抵消后，你可以摸一张牌。 |

## 技能详细规则

### 观星

- **变化**：观星数≥3时，可先移到底再放到顶，控制力增强

### 空城

- **变化**：新增效果——杀被闪抵消后摸一张牌

## FAQ/争议点

1. **Q：界诸葛亮杀被闪抵消后摸牌，破空城吗？**
   A：破空城。摸牌后就有手牌了。
