# 魏延

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 魏延 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 狂骨 | 锁定技 | 当你对距离1以内的一名角色造成1点伤害时，你回复1点体力。 |

## 技能详细规则

### 狂骨

- **触发时机**：对距离1以内的角色造成伤害时
- **发动条件**：无特殊条件（锁定技）
- **效果**：回复1点体力
- **限制**：锁定技，每造成1点伤害回复1点
- **备注**：
  - 距离1以内包括距离为0（自己）和1的角色
  - 如果对自己造成伤害（如火攻自己），也可以触发狂骨
  - 回复体力可以超过体力上限吗？不能
  - 配合AOE或属性伤害可以多次触发

## FAQ/争议点

1. **Q：魏延装备-1马时，攻击范围如何计算？**
   A：-1马使魏延与所有角色的距离减少1，相当于可以触发狂骨的距离增大。

2. **Q：魏延通过【南蛮入侵】对多名角色造成伤害能否多次触发？**
   A：可以。每造成1点伤害触发一次狂骨。

3. **Q：魏延被小乔天香转移伤害时，若对自己造成伤害，能否触发狂骨？**
   A：可以。对自己造成伤害时距离为0，满足条件。
