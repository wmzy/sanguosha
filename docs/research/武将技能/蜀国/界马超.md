# 界马超

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 界马超 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 马术 | 锁定技 | 你计算与其他角色的距离时始终-1。 |
| 铁骑 | 被动技 | 当你使用【杀】指定一名角色为目标后，你可以进行判定，若结果为红色，此【杀】不可被【闪】响应；若结果为黑色，你摸一张牌且此【杀】不可被【闪】抵消。 |

## 技能详细规则

### 铁骑

- **变化**：黑色判定改版——改版后红黑都不可用闪响应，且黑色还摸牌
- **效果**：
  - 判定为红色：杀不能被闪响应
  - 判定为黑色：杀不能被闪响应，且马超摸一张牌
- **备注**：无论什么结果都有收益

## FAQ/争议点

1. **Q：铁骑判定后能否被鬼才改判？**
   A：可以。判定牌生效前可被改判。
