# 祝融

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 祝融 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 女 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 巨象 | 锁定技 | 【南蛮入侵】对你无效；若其他角色使用的【南蛮入侵】在结算完时进入弃牌堆，你立即获得它。 |
| 烈刃 | 被动技 | 每当你使用【杀】造成伤害后，可与受伤害的角色拼点：若你赢，你获得对方的一张牌。 |

## 技能详细规则

### 巨象

- **触发时机**：1.南蛮入侵结算时；2.南蛮入侵进入弃牌堆时
- **发动条件**：锁定技，自动生效
- **效果**：
  - 祝融免疫【南蛮入侵】的伤害
  - 其他角色使用的【南蛮入侵】结算后进入弃牌堆时，祝融立即获得之
- **限制**：锁定技
- **备注**：
  - 只能获得"使用"的南蛮入侵（打出、弃置等不触发）
  - 可以获得敌方的南蛮入侵，达到反复使用的效果

### 烈刃

- **触发时机**：使用【杀】造成伤害后
- **发动条件**：受伤害角色存活
- **效果**：
  1. 与受伤害角色拼点
  2. 若你赢，获得对方一张牌（手牌或装备区）
- **限制**：无次数限制
- **备注**：
  - 拼点赢可以获得对方任意一张牌
  - 拼点没赢则无事发生

## FAQ/争议点

1. **Q：巨象获得的南蛮入侵可以在本回合使用吗？**
   A：可以。获得后立即进入手牌。

2. **Q：烈刃拼点赢后，获得对方哪张牌由谁选择？**
   A：由祝融选择获得对方手牌还是装备区的牌（随机获得手牌，或选择装备区的一张牌）。
