# 刘禅

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 刘禅 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 享乐 | 锁定技 | 当其他角色使用【杀】指定你为目标时，需额外弃置一张基本牌，否则该【杀】对你无效。 |
| 放权 | 主动技 | 你可以跳过出牌阶段，然后在回合结束时弃置一张手牌，令一名其他角色进行一个额外回合。 |
| 若愚 | 主公技 | 觉醒技，回合开始阶段，若你的体力是全场最少的（或之一），你须增加1点体力上限并回复1点体力，然后永久获得技能"激将"。 |

## 技能详细规则

### 享乐

- **触发时机**：成为【杀】的目标时
- **发动条件**：锁定技，自动生效
- **效果**：使用【杀】的角色必须额外弃一张基本牌，否则杀无效
- **限制**：锁定技
- **备注**：
  - 基本牌包括【杀】、【闪】、【桃】
  - 可以有效消耗对手的基本牌
  - 如果对方没有基本牌可弃，杀无效

### 放权

- **触发时机**：出牌阶段开始时
- **发动条件**：无特殊条件
- **效果**：
  1. 跳过出牌阶段
  2. 回合结束时弃一张手牌
  3. 令一名其他角色进行一个额外回合
- **限制**：每回合限一次
- **备注**：
  - 刘禅放弃自己的出牌阶段，让队友多一个回合
  - 适合配合高输出队友
  - 是强大的辅助技能

### 若愚

- **触发时机**：回合开始阶段
- **发动条件**：刘禅为主公，且体力为全场最少（或之一）
- **效果**：
  1. 增加1点体力上限
  2. 回复1点体力
  3. 永久获得技能"激将"
- **限制**：觉醒技

## FAQ/争议点

1. **Q：放权给的角色进行额外回合时，判定区的牌如何处理？**
   A：额外回合同样要进行判定阶段。

2. **Q：享乐弃置的牌由刘禅选择吗？**
   A：由使用【杀】的角色选择弃哪张基本牌。

3. **Q：放权能否跳过出牌阶段的同时发动其他出牌阶段技能？**
   A：不能。跳过出牌阶段意味着不能进行任何出牌阶段的操作。
