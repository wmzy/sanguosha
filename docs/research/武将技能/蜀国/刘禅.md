## 技能详解

### 享乐（锁定技）

> **机制简述**：成杀目标时对方弃基本否则无效

**官方效果**：锁定技，当你成为一名角色使用【杀】的目标后，除非其弃置一张基本牌，否则令此【杀】对你无效。

**实现要点**：

- 锁定技，before-hook 挂在「检测有效性」（与仁王盾同时机）。
- 触发条件：target=刘禅 且 source≠刘禅 且 当前结算牌是杀。
- source 无基本牌 → cancel（杀对刘禅无效）；source 有基本牌 → 请求回应（"享乐/discard", target=source）让 source 选一张基本牌弃置。
- 弃了 → pass（杀有效）；没弃/超时 → cancel（杀无效）。
- 基本牌：type==="基本牌"（杀/闪/桃）；锁定技无限次。
- respond 需为所有玩家注册（source 是其他玩家）。

**实现差异/边界**：实现与官方描述一致。与仁王盾同 atom 天然互斥——cancel 后后续 hook 与杀结算跳过。

### 放权（主动技）

> **机制简述**：跳出牌阶段令他人额外回合

**官方效果**：你可以跳过出牌阶段，然后本回合结束时，你可以弃置一张手牌并令一名其他角色执行一个额外的回合。

**实现要点**：

- before hook 挂在「阶段开始」(出牌)：询问是否发动；发动则跳过出牌阶段 + 设 放权/active 标记。
- before hook 挂在「回合结束」处理两种情况：① 刘禅回合结束 + active：弃一张手牌 → 选目标 → cancel 本回合结束 → 手动 clearPerTurnState → 亲自启动目标的额外回合。② 额外目标回合结束：cancel → 手动清理 → 亲自启动正常下家（刘禅的正常下家）。
- 额外回合机制完全在放权技能内部实现，不修改 回合管理.ts；cancel 后需手动复刻 回合结束 的 per-turn 清理（clearPerTurnState）。
- 跳过出牌阶段手法同神速/巧变：applyAtom(阶段结束, 出牌) 推进到弃牌，再 cancel。

**实现差异/边界**：实现与官方描述一致。额外回合机制参考据守"亲自推进回合"先例。

### 若愚（觉醒技）

> **机制简述**：主公体力最低时加上限获激将

**官方效果**：主公技，觉醒技，准备阶段，若你是体力值最小的角色，你增加1点体力上限，回复1点体力，然后获得“<font href='32'>激将</font>”。

**实现要点**：

- 觉醒主公技，after-hook 挂在「回合开始」（ownerId 是主公 ownerId===0 + 体力全场最少或之一 + 未觉醒）。
- 强制触发（无选择）：① 增 1 体力上限（设 maxHealth+1）；② 回复 1 点体力（此时上限已+1，可回复）；③ 永久获得"激将"（添加技能）。
- 关键：增上限必须在回复前（先增上限再回复，否则回复被旧上限 clamp）。
- 觉醒标记：player.vars["若愚/awakened"]（永久，整局一次）。

**实现差异/边界**：实现与官方描述一致。主公判定 ownerId===0（参考激将/护驾）。
