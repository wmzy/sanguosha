## 技能详解

### 观星（主动技）

> **机制简述**：准备阶段观看牌堆顶X张并以任意顺序置顶或底

**官方效果**：准备阶段，你可以观看牌堆顶的X张牌（X为存活角色数且至多为5），然后以任意顺序置于牌堆顶或牌堆底。

**实现要点**：

- 主动技；时机：阶段开始(准备) after-hook（参考洛神）
- X = min(全场角色数 state.players.length, 5)
- 询问是否发动 → 询问排列，玩家回应 { top:[...], bottom:[...] }（top[0]=最先摸到的牌=牌堆顶）
- 构建 newDeck：`[...bottom, ...middle, ...top.reverse()]`，applyAtom(整理牌堆)
- 牌堆方向约定（与 摸牌 atom 一致）：deck[len-1]=牌堆顶（最先摸）

**实现差异/边界**：实现与官方描述一致。

### 空城（锁定技）

> **机制简述**：无手牌时不能成为杀或决斗的目标

**官方效果**：锁定技，若你没有手牌，你不能成为【杀】或【决斗】的目标。

**实现要点**：

- 锁定技；before-hook 挂「成为目标」（结算阶段第一个 atom）
- target===owner 且 owner 无手牌，且当前结算的牌是 杀 或 决斗 → cancel 该目标
- 杀.use / 决斗.use 检测 成为目标 返回 false 后跳过该目标的结算
- 判定「杀/决斗」：优先看 atom.cardId 对应卡牌名；无 cardId（虚拟杀等）时回退看栈顶帧 skillId

**实现差异/边界**：实现与官方描述一致。

**规则备注**：已被杀指定为目标后再失去手牌，不影响该杀的结算——本实现因挂在「成为目标」（结算阶段入口）天然满足。
