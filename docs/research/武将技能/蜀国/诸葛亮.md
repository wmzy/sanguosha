# 诸葛亮

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 诸葛亮 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 观星 | 主动技 | 准备阶段，你可以观看牌堆顶的X张牌（X为存活角色数且最多为5），然后将其中任意数量的牌置于牌堆顶，其余置于牌堆底。 |
| 空城 | 锁定技 | 当你没有手牌时，你不能成为【杀】或【决斗】的目标。 |

## 技能详细规则

### 观星

- **触发时机**：准备阶段
- **发动条件**：无特殊条件
- **效果**：
  1. 观看牌堆顶的X张牌（X为存活角色数，最多5张）
  2. 将任意数量的牌置于牌堆顶（顺序自定）
  3. 其余牌置于牌堆底
- **限制**：每回合限一次
- **备注**：
  - 观看的牌数等于存活角色数，但最多5张
  - 可以全部放在牌堆顶或牌堆底
  - 放在牌堆顶的牌顺序由你决定

### 空城

- **触发时机**：成为【杀】或【决斗】的目标时
- **发动条件**：没有手牌
- **效果**：不能成为【杀】或【决斗】的目标
- **限制**：锁定技，自动生效
- **备注**：
  - 已经被【杀】指定为目标后，再失去手牌不影响该【杀】的结算
  - 空城状态下，【决斗】不能指定诸葛亮为目标

## FAQ/争议点

1. **Q：观星能否看到自己的判定牌？**
   A：不能。观星看的是牌堆顶的牌，判定牌也是从牌堆顶翻开，但两者独立。

2. **Q：空城状态下能否被【南蛮入侵】或【万箭齐发】影响？**
   A：可以。空城只防止成为【杀】和【决斗】的目标。

3. **Q：观星能否将所有牌都放在牌堆底？**
   A：可以。但这样会导致接下来的角色摸到你看过牌之后的牌。
