# 诸葛亮

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 诸葛亮 |
| 势力 | 蜀 |
| 体力上限 | 3 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 观星 | 主动技 | 准备阶段开始时，你可以观看牌堆顶的X张牌（X为全场角色数且至多为5），并改变其中任意数量的牌的顺序，将其余的牌置于牌堆底。 |
| 空城 | 锁定技 | 若你没有手牌，你不是【杀】和【决斗】的合法目标。 |

## 技能详细规则

### 观星

- **触发时机**：准备阶段开始时
- **发动条件**：无特殊条件
- **效果**：
  1. 观看牌堆顶的X张牌（X = **全场角色数**，且至多为5）
  2. 改变其中任意数量的牌的顺序（置于牌堆顶）
  3. 其余的牌置于牌堆底
- **限制**：准备阶段限一次
- **备注**：
  - X 是「全场角色数」，不是「存活角色数」
  - 可以全部放在牌堆顶或全部放在牌堆底
  - 放在牌堆顶的牌顺序由你决定

### 空城

- **触发时机**：持续生效
- **发动条件**：没有手牌
- **效果**：你不是【杀】和【决斗】的合法目标
- **限制**：锁定技，自动生效
- **备注**：
  - 「合法目标」含义：不能成为【杀】或【决斗】的目标
  - 已经被【杀】指定为目标后，再失去手牌不影响该【杀】的结算

## FAQ/争议点

1. **Q：观星能否看到自己的判定牌？**
   A：不能。观星看的是牌堆顶的牌，判定牌也是从牌堆顶翻开，但两者独立。

2. **Q：空城状态下能否被【南蛮入侵】或【万箭齐发】影响？**
   A：可以。空城只防止成为【杀】和【决斗】的目标。

3. **Q：观星能否将所有牌都放在牌堆底？**
   A：可以。但这样会导致接下来的角色摸到你看过牌之后的牌。
