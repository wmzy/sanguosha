## 技能详解

### 享乐（锁定技）

> **机制简述**：成杀目标时对方需弃基本牌否则无效

**官方效果**：锁定技，当你成为一名角色使用【杀】的目标后，除非其弃置一张基本牌，否则令此【杀】对你无效。

**实现要点**：

- 锁定技；时机：杀的「检测有效性」（与仁王盾同时机）
- 触发条件：target=刘禅 且 source≠刘禅 且 当前结算牌是 杀
- before-hook 在 检测有效性：source 无基本牌 → cancel（杀对刘禅无效）；source 有基本牌 → 请求回应 requestType='享乐/discard' 让 source 选一张基本牌弃置；弃了 → pass（杀有效），没弃/超时 → cancel（杀无效）
- 基本牌判定：type==='基本牌'（杀/闪/桃）
- 与仁王盾同 atom（检测有效性）天然互斥——cancel 后后续 hook 与 杀结算跳过

**实现差异/边界**：实现与官方描述一致。

### 界放权（主动技）

> **机制简述**：跳过出牌阶段令他人额外回合

**官方效果**：你可以跳过出牌阶段，然后弃牌阶段开始时，你可以弃置一张手牌并令一名其他角色执行一个额外的回合。

**实现要点**：

- 主动技；before hook 挂「阶段开始」(出牌) 询问是否发动；发动则跳过出牌阶段 + 设 放权/active 标记
- before hook 挂「回合结束」处理两种情况——
  - 情况 1（刘禅回合结束 + active）：弃一张手牌 → 选额外回合目标 → cancel 本回合结束 → 手动清理 per-turn 状态 → 亲自启动目标的额外回合
  - 情况 2（额外目标回合结束）：cancel 本回合结束 → 手动清理 → 亲自启动正常下家（刘禅的正常下家）的回合
- 额外回合机制完全在放权技能内部实现，不修改 回合管理.ts；cancel 掉 回合结束 atom 后需手动复刻 per-turn 清理
- 跳过出牌阶段手法同神速/巧变：applyAtom(阶段结束, 出牌) 推进到弃牌 + return cancel

**实现差异/边界**：**触发时机差异**：官方「**弃牌阶段开始时**，你可以弃置一张手牌并令一名其他角色执行一个额外的回合」；实现把弃牌+启动额外回合的时机放在 **回合结束时**（after 回合结束 hook）。语义上等价（都是出牌阶段之后），但与官方明确表述的「弃牌阶段开始时」时机不同。

**与标版差异**：**界限突破新增**：额外回合目标**可选择自己**（标版放权仅限其他角色）。implNotes 明确「原版仅限其他角色」，界版放宽到任意角色。

标版放权：跳过出牌阶段 + 弃一张手牌令**一名其他角色**进行额外回合。界版：可选自己。

### 若愚（觉醒技）

> **机制简述**：体力最小时觉醒加体上限获技能

**官方效果**：主公技，觉醒技，准备阶段，若你是全场体力值最小的角色，你增加1点体力上限，回复至3点体力，然后获得<font href='3045'>“激将</font>”<font href='895'>“思蜀</font>”。

**实现要点**：

- 主公技/觉醒技；时机：回合开始阶段（「回合开始」atom 的 after-hook）
- 触发条件：ownerId 是主公(ownerId===0) + 体力全场最少（或之一）+ 未觉醒
- 流程（强制无选择）：增加 1 点体力上限（设上限 amount = maxHealth + 1）→ 回复 1 点体力 → 永久获得「激将」（添加技能 skillId='激将'）
- 觉醒标记：`player.vars['若愚/awakened']`
- 增上限必须在回复前（先增加上限，再回复），否则回复被旧上限 clamp

**实现差异/边界**：**多处不完整/与官方不符**：

1. **回复量**：官方「**回复至 3 点体力**」；实现「**回复 1 点体力**」——若刘禅当前体力为 1 时被觉醒，实现回复到 2 而非官方的 3。

2. **获得的技能**：官方明确「获得『**激将**』『**思蜀**』」；实现**仅添加「激将」**，**「思蜀」未添加**。
