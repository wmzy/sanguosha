# 黄忠

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 黄忠 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 烈弓 | 被动技 | 当你使用【杀】指定目标时，若目标的体力值或手牌数不小于你，你可以令其不能使用【闪】抵消此【杀】。 |

## 技能详细规则

### 烈弓

- **触发时机**：使用【杀】指定目标后
- **发动条件**：目标的体力值 ≥ 你的体力值，或目标的手牌数 ≥ 你的手牌数
- **效果**：令该角色的【闪】无效，此【杀】强制命中
- **限制**：无次数限制
- **备注**：
  - 两个条件满足任意一个即可
  - 优先对体力值高或手牌多的角色使用
  - 可以配合其他增加伤害的技能

## FAQ/争议点

1. **Q：烈弓能否被【八卦阵】抵消？**
   A：不能。烈弓直接令【闪】无效，八卦阵也无法使用。

2. **Q：如果黄忠体力值为4，目标体力值为3，但目标手牌数为5，黄忠手牌数为2，能否发动？**
   A：可以。目标手牌数（5）≥ 黄忠手牌数（2），满足条件。
