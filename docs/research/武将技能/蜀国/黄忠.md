## 技能详解

### 烈弓（被动技）

> **机制简述**：杀指定目标满足条件则不可抵消

**官方效果**：当你于出牌阶段使用【杀】指定目标后，若其手牌数不小于你的体力值或不大于你的攻击范围，你可以令其不能抵消此【杀】。

**实现要点**：

- 被动技；指定目标 after-hook（source===ownerId, card 是杀）
- 条件满足时询问「是否发动烈弓」→ 确认 → 给目标加标签 `烈弓/禁闪`
- 询问闪 before-hook（source===ownerId 且 target 有 `烈弓/禁闪` 标签）→ 去标签 + cancel（强制命中）
- 标签生命周期：指定目标产出，询问闪消费并清除——按单次杀结算

**实现差异/边界**：**重大差异（条件实现错误）**：

- 官方条件：目标的**手牌数 ≥ 你的体力值** 或 目标的**手牌数 ≤ 你的攻击范围**
- 实现条件：`target.health >= owner.health` 或 `target.hand.length >= owner.hand.length`

实现的两个分支均与官方不符：第一分支应是「目标手牌数 ≥ 自己体力值」而非「目标体力 ≥ 自己体力」；第二分支应是「目标手牌数 ≤ 自己攻击范围」而非「目标手牌数 ≥ 自己手牌数」。判定结果会与官方规则产生系统性偏差。
