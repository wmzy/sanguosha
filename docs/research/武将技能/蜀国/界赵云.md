## 技能详解

### 龙胆（转化技）

> **机制简述**：杀闪互转，酒桃互转使用或打出

**官方效果**：你可以将一张【杀】当【闪】、【闪】当【杀】、【酒】当【桃】、【桃】当【酒】使用或打出。

**实现要点**：

- 转化技；组合 action（武圣+倾国双向）：一个 transform action 带 `to` 参数（'闪'|'杀'）
- 杀当闪：preceding=[龙胆.transform{to:'闪'}] + 主 action=闪.respond（被询问闪时，防御向）
- 闪当杀：preceding=[龙胆.transform{to:'杀'}] + 主 action=杀.use（自己回合）/杀.respond（决斗/南蛮）
- 后端 dispatch 先执行 龙胆.transform（创建影子卡），再 闪/杀 validate 读 cardMap[影子id] 通过
- transform validate 不限定回合/阶段：杀当闪是防御向（非自己回合），闪当杀的回合/次数由 杀.use/respond 校验

**实现差异/边界**：**不完整实现**：官方描述为四向转化——杀↔闪 + **酒↔桃**；implNotes 与实现仅覆盖 **杀↔闪 双向**，**酒↔桃 互转完全未实现**。

### 涯角（被动技）

> **机制简述**：回合外用打出牌时展示牌堆顶，比类型触发效果

**官方效果**：当你于回合外使用或打出手牌时，你可以展示牌堆顶一张牌，若这两张牌类型：相同，你可以令一名角色获得展示牌；不同，你可以弃置攻击范围内包含你的一名角色区域里的一张牌。

**实现要点**：

- 被动技；after hook 挂「移动牌」
- 触发条件 = 手牌(from)→处理区(to) 且非自己回合 且牌堆非空
- 询问发动 → 展示牌堆顶（展示 atom，广播身份）→ 比较类别（cardMap 的 type 字段：基本牌/锦囊牌/装备牌）
- 相同分支 → 请求回应 choosePlayer 选目标（含自己）→ 移动牌（牌堆顶→目标手牌）
- 不同分支 → 请求回应 confirm 是否弃置 → 移动牌（牌堆顶→弃牌堆）
- 每张手牌的使用/打出 = 一次 移动牌，天然满足「每张手牌限一次」

**实现差异/边界**：**重大差异（不同分支实现错误）**：

- 官方「不同」分支：你可以**弃置攻击范围内包含你的一名角色区域里的一张牌**（即弃他人区域的牌）
- 实现「不同」分支：把**展示牌（牌堆顶那张）置入弃牌堆**（弃的是展示牌，不是他人区域的牌）

两者完全不同——实现把展示牌丢掉，而官方是借机拆他人装备/判定牌。同分支（交给一名角色）实现与官方一致。
