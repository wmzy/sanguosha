# 马超

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 马超 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 马术 | 锁定技 | 你计算与其他角色的距离时，始终-1。 |
| 铁骑 | 被动技 | 当你使用【杀】指定一名角色为目标后，你可以进行一次判定，若结果为红色，该角色不能使用【闪】抵消此【杀】。 |

## 技能详细规则

### 马术

- **触发时机**：计算距离时
- **发动条件**：无特殊条件（锁定技，自动生效）
- **效果**：与其他角色的距离-1
- **限制**：锁定技，自动生效
- **备注**：
  - 可以与武器距离叠加
  - 可以使原本不在攻击范围内的角色进入攻击范围

### 铁骑

- **触发时机**：使用【杀】指定目标后
- **发动条件**：无特殊条件
- **效果**：
  1. 进行一次判定
  2. 若结果为红色（红桃或方块），目标角色不能使用【闪】
- **限制**：无次数限制
- **备注**：
  - 判定在【杀】生效前进行
  - 判定结果为黑色则无效果
  - 可以被鬼才等技能改变判定结果

## FAQ/争议点

1. **Q：马术能否使距离变为0？**
   A：不能。最小距离为1。

2. **Q：铁骑判定为红色后，目标角色能否使用【八卦阵】？**
   A：可以使用【八卦阵】进行判定，但即使判定出【闪】也不能使用。

3. **Q：铁骑能否被【无懈可击】抵消？**
   A：不能。铁骑不是锦囊牌效果。
