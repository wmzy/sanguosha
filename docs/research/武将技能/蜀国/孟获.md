# 孟获

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 孟获 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 祸首 | 锁定技 | 【南蛮入侵】对你无效；你是任何【南蛮入侵】造成伤害的来源。 |
| 再起 | 主动技 | 摸牌阶段，若你已受伤，你可以放弃摸牌并展示牌堆顶X张牌（X为你已损失体力值），每有一张红桃回复1点体力，然后弃掉这些红桃牌，将其余的牌收入手牌。 |

## 技能详细规则

### 祸首

- **触发时机**：【南蛮入侵】结算时
- **发动条件**：锁定技，自动生效
- **效果**：
  - 【南蛮入侵】对孟获无效
  - 孟获是任何【南蛮入侵】造成伤害的来源
- **限制**：锁定技
- **备注**：
  - 孟获免疫南蛮入侵的伤害
  - 如果有人被南蛮入侵伤害，伤害来源视为孟获（可能让孟获背锅）

### 再起

- **触发时机**：摸牌阶段
- **发动条件**：已受伤（当前体力值<体力上限）
- **效果**：
  1. 放弃摸牌（不摸2张牌）
  2. 展示牌堆顶X张牌（X=已损失体力值）
  3. 每有一张红桃，回复1点体力
  4. 弃掉红桃牌，其余牌收入手牌
- **限制**：每回合限一次
- **备注**：
  - 回血和摸牌可以兼顾
  - 红桃牌越多收益越大（回血+摸牌）
  - 没有红桃则只能摸X张牌不回血

## FAQ/争议点

1. **Q：再起展示的红桃牌会进入弃牌堆吗？**
   A：会。红桃牌弃置，非红桃牌收入手牌。

2. **Q：祸首导致孟获成为南蛮入侵的伤害来源，有什么影响？**
   A：杀死反贼时，孟获会获得奖励/惩罚（摸牌或弃牌）。
