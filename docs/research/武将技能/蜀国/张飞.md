# 张飞

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 张飞 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 咆哮 | 锁定技 | 出牌阶段，你使用【杀】无次数限制。 |

## 技能详细规则

### 咆哮

- **触发时机**：出牌阶段
- **发动条件**：无特殊条件（锁定技，自动生效）
- **效果**：使用【杀】不受次数限制
- **限制**：无
- **备注**：
  - 锁定技，必须生效
  - 正常情况下每回合只能使用一张【杀】，张飞无此限制
  - 可以配合【诸葛连弩】等武器多次使用【杀】

## FAQ/争议点

1. **Q：咆哮能否被【乐不思蜀】阻止？**
   A：【乐不思蜀】跳过出牌阶段，因此张飞无法使用【杀】。

2. **Q：咆哮能否与【诸葛连弩】叠加？**
   A：咆哮已经移除了使用次数限制，【诸葛连弩】的效果变得冗余。

3. **Q：咆哮能否在回合外发动？**
   A：不能。咆哮仅在出牌阶段有效。
