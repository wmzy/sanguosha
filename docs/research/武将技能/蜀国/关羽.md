# 关羽

## 基本信息

| 属性 | 值 |
|------|-----|
| 名称 | 关羽 |
| 势力 | 蜀 |
| 体力上限 | 4 |
| 性别 | 男 |

## 技能列表

| 技能名称 | 类型 | 描述 |
|----------|------|------|
| 武圣 | 主动技 | 你可以将一张红色牌当【杀】使用或打出。 |

## 技能详细规则

### 武圣

- **触发时机**：需要使用或打出【杀】时
- **发动条件**：有红色牌（手牌或装备区的牌）
- **效果**：将一张红色牌当【杀】使用或打出
- **限制**：无次数限制
- **备注**：
  - 红色牌包括红桃和方块花色的牌
  - 可以使用装备区的红色牌
  - 使用后原牌进入弃牌堆
  - 受【杀】的使用次数限制（每回合通常只能使用一张【杀】）

## FAQ/争议点

1. **Q：武圣使用的【杀】能否被【闪】抵消？**
   A：可以。武圣使用的【杀】与普通【杀】相同，可以被【闪】抵消。

2. **Q：武圣能否使用装备区的红色装备牌？**
   A：可以。但装备会因此被弃置。

3. **Q：武圣能否与【青龙偃月刀】配合？**
   A：可以。武圣使用的【杀】享受武器效果。
