# 三国杀标准版 + 军争篇 - 完整游戏规则

## 1. 游戏流程

### 1.1 游戏准备

#### 1.1.1 人数与身份

三国杀标准版支持 **4-8 人**游戏（最常见为 8 人局）。游戏中有四种身份：

| 身份 | 数量（8人局） | 说明 |
|------|-------------|------|
| 主公 | 1 人 | 公开身份，游戏开始时即亮明 |
| 忠臣 | 2 人 | 隐藏身份，保护主公 |
| 反贼 | 4 人 | 隐藏身份，击杀主公 |
| 内奸 | 1 人 | 隐藏身份，需先清除反贼再单挑主公 |

**各人数局身份分配：**

| 总人数 | 主公 | 忠臣 | 反贼 | 内奸 |
|--------|------|------|------|------|
| 4 | 1 | 1 | 1 | 1 |
| 5 | 1 | 1 | 2 | 1 |
| 6 | 1 | 1 | 3 | 1 |
| 7 | 1 | 2 | 3 | 1 |
| 8 | 1 | 2 | 4 | 1 |

#### 1.1.2 身份分配

1. 将身份牌洗混，每人随机抽取一张
2. 主公身份牌亮出（公开），其余身份保密
3. 游戏过程中，所有玩家的身份牌均暗置，直到死亡时才翻开

#### 1.1.3 角色选择

1. 每人随机发给若干张武将牌（通常 3 张），从中选择一张作为本局使用的武将
2. 主公可优先选择，且通常有更多的武将牌可选
3. 选定后亮出武将牌，记录该武将的体力上限和技能

#### 1.1.4 初始手牌

- 每人从牌堆摸 **4 张手牌** 作为起始手牌
- 主公额外摸 **1 张**（共 5 张），作为先手优势的补偿

#### 1.1.5 游戏开始

- 从主公开始，按逆时针（或顺时针，视版本而定）方向依次进行回合
- 每位玩家的回合按固定阶段顺序执行

---

### 1.2 回合流程

每个玩家的回合包含以下 **6 个阶段**，按顺序依次执行：

#### 1.2.1 准备阶段（Start Phase）

- **定义**：回合开始时的第一个阶段
- **主要功能**：
  - 部分武将技能在此阶段触发（如诸葛亮的"观星"）
  - 延时锦囊（如乐不思蜀、闪电）在此阶段进行判定
- **规则**：
  - 准备阶段可以跳过（某些技能效果）
  - 此阶段通常不进行常规操作

#### 1.2.2 判定阶段（Judge Phase）

- **定义**：处理延时锦囊判定的阶段
- **主要功能**：
  - 判定区域内有延时锦囊时，进行判定
  - 判定顺序：后放置的先判定（后进先出）
- **判定流程**：
  1. 从牌堆顶翻出一张牌作为判定牌
  2. 根据判定结果决定延时锦囊是否生效
  3. 判定牌进入弃牌堆
  4. 如有"改判"技能（如司马懿的"鬼才"），可在此时发动

#### 1.2.3 摸牌阶段（Draw Phase）

- **定义**：从牌堆摸取手牌的阶段
- **规则**：
  - 默认从牌堆顶摸 **2 张牌** 加入手牌
  - 某些武将技能可改变摸牌数量（如周瑜的"英姿"摸 3 张）
  - 牌堆用完时，将弃牌堆洗混重新形成牌堆
  - 摸牌阶段不可跳过（除非有特殊效果）

#### 1.2.4 出牌阶段（Play Phase）

- **定义**：使用卡牌和发动技能的主要阶段
- **可执行的操作**：
  - 使用基本牌（杀、闪、桃）
  - 使用锦囊牌
  - 装备装备牌
  - 发动武将技能（部分技能）
- **重要限制**：
  - **每回合只能使用 1 张【杀】**（除非有技能突破此限制，如诸葛连弩）
  - 手牌上限 = 当前体力值（回合结束时弃牌）
  - 出牌阶段可以随时结束，进入弃牌阶段
- **出牌规则**：
  - 使用卡牌需指定目标（部分卡牌无需指定）
  - 目标可以对卡牌进行"响应"（如用闪响应杀）
  - 所有玩家可使用"无懈可击"来抵消锦囊牌

#### 1.2.5 弃牌阶段（Discard Phase）

- **定义**：超出手牌上限时弃牌的阶段
- **规则**：
  - 检查手牌数是否超过当前体力值
  - 若超出，必须弃牌至手牌上限
  - 弃牌由玩家自行选择弃哪些牌
  - 弃牌进入弃牌堆
- **注意**：
  - 当前体力值可能因伤害而减少，导致手牌上限降低
  - 若手牌数未超过体力值，则无需弃牌

#### 1.2.6 结束阶段（End Phase）

- **定义**：回合的最后阶段
- **主要功能**：
  - 部分武将技能在此阶段触发
  - 回合结束的标志
  - 某些持续效果在此阶段结算
- **规则**：
  - 回合结束后，轮到下一位玩家

---

### 1.3 回合切换规则

- 回合按座位顺序依次进行（通常为逆时针）
- 当前玩家回合结束后，下一位玩家开始新回合
- 死亡玩家跳过，不占用回合
- 若所有存活玩家均完成一轮，则游戏进入下一轮
- 回合数不限，直到满足胜利条件

## 2. 卡牌使用规则

### 2.1 基本牌

#### 2.1.1 杀（Attack）

| 属性 | 说明 |
|------|------|
| 类型 | 基本牌 |
| 数量 | 30 张（标准版） |
| 使用时机 | 出牌阶段 |
| 目标 | 攻击范围内的 1 名其他角色 |
| 效果 | 对目标造成 1 点伤害（若目标未使用闪） |

**使用规则：**
- 每回合只能使用 **1 张杀**（基础规则，武器技能可突破）
- 只能对 **攻击范围内** 的角色使用
- 目标角色可以使用 **1 张闪** 来抵消
- 使用杀时必须指定目标
- 杀造成的伤害类型为"杀"（非属性伤害）

**属性杀（扩展包）：**
- 火杀：造成火焰伤害，可触发藤甲等
- 雷杀：造成雷电伤害，可触发铁索连环等

#### 2.1.2 闪（Dodge）

| 属性 | 说明 |
|------|------|
| 类型 | 基本牌 |
| 数量 | 15 张（标准版） |
| 使用时机 | 响应杀时使用 |
| 目标 | 无（被动使用） |
| 效果 | 抵消 1 次杀的伤害 |

**使用规则：**
- **只能在被使用杀时** 作为响应使用
- 不能主动使用（非出牌阶段可使用的牌）
- 使用 1 张闪抵消 1 张杀
- 八卦阵等装备可代替闪的效果

#### 2.1.3 桃（Peach）

| 属性 | 说明 |
|------|------|
| 类型 | 基本牌 |
| 数量 | 8 张（标准版） |
| 使用时机 | 出牌阶段 或 濒死状态时 |
| 目标 | 自己 或 濒死的角色 |
| 效果 | 回复 1 点体力 |

**使用规则：**
- **出牌阶段**：只能对自己使用，回复 1 点体力（体力未满时才能使用）
- **濒死状态**：任何角色处于濒死状态时，可以对濒死角色使用桃（包括自己救自己）
- 桃是救命的关键牌，数量稀少
- 使用桃回复体力时，体力值不能超过体力上限

---

#### 2.1.4 酒（Wine）

| 属性 | 说明 |
|------|------|
| 类型 | 基本牌 |
| 数量 | 5 张（军争篇） |
| 使用时机 | 出牌阶段 或 濒死状态时 |
| 目标 | 自己 或 濒死的角色 |
| 效果 | 出牌阶段：本回合下一张杀伤害+1；濒死时：回复1点体力 |

**使用规则：**
- **出牌阶段**：对自己使用，使本回合使用的下一张【杀】伤害+1
- **濒死状态**：对濒死角色使用，回复1点体力（此时不提供伤害加成）
- 每回合只能使用1次【酒】（提升杀伤害的模式）
- 酒杀命中造成2点伤害，可与技能加成叠加

---

#### 2.1.5 火杀（Fire Attack）

| 属性 | 说明 |
|------|------|
| 类型 | 基本牌 |
| 数量 | 5 张（军争篇） |
| 使用时机 | 出牌阶段 |
| 目标 | 攻击范围内的一名其他角色 |
| 效果 | 造成1点火焰伤害 |

**使用规则：**
- 与普通【杀】用法相同，但造成**火焰伤害**
- 火焰伤害可触发铁索连环的传导
- 对装备【藤甲】的角色伤害+1
- 与普通杀共享每回合使用次数限制

---

#### 2.1.6 雷杀（Thunder Attack）

| 属性 | 说明 |
|------|------|
| 类型 | 基本牌 |
| 数量 | 9 张（军争篇） |
| 使用时机 | 出牌阶段 |
| 目标 | 攻击范围内的一名其他角色 |
| 效果 | 造成1点雷电伤害 |

**使用规则：**
- 与普通【杀】用法相同，但造成**雷电伤害**
- 雷电伤害可触发铁索连环的传导
- 不受藤甲防止（藤甲只防止普通杀和非属性锦囊伤害）
- 与普通杀共享每回合使用次数限制

---

### 2.2 锦囊牌

锦囊牌分为三类：**单目标锦囊**、**全体锦囊**、**延时锦囊**。

#### 2.2.1 单目标锦囊

**过河拆桥（Dismantlement）**
- 目标：1 名其他角色
- 效果：弃置目标角色的 1 张牌（手牌或装备区的牌，由使用者选择）
- 说明：无需判定，直接生效

**顺手牵羊（Snatch）**
- 目标：距离 1 以内的 1 名其他角色
- 效果：获得目标角色的 1 张牌
- 说明：需要满足距离条件

**决斗（Duel）**
- 目标：1 名其他角色
- 效果：双方轮流使用杀，先无法使用杀的一方受到 1 点伤害
- 说明：由使用者先出杀，目标后出

**借刀杀人（Collateral）**
- 目标：装备区有武器的 1 名角色
- 效果：该角色必须对其攻击范围内的一名角色使用杀，否则失去武器

#### 2.2.2 全体锦囊

**万箭齐发（Arrow Barrage）**
- 目标：所有其他角色
- 效果：每名目标角色需使用 1 张闪，否则受到 1 点伤害
- 说明：每人单独响应，不使用者受伤害

**南蛮入侵（Barbarian Invasion）**
- 目标：所有其他角色
- 效果：每名目标角色需使用 1 张杀，否则受到 1 点伤害
- 说明：每人单独响应

**桃园结义（Bountiful Harvest）**
- 目标：所有角色（含使用者）
- 效果：每名角色回复 1 点体力（已满血的无效）

**五谷丰登（Abundant Harvest）**
- 目标：所有角色
- 效果：从牌堆摸等同于存活人数的牌，按座位顺序每人选择 1 张

**无中生有（Ex Nihilo）**
- 目标：自己
- 效果：摸 2 张牌

#### 2.2.3 延时锦囊

延时锦囊需放置在目标角色的判定区，在该角色的判定阶段进行判定后生效。

**乐不思蜀（Indulgence）**
- 目标：1 名其他角色
- 放置：放在目标角色的判定区
- 判定：翻出判定牌，若为红桃则无效，其他花色生效
- 效果：若生效，该角色跳过出牌阶段

**闪电（Lightning）**
- 目标：自己（可传递）
- 放置：放在自己的判定区
- 判定：翻出判定牌，若为黑桃 2-9 则生效
- 效果：若生效，该角色受到 3 点雷电伤害
- 传递：判定未生效时，传递给下家

**兵粮寸断（Supply Shortage）**
- 目标：距离 1 以内的 1 名其他角色
- 放置：放在目标角色的判定区
- 判定：翻出判定牌，若为梅花则无效，其他花色生效
- 效果：若生效，该角色跳过摸牌阶段

---

### 2.3 装备牌

装备牌放在装备区，提供持续效果。每人有 4 个装备栏位：武器、防具、+1马、-1马。

#### 2.3.1 武器

| 武器 | 攻击范围 | 特殊效果 |
|------|----------|----------|
| 诸葛连弩 | 1 | 出牌阶段可使用任意数量的杀 |
| 雌雄双股剑 | 2 | 对异性角色使用杀时，对方需弃一张手牌或使用者摸一张牌 |
| 青釭剑 | 2 | 杀无视目标的防具 |
| 青龙偃月刀 | 3 | 杀被闪抵消时，可继续对同一目标使用杀 |
| 丈八蛇矛 | 3 | 可将两张手牌当杀使用 |
| 贯石斧 | 3 | 杀被闪抵消时，可弃两张牌强制命中 |
| 方天画戟 | 4 | 手牌数为0时，杀可指定三个目标 |
| 麒麟弓 | 5 | 杀造成伤害时，可弃一张装备区的马 |
| **古锭刀**（军争） | 2 | 若目标没有手牌，此杀伤害+1 |
| **朱雀羽扇**（军争） | 4 | 可将普通杀转化为火杀 |

#### 2.3.2 防具

| 防具 | 效果 |
|------|------|
| 八卦阵 | 需要使用或打出闪时，可进行判定：红色牌视为使用了闪 |
| 仁王盾 | 黑色的杀对装备者无效 |
| **藤甲**（军争） | 防止普通杀和非属性锦囊伤害；火焰伤害+1 |
| **白银狮子**（军争） | 受到伤害最多为1；失去装备时回复1点体力 |

#### 2.3.3 坐骑

| 坐骑 | 效果 |
|------|------|
| +1马（防御马） | 其他角色与你的距离 +1（更难打到你） |
| -1马（进攻马） | 你与其他角色的距离 -1（更容易打到别人） |

**军争篇新增：** 骅骝（♦K），效果同+1马。

#### 2.3.4 装备规则

- 装备牌在出牌阶段使用
- 每个栏位只能装备 1 件，新装备替换旧装备（旧装备进入弃牌堆）
- 装备牌的效果持续生效，直到被替换或弃置
- 过河拆桥等可以移除装备区的牌

## 3. 距离和攻击范围

### 3.1 距离计算规则

**基础距离：**
- 相邻座位（左右邻居）的距离为 1
- 隔一个座位的距离为 2，以此类推
- 距离取两个方向的较小值（环形座位）

**示例（8人局）：**
```
        P1
    P8      P2
  P7          P3
    P6      P4
        P5

P1 到 P2 = 1（相邻）
P1 到 P3 = 2（隔一人）
P1 到 P4 = 3
P1 到 P5 = 4（最远距离，取两个方向最小值）
P1 到 P6 = 3（另一方向：P1→P8→P7→P6 = 3，取 min(4,3) = 3）
```

**坐骑影响：**
- 你装备 **-1 马**：你与其他所有角色的距离 -1
- 目标装备 **+1 马**：你到该角色的距离 +1
- 两种效果可叠加

### 3.2 攻击范围

- 基础攻击范围为 **1**（只能打到相邻角色）
- 装备武器后，攻击范围变为 **武器的攻击范围值**
- 攻击范围与距离是独立计算的

**判定条件：**
- 使用杀时，目标必须在你的攻击范围内
- 即：`距离(你, 目标) <= 你的攻击范围`

### 3.3 +1 马 / -1 马的效果

**-1 马（进攻马/绝影等）：**
- 效果：你计算与其他角色的距离时 -1
- 作用：更容易杀到远处的角色
- 可以装备多匹（但效果不叠加，只装备一匹）

**+1 马（防御马/的卢等）：**
- 效果：其他角色计算与你的距离时 +1
- 作用：更难被远处角色的杀命中
- 注意：是别人到你的距离增加，不是你到别人的距离

**距离计算示例：**
```
若 P1 装备 -1 马，P3 装备 +1 马：
P1 到 P3 的距离 = 2（基础） - 1（P1的-1马） + 1（P3的+1马） = 2

若只有 P1 装备 -1 马：
P1 到 P3 的距离 = 2 - 1 = 1
```

## 4. 伤害和死亡

### 4.1 伤害结算流程

当一名角色即将受到伤害时，按以下流程结算：

1. **伤害来源确认**：确定伤害的来源角色（用于奖惩判定）
2. **伤害类型确认**：普通伤害、火焰伤害、雷电伤害
3. **防具效果**：检查是否有防具可以减免（如仁王盾、藤甲）
4. **伤害结算**：
   - 扣减目标角色的体力值
   - 体力值扣减后检查是否进入濒死状态
5. **伤害后效果**：触发相关技能（如奸雄、遗计等）

**伤害类型：**
| 类型 | 来源 | 说明 |
|------|------|------|
| 普通伤害 | 杀、锦囊 | 无属性 |
| 火焰伤害 | 火杀、火攻 | 可触发藤甲额外伤害 |
| 雷电伤害 | 雷杀、闪电 | 可通过铁索连环传导 |

### 4.2 濒死状态和桃救援

#### 4.2.1 濒死状态

- 当角色的体力值 **降到 0 或以下** 时，进入濒死状态
- 濒死状态的角色 **立即** 需要回复体力

#### 4.2.2 桃救援流程

1. **进入濒死**：体力值 <= 0
2. **求桃**：濒死角色从当前回合玩家开始，按顺序询问所有存活角色是否使用桃
3. **使用桃**：
   - 任何角色可以对濒死角色使用桃（从手牌中使用）
   - 每个桃回复 1 点体力
   - 需要回复到体力值 > 0 才算脱离濒死
4. **濒死角色自救**：濒死角色自己也可以使用桃自救
5. **无人救援**：若所有角色均无桃或不愿使用，该角色死亡

**救援顺序：**
- 从当前回合玩家开始，按座位顺序依次询问
- 每人只能使用 1 张桃（除非有技能允许更多）
- 桃的使用属于即时结算，不进入出牌阶段

### 4.3 死亡处理

#### 4.3.1 死亡流程

1. **体力归零**：体力值降到 0 以下且无人救援
2. **弃置所有牌**：死亡角色弃置手牌和装备区所有牌
3. **翻开身份牌**：死亡角色的身份公开
4. **触发奖惩**：根据击杀关系执行奖惩
5. **移除角色**：死亡角色退出游戏，不再参与后续流程

#### 4.3.2 奖惩规则

**击杀反贼：**
- 击杀者（无论身份）立即摸 **3 张牌** 作为奖励
- 这是鼓励攻击反贼的机制

**击杀忠臣：**
- 若主公击杀忠臣，主公必须 **弃置所有手牌和装备**（巨大惩罚）
- 其他身份击杀忠臣无特殊惩罚

**击杀主公：**
- 反贼击杀主公：游戏立即结束，反贼胜利
- 忠臣/内奸击杀主公：通常视为叛变，但规则上不额外惩罚（由局势决定）

**击杀内奸：**
- 无特殊奖惩

**特殊情况 - 杀死自己：**
- 若因技能或锦囊导致自己死亡，无奖惩

## 5. 胜利条件

### 5.1 主公/忠臣胜利条件

**胜利条件：所有反贼和内奸均死亡**

- 主公和忠臣属于"主忠方"
- 需要清除所有反贼和内奸
- 主公存活是必要条件（主公死亡则主忠方失败）

### 5.2 反贼胜利条件

**胜利条件：主公死亡**

- 反贼只需击杀主公即可获胜
- 不需要清除忠臣或内奸
- 反贼之间的合作是关键
- 即使反贼全部死亡，只要主公也死亡，反贼仍然胜利（但这种情况很少）

### 5.3 内奸胜利条件

**胜利条件：存活到最后，与主公单挑并获胜**

- 内奸是最难的身份，需要：
  1. 先帮助主公清除所有反贼
  2. 再帮助主公清除所有忠臣
  3. 最后与主公 1v1 单挑并获胜
- 内奸必须存活到最后才算胜利
- 若主公死亡时内奸存活但场上还有反贼，内奸失败

### 5.4 胜负判定流程

游戏在以下时刻判定胜负：

1. **主公死亡时**：
   - 场上存活角色中若有反贼 → 反贼胜利
   - 场上存活角色中无反贼 → 游戏继续（特殊情况）

2. **所有反贼和内奸死亡时**：
   - 主公和忠臣胜利

3. **只剩主公和内奸时**：
   - 进入单挑阶段
   - 内奸击杀主公 → 内奸胜利
   - 主公击杀内奸 → 主公胜利

## 6. 特殊规则

### 6.1 无懈可击的使用规则

**无懈可击（Nullification）**

| 属性 | 说明 |
|------|------|
| 类型 | 锦囊牌 |
| 数量 | 3 张（标准版） |
| 使用时机 | 任何锦囊牌生效前 |
| 效果 | 抵消 1 张锦囊牌的效果 |

**使用规则：**
- **使用时机**：在任何锦囊牌结算前，任何角色都可以使用无懈可击
- **抵消对象**：可以抵消单目标锦囊、全体锦囊、延时锦囊的效果
- **连锁使用**：无懈可击本身也可以被另一张无懈可击抵消
- **使用优先级**：从当前回合玩家开始，按座位顺序依次响应

**使用场景：**
1. **抵消敌方锦囊**：如抵消对方的过河拆桥、顺手牵羊
2. **保护队友**：如抵消对队友使用的决斗
3. **延时锦囊**：在判定前使用，使延时锦囊直接失效

**无懈可击的连锁：**
- 当使用无懈可击时，其他玩家可再使用无懈可击来抵消
- 形成"无懈可击大战"
- 最后一张无懈可击决定原锦囊是否生效

### 6.2 判定系统

#### 6.2.1 判定的基本流程

1. **触发判定**：延时锦囊或技能要求进行判定
2. **翻判定牌**：从牌堆顶翻出 1 张牌作为判定牌
3. **确定结果**：根据牌的花色/点数判定是否生效
4. **处理结果**：生效或无效
5. **弃置判定牌**：判定牌进入弃牌堆

#### 6.2.2 判定牌的性质

- 判定牌在判定前可以被技能修改（如司马懿的"鬼才"）
- 判定牌翻出后，进入弃牌堆前可以被获得（如张角的"鬼道"）
- 判定牌不属于任何人的手牌

#### 6.2.3 改判技能

某些武将拥有改判技能，可以在判定牌翻出后修改结果：

- **司马懿 - 鬼才**：可打出一张手牌代替判定牌
- **张角 - 鬼道**：可用一张黑色牌替换判定牌，并获得原判定牌

### 6.3 延时锦囊的判定和生效

#### 6.3.1 乐不思蜀

- **放置时机**：出牌阶段对目标使用
- **判定时机**：目标角色的判定阶段
- **判定规则**：翻出判定牌
  - **红桃** → 无效（判定牌弃置，正常进行回合）
  - **其他花色** → 生效（跳过出牌阶段）
- **效果**：跳过整个出牌阶段，不能使用任何牌

#### 6.3.2 闪电

- **放置时机**：出牌阶段对自己使用
- **判定时机**：自己的判定阶段
- **判定规则**：翻出判定牌
  - **黑桃 2-9** → 生效（受到 3 点雷电伤害）
  - **其他** → 无效（闪电传递给下家）
- **传递规则**：
  - 无效时，闪电移动到下家的判定区
  - 下家在自己的判定阶段继续判定
  - 闪电在所有人之间循环，直到有人判定生效

#### 6.3.3 兵粮寸断

- **放置时机**：出牌阶段对距离 1 以内的目标使用
- **判定时机**：目标角色的判定阶段（在摸牌阶段前）
- **判定规则**：翻出判定牌
  - **梅花** → 无效（正常摸牌）
  - **其他花色** → 生效（跳过摸牌阶段）
- **效果**：跳过摸牌阶段，不摸任何牌

#### 6.3.4 延时锦囊的通用规则

- **判定顺序**：后放置的先判定（堆栈原则）
- **不可叠加**：同一种延时锦囊只能放置 1 张在判定区
- **清除方式**：
  - 判定后自动清除
  - 过河拆桥可弃置判定区的延时锦囊
  - 部分技能可移除

## 附录

### A. 常见问题

**Q: 杀和闪的使用时机是什么？**
A: 杀在出牌阶段主动使用；闪只能在被使用杀时作为响应使用。

**Q: 可以对自己使用杀吗？**
A: 不可以，杀只能对其他角色使用。

**Q: 桃可以在出牌阶段对其他人使用吗？**
A: 不可以，出牌阶段的桃只能对自己使用。只有在濒死救援时才能对他人使用桃。

**Q: 无懈可击可以抵消杀或闪吗？**
A: 不可以，无懈可击只能抵消锦囊牌的效果，不能抵消基本牌。

**Q: 距离和攻击范围是一样的吗？**
A: 不一样。距离是两个角色之间的数值；攻击范围是武器赋予的数值。杀的条件是目标在攻击范围内，即距离 <= 攻击范围。

**Q: 装备新武器时，旧武器去哪？**
A: 旧武器进入弃牌堆。

**Q: 内奸可以在游戏初期就攻击主公吗？**
A: 可以，但通常不明智。内奸的最佳策略是先清除反贼，再清除忠臣，最后单挑主公。
