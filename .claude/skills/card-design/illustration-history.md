# 卡牌插画历史外形描述参考

> **用途**:本表供 `card-design` skill 与 `scripts/gen-card.ts` 复用。生成卡牌时,从本表取该牌的**外形描述**(纯视觉特征,不含历史典故背景),嵌入 prompt 的插画描述部分。
>
> **来源**:依据《三国演义》原文、《古今刀剑录》、史料记载、BWIKI 官方条目,提炼为纯外形特征。
>
> **原则**:prompt 中只放外形描述(供模型绘制),不放历史典故(典故会干扰画面文字)。若需补充新牌,按同样格式:先查权威来源,再提炼为外形特征。

## 武器牌

### 古锭刀
- **外形**:一柄中国古代环首刀。刀身细长、略弯曲、轻薄,刀尖上翘,刀身可有松纹纹理;刀柄末端有圆环(环首),刀柄缠绳;整体古朴庄重,属单刃长刀(非双刃直剑)
- **来源**:《三国演义》第五回"横古锭刀";史料载环首刀类,长约1.2米,适马上作战

### 青龙偃月刀
- **外形**:一柄长柄大刀。刀身宽厚呈月牙形,刀背厚重,刀刃寒光;长木柄;整体气势威武
- **来源**:《三国演义》第一回关羽打造,重八十二斤,又名"冷艳锯"

### 丈八蛇矛
- **外形**:一杆长矛。矛头蛇形蜿蜒,矛尖锋利;长木柄,全用镔铁点钢
- **来源**:《三国演义》第一回张飞打造,长一丈八寸

### 雌雄双股剑
- **外形**:两柄古剑并陈。一雄一雌,剑身一长一短,剑柄纹饰有别
- **来源**:《三国演义》第一回刘备打造双剑

### 青釭剑
- **外形**:一柄寒光凛凛的古剑。剑身青光,剑鞘华贵,削铁如泥
- **来源**:《三国演义》曹操宝剑,后赐夏侯恩,长坂坡被赵云夺之

### 方天画戟
- **外形**:一柄画戟。戟头双月牙形,戟身华丽纹饰;长柄
- **来源**:《三国演义》吕布兵器

### 诸葛连弩
- **外形**:精巧的连弩机械结构。可容纳多支箭矢,木质弩身,铜质机件
- **来源**:诸葛亮所造,可连发十矢

### 寒冰剑
- **外形**:一柄剑身凝结冰晶的冷剑。剑身泛冷蓝光泽
- **来源**:标准包EX武器

### 贯石斧
- **外形**:一柄厚重的石质战斧。斧身粗犷
- **来源**:标准包武器

### 麒麟弓
- **外形**:一柄雕饰麒麟纹的长弓。弓身华丽
- **来源**:标准包武器

### 朱雀羽扇
- **外形**:一柄饰有朱雀羽毛的羽扇。扇面火红羽毛,扇柄精致
- **来源**:军争篇武器

## 获取新牌历史外形描述的步骤(skill 复用)

当本表未收录某张牌时,按以下步骤获取:

1. **查权威来源**:
   - BWIKI 条目页:`https://wiki.biligame.com/sgs/<牌名>`(武器/防具/锦囊各有合集页)
   - 《三国演义》原文检索(百度/Google 搜"三国演义 牌名")
   - 《古今刀剑录》(维基文库:`zh.wikisource.org/wiki/古今刀劍錄`)
   - 官方规则集 3.0:`gltjk.com/sanguosha/rules/card/equipment.html`

2. **提炼外形特征**(纯视觉,不含典故):
   - 兵器:刀/剑/矛/戟/弓/弩 + 形状(直/弯/宽/细)+ 纹饰 + 材质
   - 防具:盾/甲/铠 + 形状 + 纹饰 + 颜色
   - 坐骑:马匹 + 毛色 + 动态(奔驰/跃起)
   - 基本/锦囊:武将/场景 + 动态 + 氛围

3. **加入本表**:按上述格式(牌名→外形→来源)追加到对应分类下

4. **同步脚本**:若 `scripts/gen-card.ts` 的 `ILLUSTRATION_HISTORY` 也维护一份,同步更新(或改为从本 md 读取)
